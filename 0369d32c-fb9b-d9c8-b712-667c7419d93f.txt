
<br/>　　<br/>　　    范成梁叫的人匆匆赶来了。<br/>　　<br/>　　    来的这四个人，一出场身上就标榜着四个闪着金光的大字。<br/>　　<br/>　　    “成功人士”<br/>　　<br/>　　    标准的制服OL，衣冠楚楚，脸上泛着正经的一丝不苟的表情，身上散发出了浓浓的商业和知识储备粮的气息，齐刷刷的走进来时，你就会觉得，这四个人就应该是财经论坛和商务周刊上经常出场的那种人物。<br/>　　<br/>　　    事实也是如此，如果有商场中人看见他们的话，可能会有的觉得眼熟，有的则会说一声，原来是他。<br/>　　<br/>　　    四个人，三男一女，商业银行的财务官叫董良，今年五十多岁，曾经供职于某个投行，在多年前被范氏家族找到麾下，然后委任在了他们家旁系控股的一家商业银行中掌管财务运作，两个房地产公司的人也同样如此，也是出自范家的地产企业，年龄四五十岁上下，一个叫张泽雨，另外一个叫谭明哲，剩下那位律师事务所的首席律师就年轻多了，大概三十多岁，浑身上下都散发着一股成熟女性的气息。<br/>　　<br/>　　    这四个人都是绝对忠于范家的人，也可以说是忠于范家给的待遇和关系，范成梁亲自给他们打的电话。<br/>　　<br/>　　    出于礼貌，王惊蛰起身朝着他们打了声招呼，示意人坐下，然后主动泡了几杯茶，这几人一进来时，他就从对方的面上看出来了，这肯定都是那种稀缺的商业人才，本事绝对杠杠的。<br/>　　<br/>　　    在几人面前依次递过一杯茶，王惊蛰问道：“你们来的时候，范成梁是这么和你说的？”<br/>　　<br/>　　    董良扶了扶鼻梁上的金丝框眼镜，说道：“成梁说，我们来了以后，一切都听王先生的”<br/>　　<br/>　　    “呵呵，说什么听什么？”<br/>　　<br/>　　    董良点头说道：“对，您的任何指示”<br/>　　<br/>　　    “唉，懂事！”王惊蛰坐在沙发上，翘着二郎腿说道：“喝茶，喝茶”<br/>　　<br/>　　    郭茜茜转过脑袋抚着额头，心里一声长叹，真是人比人得死啊，你看这几个一出场就自带背景音乐，但王惊蛰往那一坐十足像个早年肄业的不良青年，一身行头下来不超过两百块，脑袋上还顶着个乱遭的鸡窝头，说好听点这叫不修边幅，说的难听了那就是寒碜！<br/>　　<br/>　　    偏偏他还得对这几人发号施令，尴尬不？<br/>　　<br/>　　    不过你从这几个精英的脸上，到没有看出他们有什么不满和怠慢的表情，就是表现的挺平淡的。<br/>　　<br/>　　    “啪”王惊蛰翘着腿点了根烟，然后依靠着沙发，很快就进入了主题：“既然范成梁跟你们说，我说什么听什么，那我就开始了哈，之前他应该跟你们提过一家企业吧？”<br/>　　<br/>　　    张泽雨“哦”了一声，从公文包里，拿出一叠资料递了过去，王惊蛰接到手里后低头翻看了几眼，第一页上是恒晟地产几个字，再往下就是关于这家地产公司的一些概况了，都是一些专业的数据和企业状况，王惊蛰是勉强能把上面写的字都认全了，但至于是什么意思，他两眼一抹黑，完全俺不懂。<br/>　　<br/>　　    王惊蛰又还给对方，说道：“你们介绍，我听听就行了，然后尽量简洁明了一点，毕竟说复杂了我可能理解不了听不明白”<br/>　　<br/>　　    郭茜茜又把脑袋转了过去。<br/>　　<br/>　　    “这是我在飞机上整理出来的，然后下飞机在路上我也跟一些同行打听了下，时间太短，我查到的消息可能也有限，是这么回事王先生”张泽雨正经端坐着，徐徐的说道：“恒晟地产呢，主营项目是楼盘开发，和在多个城市运作CBD商务区，涵盖写字楼模块，商业广场……”<br/>　　<br/>　　    “等一下，那个我问问，什么叫CID？”王惊蛰打断对方，认真的问道。<br/>　　<br/>　　    对于他的不学无术，郭茜茜很熟练的又把脑袋转过去了。<br/>　　<br/>　　    张泽雨愣了愣，然后耐心的解释道：“CID是皇家警察的意思，已于1973年解散了，我刚刚说的是CBD，通俗点解释就是大型商务集中区域，通常都位于一个城市的中心，集中了这个城市的经济，科技和文化等产业”<br/>　　<br/>　　    王惊蛰说道:“哦，哦，行，你接着说”<br/>　　<br/>　　    “恒晟地产的产业做的很大，发展很迅猛，从他们往年的财务报表上看，这家公司是个很难得的优质性企业，至少他们在不良负债率上面控制的非常好，银行的贷款没有超过企业的盈利，并且每年都有结余，总得来说就是这是一家没有欠钱的企业……”<br/>　　<br/>　　    张泽雨这个人很聪明，他似乎看出王惊蛰对商业方面的问题一窍不通，于是就尽量的把话语叙述的通俗易懂了些，基本上控制在专科以下的人都能听懂的范围，但他显然低估了王惊蛰小本毕业的实力，其中提了不少让他非常懵逼的问题，到最后张泽雨解释的都要崩溃了，他都有点搞不清楚范家的人，是怎么灵光一闪，才让他们过来听令于王惊蛰的，不过好在这帮人的素质非常高，尽管王惊蛰颇为小白了一些，可至少在表面上他们没有露出任何不满和厌烦的表情。<br/>　　<br/>　　    关于徐少中的公司，介绍完了以后，其实王惊蛰在心里已经有点概念了，那就是这家公司是一家大大的良民企业，好像一张没有挥笔泼墨过的白纸一样，干净的令人发指了。<br/>　　<br/>　　    于是，王惊蛰最后提了个很有意思的问道：“你们觉得，这么干净，透明的公司正常么？我虽然不懂商业，但我知道有句话叫无商不奸，不管是多大的企业，还是贩夫干的小本买卖，只要是做生意，就会掉进这个大染缸里，不存在干净到令人发指的可能，我这么说对嘛？”<br/>　　<br/>　　    银行的财务官董良笑了：“王先生说的很对，这一点绝对没有任何可能，所有的商人，生意做的越大那就会问题越多，而这家恒晟地产表现出的可能是一种假象，就是做给别人看的”<br/>　　<br/>　　    “啪”王惊蛰拍了拍手，点头说道：“完美，只要有问题就行了。”
