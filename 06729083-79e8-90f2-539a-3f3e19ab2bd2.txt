        “我没给过你时间吗？”童筱颖冷冷地嗤笑，回眸之际，哀怨地瞪向傅昼景，怒吼道，“我愿意接受小尚轩，愿意陪你打官司争夺小尚轩的抚养权，愿意当众信誓旦旦地说‘我爱你’，甚至愿意脱光了躺床上跟你生个孩子去救小尚轩。你在医院里日日夜夜照顾季思妍的时候，我在家日日夜夜地照顾小尚轩，我图什么？可是，你呢？你最后仍旧选择了季思妍，不是吗？”
<br/>
<br/>        “筱颍，我觉得你比季思妍更坚强、更懂我，所以我……”
<br/>
<br/>        “我没有你想象中的那么坚强，也没有你想象中的那么懂你！你和季思妍不是要结婚了吗？恭喜你啊！一家三口，终于团聚了！”童筱颖冷不丁地打断了傅昼景欲要说的话。
<br/>
<br/>        没错，他跟她离婚后，她确实仍旧对他还抱有一丝幻想。
<br/>
<br/>        而这最后一丝幻想，彻底被季思妍的那封电子邮件给打破了。
<br/>
<br/>        傅昼景顿时有些急红了眼：“筱颍，我和季思妍是假结婚，只是为了安抚她。她的抑郁症，已经很严重了。”
<br/>
<br/>        “……”童筱颖忽然间沉默了，只是低着头，脸上没有任何表情。
<br/>
<br/>        “筱颍，‘死生契阔，与子成说。执子之手，与子偕老。’，三年前我确实做了对不起你的事情，但是这三年来，我的心里只有你，现在是、将来也是！”傅昼景目光中透着忧伤，真诚地诉说。
<br/>
<br/>        童筱颖深吸了口气，抬头对上傅昼景的目光，冷冷一笑：“死生契阔，与子成说。执子之手，与子偕老。这句话是小卫对我说的，但你已经不再是我的小卫了。”
<br/>
<br/>        “筱颍，你不要这样子，好吗？我……”
<br/>
<br/>        “亲爱的，进屋里去给我擦擦药，好不好？我的眼睛很痛！”夏凌焰见傅昼景不依不饶，立即拉着童筱颖往卧室里走。
<br/>
<br/>        傅昼景上前追上一步时，却只是迎面传来一道关门的冷风。
<br/>
<br/>        “筱颍！”
<br/>
<br/>        “你走吧！”童筱颖回绝道。
<br/>
<br/>        “筱颍，我知道，你只是随便找个男人来气我的而已。你相信我，好不好？不要这样对我！”傅昼景拍了拍门板。
<br/>
<br/>        但隔着一张门板，里面突然间安静了。
<br/>
<br/>        傅昼景聋拉着脑袋，心情复杂地收回了手。
<br/>
<br/>        他把筱颍从自己的妻子变成了前妻，以前，她在他的怀里，今后，她在别的男人怀里。
<br/>
<br/>        这种事情，他怎么可能容忍？
<br/>
<br/>        卧室里，童筱颖不得不去内卫里拧了一条冷毛巾，给夏凌焰敷眼睛。
<br/>
<br/>        夏凌焰坐在床边，一手接过童筱颖递过来的毛巾敷眼睛，一边忍不住小声地谆谆教诲道：“小筱颖，我看得出，你对你前夫仍旧还有爱。但是，你这个事，我觉得吧！你就应该跟你前夫划清界限，哪怕你爱他，你也不能跟他在一起了。事实摆在眼前：第一，他跟小三有了孩子，你跟他又没孩子；第二，你现在二婚了，哪怕你不爱你的二婚老公，但也不能跟前夫藕断丝连，对不起你这位二婚老公！你说是不是？”
<br/>
<br/>        “你是让我认命吗？”童筱颖黯然神伤地问。
<br/>
<br/>        夏凌焰点了点头，随即又摇了摇头：“这不叫认命，而叫认清事实。你为爱也付出了很多，是你前夫不懂得珍惜你。”
<br/>
<br/>        “可他现在所做的，不就是在挽回我吗？”
<br/>
<br/>        “那你会回到他身边去吗？”
<br/>
<br/>        “我不知道……”
<br/>
<br/>        或许，真像傅昼景所说的那样，三年前他是对不起她，但是在澳洲和她婚后的这三年里，他没有做对不起她的事情。
<br/>
<br/>        只是，他们之间，突然插进来一个季思妍，破坏了他们的婚姻。
<br/>
<br/>        “有一种爱，叫放手。小筱颖，放手吧！这样，你也能得到解脱啊！”夏凌焰啧啧嘴说。
<br/>
<br/>        童筱颖此刻沉默了。
<br/>
<br/>        她和小卫自幼相识，长大后再次相遇结为夫妻。
<br/>
<br/>        这不仅仅只是爱，还有命中注定的缘分。
<br/>
<br/>        明明劝诫过自己要忘记，但是再次看到小卫的时候，她仍旧无法洒脱。
<br/>
<br/>        “我有个法子，可以让你的前夫对不再纠缠！”夏凌焰突然站起身来，咧着嘴打了个响指。
<br/>
<br/>        童筱颖惊怔地看着夏凌焰，只见他从他的行李箱里拿出了平板电脑，然后他又点开了一段视频。
<br/>
<br/>        屏幕上，纠缠着两具白花花的身体。
<br/>
<br/>        “嗯——啊——啊——嗯——老公——我还要——”
<br/>
<br/>        夏凌焰把平板电脑的声音调到了最大。
<br/>
<br/>        童筱颖顿时黑脸了。
<br/>
<br/>        “别这样看着我啊！哪个男人没有自己的私藏？你回去搜搜你二婚老公的电脑、平板、手机，绝对有私藏的！”夏凌焰勾唇一笑。
