　　繁荣的街道上，小医仙乖巧的任慕清霄牵着手，美眸温柔的望着他的英俊侧脸，虽然相识只有几天，但他却已经成为自己的依靠。
<br/>　　相遇相知，前人洞府中的他是那般温柔，整个洞府中的东西，只是取走了几枚价值不高的龙阳果......
<br/>　　小医仙望着他的侧脸，不禁有些痴了，世间怎会有如此完美的男子，而且还是自己人生中的另一半，简直就像做梦一样。
<br/>　　两人走在繁荣的街道上，这里瞧瞧那里瞧瞧，小医仙也是极其轻松，见她美眸中的开心之色，慕清霄脸上也挂起一丝温柔笑容。
<br/>　　半个时辰后，玩的差不多，慕清霄道：“对了，黑岩城正好有炼药师公会，我准备去考核，对我们有利无弊。”
<br/>　　炼药师是斗气大陆中的一种职业，等级分为一至九品，每品分为上、中、下三等。
<br/>　　炼药师能炼制各种神奇丹药，身份地位在斗气大陆显赫之极，且拥有无比强大的号召力，因为不管是谁能望炼药师可以欠他们一个人情。
<br/>　　慕清霄看中的并不是号召力，而是通过考核成为炼药师后，能够在公会的帮助下，寻找自己需要的灵药。
<br/>　　如此一来，自己就不必东奔西跑了。
<br/>　　何况，小医仙修炼的七彩毒经，其中所需要的特殊灵药不少，若是在路上寻找，无疑会耽搁许多时间。
<br/>　　但是通过考核，成为炼药师就不同了，公会在各地区都有分舵，而在公会的帮助下，寻找起各类药材无疑方便很多。
<br/>　　得知慕清霄是炼药师，小医仙也见怪不怪，相比起炼药师的身份，她更吃惊于后者的恐怖实力。
<br/>　　如此年轻的斗皇，哪怕她在孤落寡闻，也知晓资质有何等恐怖，毕竟她才一星斗者啊！
<br/>　　炼药师公会在黑岩城可谓是鼎鼎大名，在路上随便找个行人问一下，就知道了大致的位置。
<br/>　　顺着路人所指的路线，慕清霄带着小医仙走过几条繁华的街道，足足半刻钟才停下脚步，抬头望着眼中，有些大气磅礴的建筑。
<br/>　　映入眼中的高楼，外形颇为别致，或者说有些另类，整体看上去就如同一座药鼎，在建筑的周围，还设置了犹如药鼎通火口一样的窗户。
<br/>　　高高的房顶上，硕大的鼎盖覆盖其上，将下面的空间都遮掩住。
<br/>　　目光在建筑物外，紫檀色的檀香牌匾上，五个自己隐隐有些模糊的古朴字体，闪烁着淡淡的光芒。
<br/>　　“炼药师公会。”
<br/>　　慕清霄看了眼四周，炼药师公会的人流本就少，由于是夜晚的关系，就显得更加冷清。
<br/>　　除了附近街上有一些行人，以及门口的四名彪悍护卫，内部基本看不到人影。
<br/>　　拉着小医仙向公会中走去，两名彪悍的护卫就走了上来，伸手便就他们拦了下来，面无表情，道：“小兄弟，有自己导师的介绍信吗？”
<br/>　　闻言，慕清霄伸出手来，只见掌心窜出黑色的火焰，火焰没有一丝的温度，却使得护卫骤然色变。
<br/>　　“凝聚实火，四品炼药师？！”两名护卫惊骇的说道，就连他们身后两名护卫，眼中也尽是不可思议。
<br/>　　他们在炼药师公会工作，自然是见多识广，同时也知道，凝聚实火，至少需要四品炼药师才有可能办到。
<br/>　　当然，如果面前的是一位行将就木老者，他们倒还能保持镇定，可面前的青年，分明只是一位二十岁左右的青年，这特么就很恐怖了！
<br/>　　“我们可以进去了吗？”望着处于震惊中的护卫，慕清霄若无其事的说道。
<br/>　　“当......当然可以的，您快请进，请进！”
<br/>　　瞬间，护卫们的态度三百六十度的转变，赶紧卑微的弯下身子，看向慕清霄的眼神充满了敬畏！
<br/>　　慕清霄微微点头，随后向炼药师公会内部走去，小医仙紧随其后，美眸扑闪扑闪的，仿佛走在前面的青年，身上笼罩着一层神秘的面纱。
<br/>　　待慕清霄离开，护卫回过神来，面面相觑，都能看到对方眼中的惊骇之色！
<br/>　　“最起码四品炼药师啊！不会是驻颜的老妖怪吧？”
<br/>　　“别嘴欠，你不想在这里工作，可别连累我们，赶快将此事禀报上去吧，否则，我们都吃不了兜着走！”
<br/>　　......
<br/>　　......
<br/>　　前脚刚踏进炼药师公会，下一秒就有一股淡淡的药香迎面而来，使得精神有些心旷神怡。
<br/>　　大厅中的人员并不多，只有寥寥数人各自做着自己的工作，似乎是听见了脚步声，一齐抬头望了过来。
<br/>　　看到慕清霄与小医仙的容貌后，目光都有些异样，实在是两人容貌都太完美了，女的如此也罢了，男的居然也如此！
<br/>　　站在清冷的大厅中，刚想开口咨询，前面就有一名身着紫色衣裙的女子，急忙从后台中走了出来，然后向着他们走来。
<br/>　　“这位......先生，姑娘，请问两位这么晚来公会，是有什么事情需要帮助吗？看两位有些面生，应该是第一次来黑岩城炼药师公会吧？”
<br/>　　面带职业性的微笑，紫裙女子美眸望着慕清霄，视线又落在小医仙身上，眼底却是闪过一丝惊愕，暗道：“好一对金童玉女。”
<br/>　　望着紫裙女子明眸皓齿的模样，倒是惹人喜爱，但比起倾国倾城的小医仙，却是差了不止一个档次。
<br/>　　“确实是第一次来这里，我是想领一枚炼药师的徽章。”
<br/>　　闻言，紫裙女子明显怔了怔，但也仅此而已，更年轻的炼药师，她也不是没见过，但语气却颇为客气：“没想到，您也是一名尊贵的炼药师啊？”
<br/>　　“尊贵谈不上，尽量速度快一些。”
<br/>　　“请您跟我来登记一下。”
<br/>　　话语落下，紫裙女子也简单确认了他是炼药师的身份，俏脸上也多出一丝恭敬，退后几步就向柜台走去。
<br/>　　自柜台中拿出一张泛黄的古朴羊皮纸，紫裙女子素手优雅的握着笔，冲着慕清霄盈盈一笑，道：“先生，请问您尊姓大名，年龄，导师的名字？”
<br/>　　“慕清霄，年龄二十，导师‘药老’。”慕清霄脸不红心不跳，谎报出自己的年龄。
