　　第380章
<br/>　　清醒过来的樊梨花和程大雷，面对的是屋内一片狼藉。
<br/>　　桌子塌架，杯盘碎成一片，二人的衣物，包括床上的被褥都散落在地上，有些甚至都被扯碎，地上湿漉漉一片，漂着酒香。
<br/>　　想起刚才发生的事，樊梨花羞得无地自容。同时抬起头，悄悄打量着程大雷。本想着程大雷能开口说上几句话，可见他坐在地上，呆呆发愣，也不知想着什么。
<br/>　　“喂……”樊梨花轻轻唤了一声。
<br/>　　程大雷恍然回过神来，刚才他正检查着脑海中的城池建造图纸，倒是有些出神。此刻抬起头来，见樊梨花香肩外露，表情含羞带怯，不由又有几分情动。他小心翼翼凑过去，搭住樊梨花的手，樊梨花羞得低下头去，却没有反抗。
<br/>　　心慌意乱，程大雷恍然想起什么，郑重其事道：“这次不许再弄伤我了。”
<br/>　　“嗯，我会很小力的。”
<br/>　　咦，又有那些不对。
<br/>　　……
<br/>　　次日天明，响晴薄日。
<br/>　　大家都看到程大雷手上缠着绷带，却也不知发生了什么事。问，程大雷也是缄口不言，绝不回答。
<br/>　　程大雷窝在书房，研究系统奖励的城池建设图纸。
<br/>　　目前的琴川关已经到四级，再往下发展就该转型成为一座城池。也就是说，五级山寨相当于一级城池。
<br/>　　城池和山寨还是有不同的。
<br/>　　总的说来，山寨是座军事堡垒，一切都是为战争服务。而城池不仅有军队，还有许多生活设施。
<br/>　　程大雷查看图纸，了解从四级山寨到一级城池，具体要求什么。
<br/>　　首先人口要求达到10万，随着流民不断加入，再加上山寨原先的人口，目前山寨的总人口大概在五万左右，距离目标也不远。
<br/>　　军策府一座，研究战略、阵法、外交。
<br/>　　内政府一座，研究生产、种植、城内治安等等。
<br/>　　然后市场、兵营、点将台、医馆、工匠坊、学堂等设施，是琴川关原先就有的，只要稍作改造，就可以达到要求。
<br/>　　引起程大雷注意的是其中一样建筑：钱莊。
<br/>　　钱莊大概功能类似于银行，但和银行也有很大不同。钱莊的主要功能是金银兑换，把黄金兑换成白银，白银兑换成铜板，钱莊收取一定比例的费用。以前的大型钱莊还发一些银票，只不过如今兵荒马乱的，没有哪家财团支撑得起这样的信用，到现在银票几乎和白纸差不多。
<br/>　　所以，程大雷拿着一百万两银票行走天下的事情一直没有发生，一百万两面额的银票是有的，只不过，没有哪家钱莊会兑给你。
<br/>　　程大雷想了想，差人把和珅叫到书房，简单把自己的目的说了。
<br/>　　“大当家准备铸币？”和珅问。
<br/>　　程大雷点点头，如今琴川关已经有不少人口，人口的汇聚促成商业的发展。偶尔有人买卖布匹、猪肉之类的生活用品。但商业模式还很初级，大多是以物易物。
<br/>　　帝国的金银本就属于稀缺资源，莫说琴川关，整个帝国都没有多少。而铜板之类的物事，因为太过沉重，使用起来并不方便。
<br/>　　帝国的货币并不是统一的，自然金银是通兑的，可铜钱各地诸侯都有私铸。所以对程大雷的想法，和珅并不觉得奇怪。现在琴川关已经成了大家的老巢，还不知道还要在这里多久，铸币也是自然而然的事情。
<br/>　　“大当家打算铸什么样式，要不要自己定个图案？”和珅问道以他对程大雷的了解，估计程大雷会在铜钱上印个蛤蟆。
<br/>　　“不是铸，是印，我想要印纸币。”
<br/>　　“呃……”和珅愣了愣，有些不太理解程大雷话。关键是『纸币』这两个字，他是第一次听到。
<br/>　　程大雷早有这个想法，目前帝国的货币制度其实极其落后。金银的购买力是有的，可使用时又要剪又要称，铜板的质量又太过沉重，甚至还没有绸缎好使。
<br/>　　和珅听罢之后，想了好大一会儿，及到把程大雷的话完全消化。
<br/>　　“大当家，你的想法是好的，但估计实施起来，有些困难。”
<br/>　　“有什么困难，我们可以建造一座印钞厂，确定面额、编号、防伪，慢慢推行开去，应该不太困难。”
<br/>　　“可大家用惯了金银，在大家心中，只要看得见摸得着的东西，才有价值。你告诉他们一张纸就可以买东西，怕一时半会他们不认。”
<br/>　　程大雷点点头，心里也明白和珅说得不错，固有的观念不是一时片刻就可以改变的。但这并不代表程大雷这个法子不可行，只是有些困难而已。
<br/>　　“还是先把印钞厂建起来，怎么往下推，慢慢再想办法。”
<br/>　　在琴川关，程大雷还是有绝对权力的。虽然觉得程大雷有些异想天开，但和珅也没有不答应的道理。他心里暗自琢磨着，随便印点铜钱不就可以了么，何必需要如此麻烦。
<br/>　　另外，程大雷也安排了几件事。趁着现在农闲，从各地募集流民，只有更多的人迁入，山寨才能发展起来。
<br/>　　另外让徐神机和刘悲选拔人才，筹建军策府和内政府。这两样建筑和城墙不一样，不是放在那里就可以使用，没有做事的人，也只是一个空壳子而已。
<br/>　　程大雷自己负责印钞的事情，图案很好办，无非是在上面画个蛤蟆，面额也好确定，一文、二文、五文、十文、百文。最难的事情是防伪……
<br/>　　假若蛤蟆寨发行的纸币可以推行开，到最后整个琴川关都在使用，自然会引来人仿制，伪造。如果随随便便是个人就能印出来，程大雷的成果岂不都被人毁了。而且，防伪的成本也不可太高，如果印钞的成本太高，对如今的山寨也是巨大消耗。
<br/>　　又要便宜，又要好用……对于这件事，蛤蟆寨原有的人都没有法子。
<br/>　　程大雷将注意打到了奥黛琳娜身上。她带来一批人，有冒险家、炼金术师、药剂师……说不得这些人会有法子。
<br/>　　程大雷派出人去，把这些人请到了将军府，向他们说明自己的目的。
<br/>　　
<br/>(本章完)
