　　苏莫在山林中继续等待，同时他心中暗暗沉声。
<br/>　　这次天冥星之行，一定要给予巫族血的教训，不然不足以平他心中的怒气，不足以慰藉苍穹世界千亿亡灵。
<br/>　　他生于苍穹世界，长于苍穹世界，苍穹世界的毁灭，对于他来说，就是家乡被毁的血海深仇。
<br/>　　轰！轰！轰！
<br/>　　就在此时，苏莫的耳中突然听到了阵阵的轰鸣之声，这些轰鸣声极小，若不是仔细聆听，根本无法差距。
<br/>　　“有人在战斗？”苏莫闻听此声，顿时双眉一跳。
<br/>　　略一沉吟，他立刻循声飞掠了过去，片刻之后，他来到了山脉的中央。
<br/>　　抬眼望去，他见到在山脉上空，有两人在战斗。
<br/>　　这两人是两名巫族的青年男子，一人仪表堂堂，长的俊逸非凡，另外一人则是身形瘦小，面目几位普通。
<br/>　　俊逸非凡的青年修为较高，达到了武帝境四重的境界，而那名瘦小的青年，修为则略差一些，只有武帝境三重巅峰。
<br/>　　两人的战况很是激烈，恐怖的玄力波动，让得下方的山林不断的湮灭。
<br/>　　不过，俊逸青年实力很强，明显的将瘦小青年压制，每一招每一式都将瘦小青年击的连连后退，险象环生。
<br/>　　而在两人不远处，还伫立着一名绿衣女子，在冷眼观战。
<br/>　　这名女子同样是巫族，容颜俏丽，姿色不俗，修为也达到了武帝境四重境界。
<br/>　　苏莫藏身的远处，仔细观看，巫族之人自相残杀，他倒是非常乐意见到。
<br/>　　这三人倒是很适合他夺舍，只不过不知道是不是黑耀城的巫族，而且三人修为都不低，他短时间怕是很难拿下。
<br/>　　若是闹出大动静，就麻烦了。
<br/>　　苏莫没有出手，只是静静的观看。
<br/>　　轰！
<br/>　　这时，一声爆响声响起，俊逸青年一拳将瘦小青年从空中轰了下来，让得对方直接砸在了山林中，溅起漫天尘土。
<br/>　　“噗！”瘦小青年顿时身受重伤，口中鲜血狂喷！
<br/>　　“哈哈哈！！巫尘，你就是个废物，居然还想跟我争夺玉儿，简直是自寻死路！”
<br/>　　俊逸青年傲立虚空，满脸不屑的望着下方口吐鲜血的瘦小青年。
<br/>　　“巫烈、巫玉儿，你们这对狗男女，不会有好下场的！”瘦小青年巫尘脸色苍白，阴狠嗜血的目光扫视俊逸青年和不远处的绿衣女子，放声怒喝。
<br/>　　他恨！
<br/>　　非常的恨！
<br/>　　自己的未婚妻，居然与别人苟.合，而且还联合别人来杀他，这让他几乎疯狂了。
<br/>　　“我们有没有好下场，你根本看不到了！”俊逸青年巫烈冷笑一声，拳头之上玄力涌动。
<br/>　　嗖！
<br/>　　这时，那名绿衣女子飞了过来，和巫烈一前一后对巫尘形成了夹击之势。
<br/>　　两名武帝境四重武者，夹击一名重伤的武帝境三重武者，任巫尘实力再强，也是插翅难逃了。
<br/>　　“玉儿，为什么？”巫尘目光凝视着绿衣女子，不甘的吼道。
<br/>　　他和对方自幼定亲，数十年来，两人恩爱有加，他为对方默默付出，他实在想不明白，对方为何会对他如此狠毒！
<br/>　　“你连我都不如，如何能配的上我？”巫玉儿脸色冷漠的说道。
<br/>　　“呵呵！我连你都不如？”巫尘闻言，顿时冷笑了起来，他的天赋一直都比对方高，若不是他经常将自己的修炼资源送给对方，对方的修为如何会超越他！
<br/>　　现在，对方修为超越了他，居然就嫌弃他修为低了，真是可笑！
<br/>　　“就因为我的修为不如你，你就要杀我吗？我看是因为他吧？”巫尘阴狠的目光望向了巫烈。
<br/>　　巫玉儿闻言沉默不语，眸中的杀机爆射而出，对方必须要死，不然后患无穷。
<br/>　　“哈哈哈！！”
<br/>　　这时，巫烈大笑了起来，满脸得意之色，道：“巫尘，就凭你也配和我比吗？论修为、身份、相貌，你哪一点能和我相提并论？玉儿选择我乃是明智之举。”
<br/>　　“你们这对狗男女，我杀了你们！”巫尘怒喝一声，双掌齐出，狠狠的击向巫烈和巫玉儿。
<br/>　　而巫尘的身形，则是化为一道流光，直接向远处飞逃而去。
<br/>　　他并不傻，留下来只有死路一条，只有立刻逃走，才有活命的希望。
<br/>　　巧合的是，巫尘逃跑的方向，正是苏莫藏身的方向。
<br/>　　“你逃不掉！”
<br/>　　巫烈冷笑一声，同样双拳齐出，一拳轻易的轰碎了巫尘的掌印，另一拳直接向巫尘镇杀而去。
<br/>　　这一拳威力强大无比，远超之前，拳印如同一座大山，裹挟毁天灭地之势，直击巫尘。
<br/>　　显然，之前他并没有用出全力，现在才展现了真正的实力。
<br/>　　“不！”面对巫烈这必杀的一拳，巫尘根本抵挡不住，瞬间便被击中。
<br/>　　轰！
<br/>　　一声爆响，如山般的拳印炸裂，化为汹涌的乱流横扫八方。
<br/>　　啊！
<br/>　　一声凄厉的惨叫声响起，巫尘的身躯仿佛一个破麻袋一般，向远处抛飞而去，而后砸落在了山林之中。
<br/>　　嗖！嗖！
<br/>　　随即，巫烈和巫玉儿飞了过来。
<br/>　　巫尘的掌印无法伤到巫烈，自然也无法伤到巫玉儿。
<br/>　　两人身形伫立在山林上空，脸色冷漠的望着下方全身鲜血，没有任何生命气息的巫尘。
<br/>　　“已经死了！”巫玉儿松了口气。
<br/>　　“他的战魂应该没有逃掉吧？”巫烈皱眉问道。
<br/>　　“没有，已经湮灭在了拳劲之中！”
<br/>　　巫玉儿摇了摇头道，刚才在巫尘肉身被击中之后，他看到了巫尘的战魂逃了出来，但紧接着便消失在了拳劲之中，定然被彻底湮灭了。
<br/>　　“那就好！”
<br/>　　巫烈点了点头，随即扫视了一番四周，道：“此地不宜久留，我们走吧！”
<br/>　　“嗯！”
<br/>　　而后，巫烈和巫玉儿两人，便联袂离去，现场恢复了平静。
<br/>　　少倾之后，一道人影闪烁而出，站在了巫尘的尸体身边。
<br/>　　人影不是别人，正是苏莫。
<br/>　　苏莫目光望着巫尘残破不堪的身体，面露沉吟之色，而后，他手掌一挥，一只虚幻的妖豹出现在了他的身前。
<br/>　　这只妖豹体长两丈有余，乃是一只妖豹战魂。
<br/>　　只不过，这妖豹战魂身体虚幻到了极致，到了即将消散的边缘。
<br/>　　这妖豹战魂，正是巫尘的战魂。
<br/>　　适才，妖豹战魂从巫尘的肉身中逃了出来，但根本挡不住巫烈爆炸的拳劲，在即将被毁灭之时，苏莫用吞噬之力瞬间将其擒拿了过来。
<br/>　　这也是他距离较近，比较方便动手，若是距离再远几分，他也无法得手了。
<br/>　　“你……你是谁？”妖豹战魂口吐人言，虚弱无比，正是巫尘的声音。
<br/>　　妖豹战魂无法逃走，一是太过虚弱，二是苏莫的手中有玄力吞吐，将他牢牢的束缚住了。
<br/>　　苏莫没有回答对方，而是意念一动，再次展开了搜魂术，对巫尘的战魂进行搜魂。
<br/>　　战魂中蕴含巫尘的神魂，自然也可以搜魂。
<br/>　　在苏莫的搜魂术之下，妖豹战魂剧烈的颤抖了起来，片刻之后便随风消散。
<br/>　　本就虚弱到极致的妖豹战魂，再承受了苏莫的搜魂，终于油尽灯枯。
<br/>　　苏莫身形伫立不动，眸光却是闪亮无比。
<br/>　　根据他的搜魂，这巫尘正是黑耀城的巫族之人，而且还认识巫天御，非常适合他夺舍肉身。
<br/>　　只不过，他也有些迟疑，因为此人在巫族并不受人待见，还有巫烈和巫玉儿这种敌人，他若是占据了此人的肉身，以此人的身份进入黑耀城，怕是变数太大了。
<br/>　　苏莫陷入了两难之中，如此巫族之人的肉身就在眼前，他却有些拿不定注意了。
<br/>　　足足思索了一刻钟，苏莫咬了咬牙，决定就夺舍此人了。
<br/>　　虽然此人在巫族有敌人，但他的身份只要不暴露，在黑耀城内没有人敢光明正大杀他，巫族的族规还是非常严苛的。
<br/>　　而因为不受待见，此人在巫族也是极不起眼，反而方便他行动。
<br/>　　拿定了注意，苏莫立刻盘膝而坐，背后虚影升腾，吞噬战魂瞬间释放而出，直接钻入了巫尘的体内。
<br/>　　因为巫尘已经死亡，战魂也彻底消散，所以，苏莫根本不用夺舍，直接就能占据了对方的肉身。
