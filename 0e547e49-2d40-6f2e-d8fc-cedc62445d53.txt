<br/>　　“苏兄，他们的修为并不是太高，都和你相差无几！”谢彬不甘心的再次说了一句。

<br/>　　最终，谢彬只能无奈的离开了。

<br/>　　这不仅是丢了他真圣榜第一的名头，还有损神武学府的威名。

<br/>　　巫人王阴沉着脸说道，他郁闷不已，苏莫死活不出来，看来他只能让族中祭司出手了。

<br/>　　既然苏莫不出来，他们能有什么办法，只能无奈离开了，为真圣界之行做准备。

<br/>　　外府，一座阁楼中，尤天韩伫立在窗前，微微皱眉。

<br/>　　足足耗费了三个多月的时间，苏莫才将《三千炎焱诀》修炼到第一层的巅峰，而《生生不息诀》才刚刚入门而已。

<br/>　　苏莫现在有些后悔了，后悔没有早做准备，他之前就应该留下几个分身，就算本体死了，也还有分身在，不会彻底死亡。
