　　“我、我错了……”
<br/>　　被罗昊报复性的狠狠折腾了两次，云熙已经累得快连说话的力气都没有了。
<br/>　　是谁说的亚洲男人夫妻生活的平均时间只有三分钟的？是谁说十分钟的夫妻生活已经是超长发挥了？
<br/>　　云熙现在只想把这个得出这番结论的家伙，揪出来好好揍一顿，打得他满脸桃花开。什么歪理邪论，老娘都快要被累死了好么？
<br/>　　“说，知道错了吗？”罗昊故意板起一张脸，故作凶狠对着云熙问道。
<br/>　　“错、错了！”云熙可怜巴巴的回答道。
<br/>　　“说，错在哪儿了？”罗昊又问道。
<br/>　　“不该跟哈普雷见面，见了面也不该瞒着你。”云熙可怜兮兮的看着罗昊，说道：“我真的知道错了，你就饶了我吧，要不然明天走不了路，两个小家伙问起来我都不知道要怎么跟他们解释。”
<br/>　　“看你认错态度还不错，这次就放过你，下不为例！”罗昊点着云熙的鼻子警告道。
<br/>　　云熙忙不迭的点头。
<br/>　　罗昊嘴上虽然这么说，可是实际行动却跟他说得并不一样，身体往下一沉，又一次进入到云熙身体中……
<br/>　　“你这个臭流氓！你又骗我！”
<br/>　　云熙用拳头拍打着罗昊，但是很快，云熙的抗议声就变成了断续彼伏的喘息和呻吟声……
<br/>　　一直折腾到后半夜，罗昊才抱着云熙去了浴室清理干净身体回到卧室休息。
<br/>　　……
<br/>　　云熙原来的生物钟很准时，到点就会准时醒来，给两个小家伙准备早饭，可是昨天晚上被罗昊折腾了大半夜，早上自然就睡过了头。
<br/>　　姐弟俩走到云熙和罗昊的卧室门口，看着紧闭的卧室门，云悠皱了皱眉头，说道：“妈妈平时都很准时起床了，今天怎么睡懒觉了？”
<br/>　　“笨蛋！”罗战白了自己姐姐一眼，说道：“因为我们的老爸回来了呗！”
<br/>　　“哦！”
<br/>　　云悠若有所思的点了点头，说道：“那我们早饭吃什么呀？”
<br/>　　“冰箱里有面包和牛奶。”
<br/>　　罗战和云悠两个人蹬蹬蹬跑下楼，打开冰箱去寻找他们的早餐去了。用昨天在超市买的面包和牛奶凑合了一顿，两个小鬼又开始去忙自己的事情去了。
<br/>　　直到快临近中午了，还没看到罗昊和云熙下楼，姐弟俩顿时有点急了，这早上吃了面包和牛奶，中午总不能再吃一顿吧？可是姐弟俩自己又不会做饭，要怎么解决中午饭倒是愁坏了两个小家伙。
<br/>　　云悠大眼睛眨了两下，拉了拉弟弟罗战，说道：“我们去找干爹吧？去他家里吃，如果他家里也没午饭，就让他带我们出去吃。”
<br/>　　“好主意！”
<br/>　　罗战眼睛顿时一亮，刺溜一下就从沙发上蹿了下来，跑到玄关换鞋去了，而云悠则是拿起罗战对下了铅笔，在本子上写了几个字，才离开别墅。
<br/>　　姐弟俩对唐磊家那是熟门熟路，以前没有少来这里玩。
<br/>　　罗战走到车库门口，通过缝隙往里面看了眼，看到那俩红色的法拉利跑车还在，就知道唐磊还没出门，就跑到门口去按门铃去了。
<br/>　　很快吴忆林就出来开了门，看到站在门口的姐弟俩顿时欣喜万分，弯腰在姐弟俩小脸上捏了捏，问道：“你们这两个没良心的小东西，怎么这么长时间都不来找我玩了？”
<br/>　　“军事机密，不能说！”
<br/>　　罗战板着小脸，一本正经。
<br/>　　吴忆林被逗得“噗嗤”一笑，伸出手指在罗战额头上轻轻戳了下，笑着说道：“就你还军事机密，你那点破事儿我还不知道，不就是被扔到部队里面训练去了吗？”
<br/>　　吴忆林让姐弟俩进屋后，问道：“你们怎么独自跑到这里来了，你们爸妈呢？”
<br/>　　“别说了。”罗战一脸苦恼的说道：“我家老头子回来了，亲爱的妈妈因此睡了懒觉到现在还没起来呢，我们没中午饭吃了，所以就跑来这里了，我觉得我一定是有个假爸爸，他不回来妈妈都会准备好一切，他一回来，妈妈就把她的心肝和宝贝给忘了。”
<br/>　　吴忆林听到罗战的话，自然明白了他指的是什么，脸色微红，放下水杯，对姐弟俩问道：“中午你们就在我这里吃吧，说吧，想吃什么？肯德基还是麦当劳？”
<br/>　　“我妈说了，那是垃圾食品，小孩子不能吃会变丑的。”云悠嘟着小嘴说道：“除了油炸的，什么都可以，我不挑食。”
<br/>　　“哟，没看出来云熙姐教的这么好。”吴忆林摸了摸云悠的小脑袋，转身去厨房，“那行，一会儿我就给你们做点健康食品。”
<br/>　　“昊子失踪那几年，这姐弟俩都是云熙一手带大的，再加上他们妈妈这么聪明，教出来这姐弟俩能差到哪儿去？”
<br/>　　唐磊一边扣着扣子一边从楼梯走下来。
<br/>　　罗战看着头发乱糟糟的唐磊，说道：“干爹，原来你也跟我家老头子一样在睡懒觉！”
<br/>　　唐磊闻言，顿时老脸一红。
<br/>　　“一会儿给罗昊和云熙打个电话，告诉云熙姐，她的心肝和宝贝在我们这儿，不然看到这姐弟俩不见了，云熙姐该着急了。”吴忆林在厨房里对唐磊说道。
<br/>　　唐磊点头应了一声。
<br/>　　而此时，罗昊和云熙也是刚醒。云熙睁眼一看床头柜闹钟上的时间，轻呼一声“糟了”赶紧从床上爬起来，可是刚一动，两腿间的不适就令得她忍不住皱起眉头。
<br/>　　几分钟后，云熙皱着眉头从马桶上站起身来，一瘸一拐的从浴室里走出来。
<br/>　　“真这么疼啊？”罗昊看着走路样子奇怪的云熙，说道：“又不是第一次了，不至于吧？”
<br/>　　“拿一根棍子对着你捅一晚上，看你还能说话这么轻松！”云熙一瘸一拐的走到床边，搀着腰坐下，瞪了眼还躺在床上罗昊，伸手在他身上拍打，气恼的说道：“结婚都这么多年了，怎么还这么野蛮，疼死我了。”
<br/>　　“要不要紧？”
<br/>　　看到云熙始终皱着眉头，罗昊脸上也没有之前那么轻松了，立刻从床上爬了起来，询问起云熙的情况来，看云熙的样子不像是假装出来的。
