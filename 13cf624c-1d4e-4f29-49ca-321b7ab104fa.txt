　　“是。”
<br/>　　荒古门弟子缓缓起身，恭敬的回答一声，转身离开。
<br/>　　“哼，一个北域的小子，我看你如何来跟我争？一个时辰的时间，我已经得到了一万两千枚羽化晶，就算是猎杀的再快，也不可能比得过我。”
<br/>　　荒圣君一边开口道，一边他身上的逐渐出现了阵阵白雾。
<br/>　　他整个人，都是笼罩在了这白雾之中去，身上的气息，在疯狂的攀升这。
<br/>　　突然间荒圣君整个人的气质，便是比起刚刚，截然不同，有着一种极为古老的感觉。
<br/>　　“荒古大道大成，楚风眠，下一次见到你，便是你的死期！”
<br/>　　荒圣君冷哼一声，便是身形一动，直接离开了原地。
<br/>　　云羽天宫之中。
<br/>　　楚风眠的身影，在其中不断穿行。
<br/>　　连续足足三个时辰，楚风眠便是没有任何的休息，在疯狂的劣势其他武者。
<br/>　　现在他的空戒之中，也已经是足足有着三万枚羽化晶。
<br/>　　随着时间的流逝，这每一座宫殿之中的羽化晶，已经是都被人收走，不少已经都被其他人得到。
<br/>　　所以楚风眠猎杀武者，所得到羽化晶的数量，也已经是越来越少，现在他所碰到的不少武者，都是已经没有羽化晶了。
<br/>　　不是被人抢走了，就是卖给了荒圣君等人了。
<br/>　　这对楚风眠来说，可不是一个好消息。
<br/>　　他要得到足足五万枚羽化晶，才可以得到这云羽天宫的传承，现在他手中的羽化晶的数量，还太少了，还远远不够。
<br/>　　就在楚风眠思考之际，在他的前方，突然之间一道身形，进入到了一座宫殿之中。
<br/>　　“药海？”
<br/>　　楚风眠一眼望过去，便是看得出来，此人，正是药海。
<br/>　　不过由于距离极远，那药海，似乎是没有看到楚风眠。
<br/>　　楚风眠的五感，要远超其他武者，他的战龙血脉，足以改造他身躯的一切，他才看得出来，这来人正是药王谷的药海。
<br/>　　“好，看来这药海，倒是来给我送了一份大礼来了。”
<br/>　　楚风眠目光闪烁，以这药海的实力，在这云羽天宫能够是他对手的武者绝不会多，这药海的手中，肯定会有着不少的羽化晶。
<br/>　　之前在外面，本来楚风眠是想要杀了药海的，不过那个时候药海却是悄悄的躲了起来，令楚风眠都没有发现。
<br/>　　现在正好，这次碰上，正好是顺便可以杀了药海。
<br/>　　“该死，这一座宫殿也被扫空了。”
<br/>　　宫殿之中，药海进入，得到的却是一座空的宫殿。
<br/>　　这宫殿之中的羽化晶，还有这其他的宝藏，如今统统已经是被其他人所收走。
<br/>　　“可恶的楚风眠，若是此人杀了我们药王谷的弟子，我怎么会沦落到这样孤家寡人的地步，还要为荒圣君去卖命。”
<br/>　　药海的眼神之中，露出了几分怨恨。
<br/>　　他药王谷的弟子，如今全部死在了楚风眠的手中，所有弟子都被楚风眠所斩杀，只留下他一人。
<br/>　　不然的话，这云羽天宫的传承，他也不是没有一争的资本，可是现在，他也只能够是为荒圣君卖命。
<br/>　　“该死的小子，等我离开了这归墟天，肯定要杀了他！将他挫骨扬灰！”
<br/>　　药海的眼神中满是杀意。
<br/>　　在这归墟天中，想要对付楚风眠，的确是不好对付。
<br/>　　但是到了九域中，一个小小的楚风眠，对他们七大宗门来说，简直是如同蝼蚁一样，倒时候正是楚风眠的死期。
<br/>　　只有杀了楚风眠，才能够消除他心中的愤怒，他已经是想好计划，一旦是离开归墟天，便是将楚风眠的消息去告诉药王谷。
<br/>　　倒时候药王谷的追杀之下，楚风眠肯定是必死无疑。
<br/>　　“是吗？想要杀了我，将我挫骨扬灰，很可惜，你应该没有这个机会了。”
<br/>　　一道声音，突然从宫殿外响起，只见楚风眠的身影，单独一人，从宫殿之外，走到了里面。
<br/>　　楚风眠的目光中，也尽是杀意，看来这归墟天中的所有七大宗门弟子，全部都要斩杀才行。
<br/>　　不然离开归墟天中，他们将消息传出去，七大宗门的人肯定会追杀楚风眠。
<br/>　　这些人，一个都不能留。
<br/>　　“楚风眠？”
<br/>　　药海看到楚风眠的出现，都是心中一惊，急忙后退，脸色无比忌惮跟震惊。
<br/>　　他也有些想不到，居然是在这里，碰到了楚风眠。
<br/>　　“药海，你不是想要杀了我吗？怎么，我就站在这里，我看你能如何杀了我？”
<br/>　　楚风眠看着药海，嘴角露出了几分讥讽。
<br/>　　“楚风眠，你到底想要做什么？难道真的打算跟我们药王谷为敌吗？”
<br/>　　药海缓和了一下心情，盯着楚风眠，一字一字的说道。
<br/>　　“我知道你要对付的人，是荒圣君，你可能不知道吧，荒圣君已经是达到了碎命之境，他才是你真正的对手。”
<br/>　　“你的实力再强，跟碎命之境相比，也有些差距吧，所以你必须要得到这云羽天宫的传承，得到这云羽天宫的传承，你才能够跟云羽天宫一战的资本，不然若是让荒圣君得到了这一切，那便是你的死期。”

<br/>　　药海看着楚风眠，连连开口道，他巧舌如簧，便是一下子将荒圣君，划到了对立的阵营之中。
<br/>　　要让楚风眠现在意识到，荒圣君才是他真正的敌人。

<br/>　　听完药海的话，楚风眠脸色不变，嘴角微微翘起，露出了几分嘲弄之意道。
<br/>　　“碎命之境，你以为我不知道荒圣君已经是达到了碎命之境？”
<br/>　　楚风眠冷哼一声。
<br/>　　之前碰到荒圣君的时候，楚风眠便是感觉得出，荒圣君的境界，已经是达到了碎命之境。
<br/>　　早在之前寒月漓被追杀的时候，楚风眠的心中便是有了这种猜测。
<br/>　　当时剑祖荒，鹿战神，都在追杀寒月漓，他们二人都想要得到寒月漓的先天本源，一举突破碎命之境。
<br/>　　唯有荒圣君一人，没有追杀，那个时候楚风眠心中便是感觉到，荒圣君不是已经踏入了碎命之境，就是即将踏入，所以才没有来追杀寒月漓。
<br/>　　如今一看，果然如此，荒圣君已经是踏入了碎命之境。
