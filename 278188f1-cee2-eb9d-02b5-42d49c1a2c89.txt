　　几十万斤鱼，放在现代或许没有什么。
<br/>　　但在大唐的那个时代，尤其是生产力低下的靺鞨部落，这简直就是一份惊天之礼。
<br/>　　所以钦陵这家伙最后如愿以偿的抱得美人归，争得黑水部族长的同意，有了这人世界上属于他的一份真爱。
<br/>　　当然，就算是再爱，他也只敢定下娶亲的意向，真正结婚他是不敢的，毕竟军法放在那里，如果他敢阵前结婚，估计不用他回到大唐，砍头的圣旨就能送到他手里。
<br/>　　可是人家钦陵追求真爱又有什么错呢？就算是李承乾在这里也说不出什么不好来。
<br/>　　更何况黑水部的族人可是一力支持钦陵，也不管玉儿答不答应，直接就把钦陵当成了‘自家’女婿，甚至就连军营中的那个大头兵一个个都跟着沾了不少光。
<br/>　　自从鱼被拉回族中以后，黑水部的族人对那支黑甲军队的恐惧开始慢慢减少，时不时也会有一些人来军营周围，甚至还会与执勤的大头兵打招呼，虽然他们得不到什么回应。
<br/>　　……
<br/>　　话分两头，让我们再把目光转回长安。
<br/>　　却说在那大祚荣离开之后，苏玫几女便进了李承乾的书房，围着他七嘴八舌的讨论了起来。
<br/>　　“太子哥哥，东北真的很富裕么？为什么会送来那么多百年老参啊？”
<br/>　　“是啊，殿下，还有那个什么雪蛤膏，以前也是不可多得的东西，结果现在几乎有满满一大车。”
<br/>　　“还有还有，我还发现了好多极品的皮子，紫色的貂皮都有好几件。”
<br/>　　“嗯嗯，纯白的狐皮也有，还有熊皮，虎皮，好多好多。”
<br/>　　如果说一个女人就是五百只鸭子，李承乾此时觉得自己已经被三千只鸭子包围，而且每一只都在对在他叫。
<br/>　　终于，李承乾还是忍不住了，摆手止住几个女人：“好了好了，我说你们几个，不就是一点东北特产么，至不至于这么大惊小怪的。”
<br/>　　“可是它多啊，那么多堆在一起。”程琳两手划了一个大大的圆。
<br/>　　“不是给你们说了么，东北出产这些东西，而且数量绝对超过你们的想象。”李承乾说罢似乎想起了什么，然后对苏玫说道：“回头你把这些东西整理一下，分个类，然后都给父皇和母后送去吧。”
<br/>　　“都，都送走？”
<br/>　　“那些东西留着也没啥大用处，大多数都是些补品什么的，父皇和母后比我们需要那东西，明白么？”李承乾意有所指的说道。
<br/>　　毕竟开发大东北不是他一个人可以搞定的，虽然他现在已经把老头子架空的差不多了，但是要想让下面的人动起来，还是要让他们看到其中的好处，否则……。
<br/>　　好吧，事实上那帮唯利是图的家伙就算是老头子想要干点什么也不是那么痛快的，如果不给他们一点甜头，那帮老狐狸才不会一心一意的去干工作。
<br/>　　没办法，家国天下已经刻进这帮人的骨子里了，李承乾根本无力去改变，这是他折腾了十五年之后才确定的一个事实。
<br/>　　……
<br/>　　所以李承乾把老头子当成了里好的广告，东西一股脑的往老头子那里一送，外面几乎同一时间就知道了大东北的事情，一个个老家伙立刻红了眼睛。
<br/>　　要知道，大唐那时候的老山参可不是现在人工养殖的那种，说是百年老参能绝对能发挥百年老参的功效，病重的时候切一片含在嘴里就能吊住人命的那种。
<br/>　　另外就是古人迷信，深信人参是可以成精的，相信百年老参可以变成人参娃娃，而年头儿再多一些的，则可以变成成人。
<br/>　　这也是为什么人参比较值钱的原因。
<br/>　　于是乎，接下来的时间，李承乾的东宫简直可以用人山人海来形容，很多老家伙都跑到东宫来打听消息，想要知道开发大东北的计划什么时候才会开始。
<br/>　　而且李承乾则是显得若无其事，就好像根本不关心东北的事情一样。
<br/>　　那些老家伙没有办法，便转道去了大明宫找李二，声泪俱下的陈述开发东北的好处，什么矿产丰富，土地肥沃，各种理由被想了个遍。
<br/>　　最后把老头子逼的没有办法，便将这件事情重新交给了李承乾，则令让他在短时间内拿出一套开发东北的计划书。
<br/>　　至此，李承乾终于有了明正言顺的理由插手东北的事物。
<br/>　　……
<br/>　　大明宫，长孙皇后处。
<br/>　　长孙无忌愁眉苦脸的摇着头，端着一盏茶水却却无心去喝，纠结了半天才说道：“小妹，这次东北开发的事情你听说了吧？”
<br/>　　“哥哥莫非有什么事么？”长孙皇后大致可以猜到自己这个大哥想要干什么，但因为种种说不清楚的原因，她还是故作不知的问道。
<br/>　　“小妹，咱家里是做钢铁生意的，这个你知道吧？”长孙无忌见妹妹不往正题上说，于是只能自己往主题上拉。
<br/>　　“这个知道，怎么？遇到困难了？”
<br/>　　“困难眼下道是没有，只是眼下这货源距离都有些远，每年在花在来回运输上的资金便是一笔不小的数字。”长孙老狐狸皱着眉头，就好像吃了多大的亏一样。
<br/>　　“那哥哥的意思是什么？可想到了解决的办法？”长孙皇后依旧不动声色的问题。
<br/>　　“这事儿吧，我就琢磨着能不能找一个位置比较近的矿山，最好是煤矿和铁矿之间距离近一些，这样是不是就可以省下一笔运费。”长孙无忌此也不再遮遮掩掩了，直接说道：“这不这段时间正赶上高明要开发东北么，我听说那边有两个矿山。”
<br/>　　说到这里，长孙无忌露出一丝悲伤的表情：“小妹你也知道，冲儿那小子不争气，到现在也只是混了个四品官的位置，我就是担心万一哪天我死了，怎么也要给孩子们留点家产……。”
<br/>　　话说到这个份上，几乎与长孙皇后所猜测的分毫不差，不过长孙皇后却并没有一口答应下来，反而是低头沉思片刻，然后才缓缓说道：“不如这样吧，小妹把乾儿找来，哥哥与他谈谈如何？”
