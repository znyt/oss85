        &lt;h3&gt;第三卷第二百六十七章  原来你一直都在这里&lt;/h3&gt;
<br/>
<br/>        陈子昂陪着父亲半天，手中的电话就没有断过，被父亲赶出了病房。
<br/>
<br/>        “子昂，你不要在这里呆着了，我能吃能喝的，要你守着干嘛，况且你妈妈还在这里，你二哥也在这里，你去忙你的去吧。”
<br/>
<br/>        “那我把电话静音了。”陈子昂准备把电话静音了。
<br/>
<br/>        “你这个女子咋这么犟的，你去上班去，你要是呆这里我就不呆这里了。”父亲生气了，掀起被子准备下床。
<br/>
<br/>        “好好，我走，我去公司看看，晚上回来看你。”她的老父亲啊，总是那么善解人意，总是替儿女着想。
<br/>
<br/>        陈子霖带着姑姑代凉去了泉城老家，陈子昂就去了公司。
<br/>
<br/>        西城分公司从重新改制之后，她都没有回来看过。
<br/>
<br/>        陈子霖这两天要陪代凉他们，江俞轩也就到这边公司上班了，上午江家，下午陈家。
<br/>
<br/>        他正在和易倾秋、李秀、谭九儿等人开会，陈子昂推门进来了。
<br/>
<br/>        “陈总。”大家见陈子昂突然回来，都有些吃惊的站起来。
<br/>
<br/>        陈子昂走到江俞轩身边坐下摆摆手，示意大家继续开会，她专注的听着大家讨论，大家因为她的到来会议气氛就更热烈了。
<br/>
<br/>        在讨论商务问题的时候，陈子昂一言不发，商务有江俞轩在，她是从来都不发表意见了，这是这么多年来养成的习惯。
<br/>
<br/>        易倾秋是第一次近距离的看着陈子昂，陈子昂在大家的心目中就是女神的存在，原来的易倾秋是没有资格坐在这里的和大家开会的。
<br/>
<br/>        陈子昂穿着灰色套头插肩袖针织毛衣，黑色菱格加厚a字半身裙，黑色长款过膝大衣，黑亮的长发柔顺的披洒在大衣上，白净的瓜子脸上一双灵秀的丹凤眼散发着睿智而犀利的光，似乎一眼就能将人心看透，坐在那里不怒自威，不威尔严，但是脸上却带着淡淡的恬静的微笑。
<br/>
<br/>        她的笑容很有亲和力，让人不由自主的想多看看她，易倾秋一瞬一瞬的盯着陈子昂看，陈子昂冲易倾秋点点头，易倾秋一愣神，心莫名的欢悦起来。
<br/>
<br/>        这是一个什么样的女人啊？
<br/>
<br/>        当讨论到技术问题时，陈子昂开口了：“北京总部和西城分部是一体的，可以说全国的技术部都归北京总管，你们有什么好的想法和建议可以提出来，我们会进行研发和改进，对有创造性的建议会提出奖励和表扬。技术部作为公司售前和售后起着至关重要的作用，但是核心部分不会下放到每个个体也就是个人的手中。各位如果感觉知识不够用了，可以去北京学习，谁想学习可以写申请。”说完之后她转身问旁边坐着的江俞轩：“俞轩，你觉得呢？”
<br/>
<br/>        “嗯，很好，就这样决定。”江俞轩点点头，其实陈子昂说的也就是他想说的，如果核心技术部分到了个体手中，他们就没有秘密可言了，在商业领域里，每个公司都有自己生存的命脉。
<br/>
<br/>        他们俩人似乎又回到过去一唱一和的状态了，江俞轩心里一宽，真好，他们都还是原来的状态。
<br/>
<br/>        陈子昂也看着江俞轩微微一笑，似乎在说：“原来你还在这里。”
<br/>
<br/>        在俩人的对视中，他们都看到彼此，一如初见，那甜美的笑容，那俊美的容颜，似乎他们都不曾走远。
<br/>
<br/>        “陈总回来了，大家看看还有什么要补充的，或者是要给陈总汇报的，尽量畅所欲言，你们可是难得见你们的女神一面。”江俞轩开了一个玩笑。
<br/>
<br/>        “说得你好像天天见一样。”李秀挤兑了江俞轩一句。
<br/>
<br/>        “哇，子昂，你看看他们，他们经常这样欺负我。”江俞轩叫苦了。
<br/>
<br/>        “会哭的孩子有糖吃，江总，你就使劲的哭吧，看看陈总给你吃几颗糖。”谭九儿也俏皮的开口。
<br/>
<br/>        “子昂，你看，没错吧，你把我放在西城一点威严都没有，一点地位都没有，我要离开这里。”江俞轩叫苦连天。
<br/>
<br/>        “那你想去哪里？”陈子昂微笑着看着大家，大家已经将会议内容讨论完了，似乎他们开完会就是斗嘴，这在他们公司已经形成了习惯。
<br/>
<br/>        “我要和你一起去甸城！”江俞轩脱口而出。
<br/>
<br/>        “带他走吧，带他走吧，我们看着他都已经看腻了，况且他当着你的面是一副受气包的模样，离开你就是一个周扒皮加黄世仁，哎呀，我们已经被他压榨得不成样子了。倾秋，你说对不对？”谭九儿冲易倾秋挤了挤眼睛。
<br/>
<br/>        “嗯，就是，他还压榨我去当卧底。”易倾秋也随着谭九儿的话说。
<br/>
<br/>        “咦，俞轩，你好像没有魅力了，漂亮的女孩子都对你有意见，看来我使用的美男计失效了。”陈子昂笑着说。
<br/>
<br/>        “陈总，有你这个大美女坐在他身边，谁敢对他没意见？如果没有你，他只能孤独终老了。”李秀不怕死的说了这么一句。
<br/>
<br/>        “嗨嗨，我原来是这样的一个老巫婆的形象，易倾秋，新晋的公关部经理，年龄二十二岁，外号，灵魂导师一血富婆，善工谋，谙人性，进修过人性心理学。”陈子昂巧妙的岔开了李秀的话题，对着易倾秋说。
<br/>
<br/>        “陈总还知道我呢。”易倾秋不好意思的笑了。
<br/>
<br/>        “江总想要给我推荐的人，我怎么能不了解呢？是吧，俞轩。”陈子昂露出洁白的牙齿，狡黠的一笑。
<br/>
<br/>        “这都被你知道了，一点意思都没有，我就是你的小透明。”江俞轩感觉有些委屈，他确实是想将易倾秋好好培养一阵给陈子昂送过去，可是他还没有说出来这件事情啊。
<br/>
<br/>        大家被江俞轩那略带委屈的语气逗笑了。
<br/>
<br/>        “你傻啊，这个叫做......”陈子昂的话没有说完。
<br/>
<br/>        大家齐声喊道：“心有灵犀一点通！”
<br/>
<br/>        “好吧！那就心有灵犀一点通！陈子昂我想去甸城你也早都知道了，那就这样决定了！散会，易倾秋留下，其余的人都各自忙去吧。”江俞轩宣布了散会，留下易倾秋。
<br/>
<br/>        “陈总，江总。”大家一走，易倾秋突然就有点紧张了。
<br/>
<br/>        “倾秋，江总想让你随我去北京，但是我希望你留在西城，江总一走，西城就是陈子霖在，他需要一个得力的助手，帮他掌控西城的一切，每一个分公司都是总公司不可分割的部分，一动而发全身，所以西城比北京更需要你。”陈子昂开口了。
<br/>
<br/>        “从今天起，你取代陈子霖的位置，陈子霖取代江俞轩的位置。希望你能为公司出谋划策，随我一起奋斗。”陈子昂微笑的看着易倾秋。
<br/>
<br/>        “我可以吗？”易倾秋都不相信自己的耳朵：“那么江总干什么？”
<br/>
<br/>        “我和她一起去别的地方打江山！没了我，她什么都不是。”江俞轩开心的大笑，最近的那些郁闷纠结都一扫而空。
<br/>
<br/>        “有那么高兴嘛？”陈子昂看着江俞轩微微而笑。
<br/>
<br/>        “当然，我们又可以并肩战斗了，不过，宁雅走了，谁代替她的位置？”
<br/>
<br/>        “公司还有钱喵喵和田星。李凌菲也可以胜任，分公司人才也很重要，你不能偏心的将人才都送到我身边。再说，后期要请专业的人士来管理，我也要轻松轻松了。”
<br/>
<br/>        “你这是准备撂挑子了？那你准备干啥？”
<br/>
<br/>        “寻一处心仪之地，回归与山水间。”
<br/>
<br/>        “带上我？”
<br/>
<br/>        “你？有可能？”
<br/>
<br/>        易倾秋看着陈子昂和江俞轩俩个人之间的对话，她觉得她再在这里是多余的，便去干别的工作了。
