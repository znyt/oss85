　　谁都知道南方的边境处这些丧尸来自哪里，群体从遥远的南方而来本身就是尸潮聚集的现象，好好的尸潮没有道理到达边境地带后就散开。
<br/>　　环绕在段氏基地周围的这些丧尸群的异样，很快就得到了段江伟等人的关注，但由于段氏基地的这些高层在这几天都繁忙无比，一边准备着大战物资，一边不断对周边的防护地带反复推敲，根本没有时间也没有精力来专门分析这一诡异情况的原因。
<br/>　　于是在段江伟的提议下，他们第一时间将这则消息扩散出去，希望能有人解惑，同时也在提醒这处地带越来越多的猎者。
<br/>　　上京的两名老者也同样在第一时间得到了这一情报，但他们并不能分析出结果，在两人共同的决定下，将一封询问信送到了狼牙。
<br/>　　此时的狼牙安罗城内，一片祥和干净的城市，仿佛在这个黑暗的末世纪元与世隔绝。
<br/>　　道路上不见任何残暴现象，空气中也毫无腐臭味道，与末世中疯长植物的巧妙结合，让这座城市好似一处花园天堂。
<br/>　　狼牙的幸存者大多都居住在这里，他们有着自己的家，有完善的经济链，更有丰富的泥土来种植，小本买卖亦或是各项才能的施展，让家家户户都能自给自足。
<br/>　　哪怕偶尔出现的几次危险，也在狼牙警戒队及那个时光炮弹的恐怖威力下有惊无险，想要在这座城市作乱的敌人还没开始行动就被打回了家！
<br/>　　若不是在这座城市里每隔一段距离都会实时张贴的事件榜，住在安罗城的居民恐怕都会忘记他们身处的是末世纪元。
<br/>　　“有最新事态了！”一名十几岁的少年在街道上奔跑，将声音传递到两旁的居民楼里。
<br/>　　很快从一座座房屋中陆陆续续的有人走出，他们有人刚结束一天的劳作，有人身上还系着围裙，更有人抱着小孩，这些人都相互熟知的交谈着，结伴走到距离自己最近的事件榜围观。
<br/>　　事件榜是安罗城主尚九娣上任不久后的一项益民措施，并不像大多数基地那些告示榜上的内容一样官方，安罗城的事件榜上往往都张贴着世界最新的动态，内容包含了方方面面，甚至还有着一些其他基地不敢公布与众的内容。
<br/>　　当然，所有可以张贴在事件榜上的内容，都经过了狼牙情报部最严密的审查，内容处理之后才会公布，毕竟像神秘家族内部的辛秘历史，亦或是循环论这种可怕的推理，还不适合告诉大多数的幸存者。
<br/>　　而今天新公布的最新世界动态，正是与南署基地的大战有关。
<br/>　　“是最新战报！”一名幸存者在事件榜面前激动出声：“狼牙战团又大胜拉！”
<br/>　　“我看看，我看看，每次公布的战报都有很详细的数据，这次又歼灭了多少敌军？”
<br/>　　幸存者们都围聚在事件榜周围，争先恐后的看着最新消息，比起其他事件来说，狼牙战团的消息更让他们感兴趣，也能轻而易举的调动起整个安罗城居民的情绪。
<br/>　　因为狼牙战团就是他们这座城市的军团，楚涵上将更是他们心中的守护神！
<br/>　　在这些幸存者都热烈的议论中，这座城市里很普通的一座房屋内，却有四个女人并没有出去围观，她们围聚在一张四方桌上，兴致勃勃的打起了麻将。
<br/>　　实际上这并不奇怪，并非所有人都会第一时间出去围观战报，很多妇人和孩童更倾向于听二手消息，毕竟外面那些激动无比的人很多都是狼牙战团的崇拜者，挤在人群里散发着一股臭汗味。
<br/>　　在这四个女人的麻将桌旁，一个年轻的少年从旁边的门内走出，他看着屋内不断响起的麻将声，颇为头疼的皱起眉。
<br/>　　“我说，这次事件榜公布的可是咱狼牙战团的最新战绩，你们都不去看吗？”少年出声，打断了四个女人的谈话。
<br/>　　“有什么好看的？”少年的母亲头也不抬，手法娴熟的打出牌后继续道：“狼牙战团肯定是大胜嘛！”
<br/>　　少年很不服气，语气加重了起来：“这次可不是普通战役，狼牙战团对抗的是史无前例的一次大尸潮，数量多达好几千万，相当于一个南都城那么多的丧尸！”
<br/>　　闻声，四名妇人中有一人的动作一顿，但很快就微不可查的继续摸牌。
<br/>　　其他三名妇人则是好笑的看着这名少年，同时打趣了起来。
<br/>　　“看你这猴急的样，一会儿街坊邻居自然会把得到的信息原原本本的告诉我们，去不去事件榜不都一样？”
<br/>　　“就是啊！方向吧，每次事件榜一更新，咱这些人的聊天内容少不了是这些。”
<br/>　　“谁让现在也没了报纸新闻，没啥话题可说。”
<br/>　　看着这些妇人无动于衷的样子，少年面上浮现了一丝愤怒：“你们也太过分了，狼牙战团打了这么多场仗，牺牲了那么多战士，你们竟然一点都不关心？就知道在这里打牌！”
<br/>　　望着少年愤怒的样子，几名妇人都顿住，有些不知所措。
<br/>　　少年的母亲面上浮现出了一丝尴尬，道：“没有不关心，只是不想出去弄一身臭汗回来，最近这天气也热起来了。”
<br/>　　“臭汗算什么事？”少年更加愤怒：“你们嫌弃的那些人，说不定就是哪个因为受伤而退役的狼牙战士，若不是狼牙战团的努力，你们那有在这里挑三拣四的资格？看看其他基地的幸存者，我们安罗城的生活简直就是天堂了！”
<br/>　　随着少年愤怒的咆哮，从头至尾都没有出声的那名妇人第一次抬起了眼，她看向少年道：“狼牙战团打下的天下，创造了安罗城这样一处无忧的城市，不是让我们这些居住在这里的幸存者忧心忡忡居安思危，而是让我们有除却生存外培养其他爱好的权利。”
<br/>　　妇人的话说的并不快，但却接连不断，在少年当场有些发蒙的状态下，她继续开口，带着一丝微不可查的骄傲：“安罗城的孩子能接受华夏最全面的教育，老人有最安详的晚年，商人能有发展的空间，而女人更有挑剔的资本，这都是狼牙战团给我们的好处，我们并没有不在乎，而是在对方赠予我们这一权利的时候，将之完美利用，让其他基地羡慕，让对狼牙有敌意的人嫉妒到发疯。”
<br/>　　“这理所应当，有什么不对么？”说罢，这名妇人忽而一笑，过人的容貌更加彰显。
