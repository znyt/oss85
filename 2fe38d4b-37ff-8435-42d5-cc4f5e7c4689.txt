　　兽王宫和鬼王宫、魔王宫，妖孽众多。
<br/>　　不过，苏莫对此并没有什么意外，他所要做的，就是好好表现而已。
<br/>　　这三宫天才虽然很多，但是，最顶尖之人，也不可能比姬月松强。
<br/>　　修为到了二品高级，都是顶尖妖孽，那些人的实力，差距都不会很大。
<br/>　　“天王宫之人怎么还不来？”
<br/>　　“听说天王宫有一位不出世的妖孽！”
<br/>　　“我也听说了，据说此人从不露面，由天王宫大长老亲自指导！”
<br/>　　“这人不会就是冷无敌吧？”
<br/>　　“不清楚！”
<br/>　　五大王宫的弟子，熙熙攘攘数百人，聚集在轮回古城城门之前。
<br/>　　嘈杂的议论之色，此起彼伏。
<br/>　　“月松哥，你说冷无敌，会不会是天王宫之人？”褚彦和姬月松、骆妤聚集在一起，低声询问。
<br/>　　“可能性不大！”
<br/>　　姬月松摇了摇头，继续道：“那冷无敌，应该不是轮回神朝之人，而且，也不一定是年轻一代！”
<br/>　　“我想也是，不过，还是希望他能出现！”褚彦微微叹了口气。
<br/>　　“怎么？你很想一雪前耻？”姬月松面上挂着淡淡的笑意。
<br/>　　“当然！”褚彦点了点头，不想一雪前耻，那是假的。
<br/>　　只不过，他心中清楚，就算如今在碰到冷无敌，或许自己依旧不是对方的对手。
<br/>　　毕竟，他在成长，对方也在成长。
<br/>　　“褚彦，你就不要再想那个冷无敌了，这次来轮回古城，妖孽众多，不能大意！”骆妤叮嘱道。
<br/>　　那冷无敌，只是一个虚无缥缈的人物，虽然逆天无比，但只是天浑第一斗场一个匆匆过客，何必再去较真呢！
<br/>　　“嗯，我明白！”褚彦点了点头，这众多的妖孽，才是他的对手，冷无敌已经成为了过去。
<br/>　　不远处，沁儿依旧和苏莫坐在一起，两人倒是没有引起过多的关注。
<br/>　　苏莫只是一个声名不显的人物，加之修为又不高，自然无人关注他。
<br/>　　至于沁儿，虽然很多人都知晓其身份，也知道其天赋极高，但是，还不被很多妖孽放在眼中。
<br/>　　嗖嗖嗖！
<br/>　　这时，天空中再次传来破空之声，众人为之一静，齐齐转头望向天际。
<br/>　　便见到天际的尽头，出现了一群人影。
<br/>　　这群人数量不多，约莫七八十人，为首之人，是一名面容刚毅的锦袍中年人，这群人正是天王宫之人。
<br/>　　“天王宫的人来了！”
<br/>　　“天尘宇的弟弟天尘山也来了！”
<br/>　　“前几年就听说天尘山要晋升三品了，看来还没晋升！”
<br/>　　众人的目光，纷纷在天王宫的人群中扫视。
<br/>　　苏莫亦是在观察，观察天王宫的阵容。
<br/>　　天王宫的阵容，并不强，感觉和人王宫差不多。
<br/>　　唯有一人特别的惹眼，此人一身华服，五官英挺，容貌俊朗，气质独特，给人一种高贵之感。
<br/>　　此人的气息，极为浑厚，虽然也是二品高级的修为，但是气息明虚超过了其他二品高级修为之人。
<br/>　　“此人应该就是天尘山，是个对手！”苏莫耳边听着众人的议论，心中暗道。
<br/>　　不过，很快，苏莫就将目光，落在了天尘山旁边的一人身上。
<br/>　　此人高高瘦瘦，一身月白色长衫，面色古板。
<br/>　　此人的修为，也是一品高级，看上去很是普通。
<br/>　　但是，不知为何，在此人的身上，苏莫有种熟悉之感。
<br/>　　“奇怪！”苏莫仔细的打量了一番此人，确定自己并未见过，对方气息也很陌生，更加不可能是帝一魂。
<br/>　　但是，他就是有一种熟悉之感。
<br/>　　嗖嗖嗖！
<br/>　　呼吸之间，一众天王宫的人来临，纷纷降临而下。
<br/>　　“天有为长老！”
<br/>　　“人王宫玄天长老来了，没想到有为长老也来了！”
<br/>　　“又为老兄，别来无恙！”
<br/>　　其它五宫的长老，纷纷向天王宫的长老打招呼。
<br/>　　这位天王宫的长老，名为天有为，乃是天王宫四长老。
<br/>　　“诸位客气了，老夫来晚了一步，还请诸位不要介意！”
<br/>　　天有为面带微笑，环视其它五宫的长老。
<br/>　　在场的长老之中，除了他和玄天长老两人，是中级神王之外，其他长老均是低级神王。
<br/>　　此时此刻，六宫的弟子，齐聚在轮回古城门前。
<br/>　　每一个弟子，都是天才人物，因为，能来到此地的人，都是百岁之内的真神境武者。
<br/>　　这种天赋，每一个都是天赋过人之辈。
<br/>　　不过，如今，除了天王宫之外，其它五宫的弟子，俱是在打量着天尘山，个个面色凝重。
<br/>　　就连姬月松、狄央等人，眸中都是留露出郑重之色。
<br/>　　“哈哈，诸位，看来你们俱是将我当成了对手啊！”天尘山如同一位翩翩公子，气度非凡，朗笑一声说道。
<br/>　　“天尘山，十几年前我曾败给了你，但那时你修为高于我，如今我们同阶，我定会打败你！”
<br/>　　魔王宫阵营之中，一位身穿黑色劲装，面容邪异的青年，眸中战意澎湃。
<br/>　　此人，是魔王宫百岁以内第一强者，名为归无血。
<br/>　　“天尘山，有人说你自称六宫百岁之内最强一人，我可是不信！”兽王宫阵营，一位高挑的女子，面带笑意的说道。
<br/>　　这位女子，身穿碎花长裙，面容不是很美，脸上有着不少雀斑。
<br/>　　此女，名为花小雀，兽王宫百岁以内最顶级的妖孽天才。
<br/>　　“百岁之内最强？这种无知之言也敢说出口！”姬月松冷笑着说道。
<br/>　　这个名头，可是有些吹大了！
<br/>　　“哈哈哈！”
<br/>　　天尘山见此，扬天长笑了起来，他环视五宫天才，轻笑着说道：“百岁之内最强，这话我可没有说过，不过，就凭你们这些人，还没有资格与我一战！”
<br/>　　天尘山的话，说的轻描淡写，面上并无傲气，但是，却是自信无比。
<br/>　　“猖狂！”
<br/>　　“天尘山，你未免太自大了吧！”
<br/>　　“口气这么大，到时候若是败了，看你颜面何存？”
<br/>　　众人闻言，纷纷冷笑了起来，这个天尘山，居然如此之大狂妄！
<br/>　　不远处，苏莫一直在静静的看着，这天尘山的确够自大。
<br/>　　“多说无益，到时候手下见真章！”天尘山轻笑一声，也懒得再多言。
<br/>　　他之所以自信，是因为，他的实力强大到了连他自己都不敢相信的地步。
<br/>　　这一切，全是那一位禁忌般的存在所赐。
<br/>　　当然，他也要感谢自己的大哥天尘宇，若不是对方，那位存在也不会帮助他。
