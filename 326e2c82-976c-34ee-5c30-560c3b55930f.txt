　　万多名武者，分布在这方圆三百里的辽阔森林里。
<br/>　　自从剑派内的执事耐心地说完了规则后，所有人都互相警惕起来。
<br/>　　当考核宣布开始后，一众武者进了森林，不出片刻，便四散离开，没了踪影。
<br/>　　萧逸也不例外，独自跑了数个时辰后，停了下来。
<br/>　　但，现在的他，并不是想着怎样去掠夺其他人的分数，而是一门心机想着，大睡一场。
<br/>　　这十多天以来，先是几乎不眠不休地跑了万里路途；而后又炼制了跨越自己境界之外的四品巅峰丹药，心神疲惫。
<br/>　　最后还拼着重伤的代价杀了两个洞玄一重武者。
<br/>　　层层累积下来，就算他的身体受得住，也依然会觉得疲倦。
<br/>　　当然，最重要的原因，是因为这考核有大把的时间。
<br/>　　“反正要在森林里呆够一月，姑且偷懒两日吧。”萧逸自语一笑，在一苍翠大树上做好了掩饰，而后倒头便睡。
<br/>　　不得不说，萧逸的藏身手段确实厉害。
<br/>　　不过是几片树叶玄妙地随意摆放着，远远看去，他竟如同与这颗大树融为一体。
<br/>　　如果不认真看，哪怕近在咫尺，也绝对无法发现有一人睡在这树上。
<br/>　　对于一个杀手之王来说，藏匿，是最简单不过的事。任何事物，都能在他手中配合得天衣无缝，成为最好的藏身之所。
<br/>　　这种本事，无关修为，在乎经验和手段。
<br/>　　起码这万多名天才中，哪怕修为颇高，但论经验和手段，却远远比不过他，更别说发现他了。
<br/>　　.....
<br/>　　森林内，没有人敢轻举妄动，更没有人敢率先发起战斗，成为众矢之的。
<br/>　　成为裂天剑派弟子的机会，任何人都珍而重之，不得不慎重。
<br/>　　然而，当时间来到第三天时，状况却陡然间剧烈变化。
<br/>　　仅仅数个时辰，发生的战斗便不下数百场。
<br/>　　伤亡频频出现。
<br/>　　一天后，战况更为激烈。
<br/>　　原本静谧的森林，陡然间变得随处可听到战斗声响。
<br/>　　整个森林，开始变得危机四伏，人人自危。
<br/>　　这些武者，在北山郡各地都是小有名气的天才，在其家族或势力受尽宠爱尊崇。
<br/>　　但如今，却即将面临人生中最残酷的一次失败，甚至是死亡。
<br/>　　三天后，一场场战斗几乎席卷了大半个森林，大多地方变得满目苍痍，树木被摧毁，土地被崩坏，湖泊被断流，鸟兽四处奔逃。
<br/>　　而究其原因，到底为何森林内的战斗陡然间打响；又陡然间激烈；最终变得硝烟四起？
<br/>　　仍旧在大树上的萧大爷却毫不知情，仍旧在呼呼大睡。
<br/>　　幸好，多年来养成的习惯，让他哪怕是睡觉，也会保持高度警惕，别说打呼，就连呼吸声，也微弱得几不可闻。
<br/>　　第四天，距离萧大爷那颗大树，十数米之外的地方，出现了三个狼狈的武者。
<br/>　　两个男子，一个女子。
<br/>　　两个男子，一个先天六重修为，一个先天七重修为。
<br/>　　女子，先天六重修为。
<br/>　　这三人的修为，在这万多名武者中，算是处于上游水平的。
<br/>　　此刻，却尽皆带伤，满脸狼狈之色。
<br/>　　“还想跑？你们逃不掉的。”
<br/>　　忽然，一声大喝，数十人将他们三人围了起来。
<br/>　　为首一年轻男子得意地看着三人，道，“三位前两天似乎打败了不少人，得了不少分数吧。”
<br/>　　“胡海，你到底想怎样？”三人中的其中一人站了出来，面容坚毅，冷声问道。
<br/>　　“林劲，老子也懒得与你废话。”为首男子，也就是胡海，得意地说道，“将你们三人令牌上的分数交出来，老子便饶你们小命。”
<br/>　　“不可能。”林劲，便是三人中先天七重修为之人，咬牙说着。
<br/>　　“不可能？”胡海冷笑道，“别敬酒不吃吃罚酒，乖乖将分数交出来，就算清零了，你们也还有大把机会去抢其他人的。”
<br/>　　“若是不交，就别怪我心狠手辣，将你们三人击杀了。”
<br/>　　林劲是个极为傲气之人，冷声道，“胡海，你不过是先天六重修为，若不是仗着一堆狗腿子，你以为你现在有资格在我面前说话？”
<br/>　　林劲三人，看起来不过是18岁左右。
<br/>　　但那胡海，虽是先天六重，但年纪起码在二十出头，论天赋，显然远远比不上。
<br/>　　胡海脸色一黑，“林劲，是你们自己找死，就别怪我了。”
<br/>　　说罢，胡飞大手一挥，身后数十武者蜂拥而上。
<br/>　　这数十人，皆是先天四重修为，论单打独斗，绝不是林劲三人的对手；但群起攻之，林劲三人根本没有胜算。
<br/>　　林劲脸色一凛，对身旁女子道，“嫣然，虽说我们三人相识不过数天，但起码共过患难。待会儿你先走，我和铁牛和他们拼个鱼死网破。”
<br/>　　女子，在三人中年纪最小，只是个少女。虽算不上绝色，但也是少有的美人，身若杨柳，清风缓缓；面如凉风细水，让人看起来说不出的舒服，一双水汪汪的大眼睛，说不尽的温柔。
<br/>　　但温柔的她，此刻却坚定地摇了也摇头，脸上尽是傲气。
<br/>　　前方，胡海冷笑一声，“鱼死网破？怕是你们三条小鱼死尽了，我的网仍旧毫发无损。对了，那女子，先不取她性命，留着。”
<br/>　　胡海说着，眼中看向女子的眼光，充满了下流。
<br/>　　战斗瞬间打响。
<br/>　　林劲、铁牛，被三十个先天四重武者围攻，虽未落败，却陷入下风。
<br/>　　胡海则伙同十数人，围攻女子。
<br/>　　数分钟后，女子完全落入下风，被胡海一掌震飞。
<br/>　　“噗。”女子猛地吐出一口腥血，显然受了一点儿轻伤。在半空中划过一条曼妙的弧度，直直被击飞。
<br/>　　她被击飞的方向，竟是十数米外的大树方向。
<br/>　　恰在此时，本在呼呼大睡的萧大爷似乎睡够了，又似乎是被吵杂声音吵醒了，睁开了朦胧的眼睛，准备伸个懒腰起来。
<br/>　　然而，他的懒腰还未完全伸展，一道妙影，却以极快的速度朝他撞去。
<br/>　　萧逸的反应速度相当快，双眸一冷，暗道，“偷袭？”
<br/>　　乾坤袋中的紫云精铁剑瞬间抽出。
<br/>　　正当他准备一剑将偷袭之物劈开时，才看清那是个人影，连忙收剑。
<br/>　　就是这收剑的空档，让他失去了躲开人影的机会。
<br/>　　“嘭”的一声，萧逸直接被撞下大树。
<br/>　　“额。”萧逸摔落地面，吃疼地微微叫了一声。
<br/>　　同时，他也反应了过来，撞飞自己的，竟是个年纪跟自己相仿的女子，而且女子正脸色苍白地压在自己身上。
<br/>　　好死不死的，自己的双手似乎在掉落大树时随手挥舞，搭在了她的腰间上。
<br/>　　近距离闻着女子温热的气息，触碰着那轻盈的腰肢，让得萧逸一阵尴尬，而后连忙放手。
<br/>　　“小姐，你刚才是在练天外飞仙吗？没事的话，起来好不好，你很重。”萧逸淡淡地说道。
<br/>　　事实上，女子一点儿都不重，相当轻盈，只不过是萧逸避免尴尬的说辞罢了。
<br/>　　“天外飞仙？”女子一愣，这时也反应了过来，也是一阵脸红。
<br/>　　随即便是心头慢慢的疑惑，暗道，自己刚才一路跑来时，根本不见树上有人，这人是忽然出现的吗？
<br/>　　随即，她连忙站了起来，满脸歉意地看向萧逸，道，“抱歉...额。”
<br/>　　当女子看到萧逸的面容时，忽然一愣。面前少年的样貌很是俊俏，加上那刚刚睡醒的慵懒神态，还有眉宇间流露着的截然相反的孤傲气息，竟让她心头一阵迷离。
<br/>　　“额..那个...什么是天外飞仙？”女子反应了过来，略显尴尬。
<br/>　　萧逸不语，虽然女子很美，但他只看了一眼，便没有再看，心头暗骂着，都怪自己睡得太死了。
<br/>　　这时，不远处，胡海见树上忽然掉下一个人影，也是一惊。
<br/>　　但感知了一番后，发现人影的修为不过先天三重，顿时面露狰狞之色，“去将那小子也一并杀了。”
<br/>　　“诶呀，遭了。”女子立刻想起来自己还在战斗。
<br/>　　萧逸明显听到胡海的话，脸色一冷，对女子道，“就是那个家伙将你击飞，然后把我撞下来了？”
<br/>　　‘把我撞下来’五字，声音加重。
<br/>　　女子点点头，道，“就是他，你快跑，不要被他抢了分数。”
<br/>　　女子显然也感知出了萧逸不过是先天三重的修为。
<br/>　　“哼。”萧逸冷哼一声，身影一闪，瞬间消失在原地。
<br/>　　女子眼睛一闪，惊道，“好快。”
<br/>　　下一秒，萧逸瞬间出现在胡海身前；一把紫云精铁剑，冷冷地架在胡海脖子上。
<br/>(本章完)
