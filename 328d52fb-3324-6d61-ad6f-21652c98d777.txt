　　妖灵山脉猴儿山的惊天大事件一晃七天过去，这七天内，山脉众妖津津乐道，不是讨论传说中的七十二必死天劫，就是讨论那条渡过必死天劫，却逃不了妖王魔爪的可怜蛇精。
<br/>　　历史上，还从来没有妖精可以渡过必死天劫，不得不说那条蛇精太优秀了，也正是因为太优秀，才引的两大妖王的嫉妒跟杀心。
<br/>　　妖灵山脉一共才五大妖王，一位妖王费尽手段也杀不了那条蛇精，一位妖王靠着偷袭才终结一切。
<br/>　　更让人匪夷所思的是蛇精被杀之后，一位白衣女人出现了。
<br/>　　那女人先是痛哭一场，然后便扬言要杀尽鼠类一脉，这七天，妖灵山脉一只老鼠都见不到，好像都吓得躲起来了。
<br/>　　听说，还是灰妖王亲自下的命令，让那些觉醒类，精怪类，哪怕是妖灵类的鼠妖也要全部躲起来。
<br/>　　让众妖诧异的是，好像连灰妖王本身也非常忌惮那个白衣女人，他自己最后都躲了起来。
<br/>　　可就算如此，在这三天之内，鼠类一脉的妖精还是被那个发了疯的女子杀了不少，据说光妖灵层次的，就被杀了足足五只。
<br/>　　要知道每一只妖灵都超凡脱俗，得天独厚，纵然是老鼠，都血统高贵不凡，就这样被杀死了。
<br/>　　更为惊动是的，之前低调的蛇族一脉也疯了一样，全体出动，在妖灵山脉展开了捕鼠行动。
<br/>　　短短七天内，足足多达上百万只的老鼠死于毒蛇之手。
<br/>　　一场腥风血雨引得无数种族都诚惶诚恐，每一条蛇都跟疯了一样，充满了凶性。
<br/>　　..............
<br/>　　妖灵山脉有一座非常著名的紫竹林，放眼望去就是一片竹海，一株株紫竹高耸挺拔，顶天立地，不同一般的翠竹，下面每株竹子都是紫色的，唤作紫金竹，坚韧挺拔。
<br/>　　而这片地域属于白妖王的青竹林地盘。
<br/>　　新的一天刚降临，雾霭的天，烟波浩渺，过上个把小时，金灿灿的阳光洒在竹海上，泛起碧海金波，景色奇美。
<br/>　　茂密的竹林若隐若现，竹滔阵阵，只见一片片如壁玉石般的莹绿，若要从空中俯视它们，那就像一块玉壁镶在了深山的土地中苍劲挺拔的紫竹，是要与天试比高！
<br/>　　每当有细雨落下，细雨就敲打着竹响，风声，雨声，落叶声，声声如美妙的音律回荡，美不胜收。
<br/>　　如若这样的竹林在人类社会中，一定是一等一的旅游景点，每天都会是人山人海，因为这里实在太美了。
<br/>　　都知道白妖王的本体是一只刺猬，跟竹子好像八竿子打不到一块去，但是偏偏这只刺猬特别喜欢竹子，整个青竹林地域，有各自各样的竹林。
<br/>　　紫竹林名声最显，据说很久很久以前，白妖王还没有达到妖王层次，还在人间修炼，喜欢上了一位人类女人，那女子是一位诗人，特别喜欢已竹子作诗。
<br/>　　“我喜欢竹子，因为它任风吹雨打，都屹立不倒，仿佛世间最坚强的人”
<br/>　　“你喜欢，那以后我就为你种下一片竹林。”
<br/>　　竹林有了，可伊人早尘灰尘土归土，眼前风景无限好，可仍是近黄昏，看不到，都是徒劳。
<br/>　　凡人寿命不过百，古代的人寿命更低。
<br/>　　当然，这些都是谣言，区区刺猬，那么有情调？
<br/>　　一代妖王，都是成就在尸骨遍野身上，当然，对于活了不知道多少岁月的白妖王来说，有一些不同寻常的故事也都很正常。
<br/>　　如今五大妖，曾经在民间可是五大仙，信徒众多，逍遥快活。
<br/>　　在这繁密竹海之中，春风青山紫竹，甘露绿叶暖阳，美如画卷的紫竹林，一个恰正风华绝茂的少女款款莲步而来，犹若不食人间烟火的仙子，美的令人窒息。
<br/>　　少女对这片紫竹林非常的熟悉，她每天都会来，宛如自己家的后花园。
<br/>　　紫金竹是一种特殊的竹品种，竹内有水，一般的竹子内藏水久了，水会变成红色，但是紫金竹内的水却是紫金色，而且非常的甘泉，好喝极了。
<br/>　　喝了对身子有非常显著的效果。
<br/>　　少女每次清晨之际都会来采竹水，很多健壮的竹子清晨竹身会溢出水，跟露珠一样，挂在上面，不会滑落，很神奇。
<br/>　　每一滴闪亮，透明，圆润，蕴含灵气。
<br/>　　其实她不需要那么麻烦，直接砍了竹，竹节中就有大量的竹水了。
<br/>　　只是此少女心善，连一根竹子都不忍心破坏。
<br/>　　按往常，她手中提着一个玻璃瓶，穿行在竹海中，采集每一滴紫金竹身上挂着的露珠。
<br/>　　这些露珠寿命很短，太阳一晒，它们也就被蒸发了。
<br/>　　这种从竹身采露水的行为是极其麻烦的，但是少女脸颊洋溢着青春的笑容，乐此不疲。
<br/>　　此时，当少女来到一根高大挺拔，竹身满是露珠的紫金竹面前，她忽然一愣，竹下居然盘踞着一条小蛇。
<br/>　　这蛇好似冬眠了一样，在地上一动不动，连她那么大的人站在面前，也毫无察觉。
<br/>　　“哇...好漂亮的小蛇蛇”
<br/>　　这条蛇很小很精致，蛇身既然有三种不同的条纹图案，就跟人用油画添上去一样，美轮美奂。
<br/>　　好看的不真实，完全没有一般蛇的凶煞模样。
<br/>　　少女蹲了下来，想用手摸一摸蛇头，看看这蛇是不是还活着。
<br/>　　可猛然，她那张诱人的樱桃小嘴半张的合不上，这小蛇的额头上，居然印着一个“王”字。
<br/>　　这个王字不知道是不是有微阳的衬托，既然还闪闪光泽。
<br/>　　“不愧是妖灵山脉，连蛇都跟外面的不一样”少女转了转美眸，微微一笑。
<br/>　　“小蛇蛇，你是不是受伤了”
<br/>　　她见盯着这条精致的小蛇半天也没反应，少女胆子也大了起来，双手一抓，直接将这条小蛇给捧到了手心中。
<br/>　　还别说，这条小蛇不过五十公分，在她的葱白手掌中，刚好正合适。
<br/>　　即使少女将小蛇捧到手心中，但这蛇依然没有一点反应，眼睛是闭着的，她好好的坚持了一下，这蛇身上看起来也没什么伤。
<br/>　　但是整个小身子却异常的冰冷，跟捧着一条冰棍似的。
<br/>　　不过少女别的本事没有，就是不怕冰，好像早就习惯了这种温度，因为她自己全身也很冰。
<br/>　　甚至可以说不需多让。
