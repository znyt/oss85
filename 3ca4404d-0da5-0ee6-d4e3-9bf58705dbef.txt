　　“秦氏见过太子殿下，见过太子妃，小儿给殿下添麻烦了！”找到儿子的秦氏在喜悦过后，顾不得教训儿子，连忙向跟着进来的李承乾等人致谢。
<br/>　　儿子进来之后，马上李承乾他们就跟了进来，只要不是傻子都能猜到发生了什么。
<br/>　　李承乾原本就有些心虚，哪里会让秦氏真的把礼行完，连忙上前虚扶一把，口中说道：“秦家婶婶且莫如此，折煞高明了！”
<br/>　　“臭小子，还不过来谢谢太子殿下，若不是太殿下，只怕你就回不来了。”秦氏见李承乾不受自己的礼，转头瞪了一眼儿子。
<br/>　　可怜的秦怀玉委屈的看了自己的娘亲一眼，心说如果不是这个人，自己又怎么可能会走丢，但老娘的话他终是不敢违背，加上走丢之后担心秦氏骂他，于是便磨磨蹭蹭的走了过来，小大人一样对李承乾礼了一礼：“谢太子殿下！”
<br/>　　李承乾等那小子行完了礼，在他头上拍了一把，随手说道：“好了，以后不要乱跑也就是了。”
<br/>　　……
<br/>　　儿子丢了一回，秦氏自然是没了逛街的心思，李承乾看出秦氏似乎还没有从刚刚的惊慌中回过神，也不好让她们自行离开，于是便决定亲自送她们回去。
<br/>　　一路无话到了老秦的府上，正好老秦也在家中，简单的说了一下西市发生的事情，老秦再次谢李承乾的才算是把件事掀了过去。
<br/>　　末了，李承乾拉着老秦到了所谓的书房看着那满墙的刀剑不得不苦笑赞叹：“秦叔叔，果然还是那么豪迈！”
<br/>　　“呵呵，让殿下见笑了，俺老秦没啥文学，也就是好个武！”秦琼知道李承乾拉着自己来到这里一定是有话要说，于是也没有让仆役跟进来，打个哈哈便与其相对而等，静静的等着李承乾说出来意。
<br/>　　“秦叔叔，今日怀玉之事多少也与小侄有些关系，若不是小侄拿着樱桃枝，想来那小子也不会不顾婶子，跟着小侄跑了！”
<br/>　　“些许小事，殿下不必放在心上。”老秦摇了摇头，儿子找回来就好，至于原因并不重要。
<br/>　　“嗯。”李承乾点了下头，这事儿反正他解释过了，估计老秦以后应该不会有什么想法，顿了顿说道：“秦叔叔，小侄有些话不吐不快，一直不知当讲不当讲。”
<br/>　　“殿下有话只管明言，秦某这条命都是殿下从阎王殿里拉回来的，何来当讲不当讲之说。”秦琼微微一笑，对李承乾说道。
<br/>　　他是一个义字当先的人，对这份救命之恩一直很看重，连带着对李承乾自然也有一种别样的感情，虽然说不敢陪着他造反，但一些可大可小的事情还是能听一听的。
<br/>　　李承乾很熟悉老秦的性子，见他不在乎，便咳了一声说道：“秦叔叔可有发觉怀玉的性子有些软？”
<br/>　　“怀玉……”老秦没想到李承乾竟然会把话是扯到儿子身上，不由呆了一呆，半晌才叹了口气说道：“其实他这样的性子也不错，像他娘总好过像某。”
<br/>　　“秦叔叔，小侄这么说没有别的意思，只是担心日后秦家……”说到这里李承乾停住了。
<br/>　　但秦琼却明白他是什么意思。
<br/>　　秦怀玉性子过于软了些，等到若干年后老秦仙去必然无法挑起秦家的大梁，到时候秦家只怕不会再有眼下的风光。
<br/>　　可是秦怀玉的性格就是这样，老秦也没有什么办法，总不能把这个儿子掐死再生一个吧？而且若是再生一个还是这种软性子呢？难道还能都掐死了？
<br/>　　想到这里，秦琼不由叹了口气：“儿孙自有儿孙福，此事随缘吧！”
<br/>　　随缘么？李承乾眨了眨眼睛，突然开口说道：“小侄对此事倒是有些想法。”
<br/>　　“殿下有什么话尽管直说！”秦琼人虽然直了些，但却不傻，见李承乾一再提起儿子，又提到秦家的未来，心中多少有了一些猜测。
<br/>　　李承乾也觉得前面的铺垫差不多了，当下也不再犹豫，直接开口说道：“秦叔叔可愿与小侄结个亲家？”
<br/>　　“啊？亲，亲家？”显然，老秦刚刚猜的和李承乾提出来的建议并不一样。
<br/>　　“不错，琅琊这丫头虽然比怀玉小了几岁，但却不失为一对，秦叔叔认为呢？”
<br/>　　秦琼没有说话，脑子里却转着一个十分不靠谱的念头。
<br/>　　要知道，老秦与李二是一个辈子，也就是说秦怀玉和李承乾是一个辈份，而琅琊那个小丫头却是李承乾的女儿，所以虽然两个娃娃年龄差不多，但却是差着一个辈份。
<br/>　　那就是说如果自己一点头，那两上娃娃真的在一起了，那等于是李承乾自动升了一辈，变成与自己同辈，也就是说跟他老子同辈。
<br/>　　这样一来，李承乾这小子到底跟他爹应该叫爹呢？还是叫哥呢？如果叫哥的话，会不会被老李把腿打折呢？这是一个让人很纠结的问题啊。
<br/>　　当然，如果李承乾不想被打折腿他还有别一个选择，那就是自己降一辈，跟自己闺女叫妹子，这样腿应该能保住，但是脸就不一定。
<br/>　　想着一些有的没的，老秦不由有些走神，直到李承乾在一边说道：“此事秦叔叔慢慢想想，小侄也只是突然间生出这么个念头，毕竟琅琊的外公便是卢国公，和您的关系也不远不是。”
<br/>　　……
<br/>　　李承乾把该说的都说完了，从老秦的家里出来便上了回宫的马车，留老秦一个人在书房里纠结他的目的。
<br/>　　而李承乾的目的是什么呢？老实说，他的目的十分简单，那就是牵制老秦的精力。
<br/>　　他先是提出秦怀玉性格上的缺点，使老秦不得不担心秦家的未来，然后又抛出自家闺女，这等于是给秦琼吃了一颗定心丸，至于说辈份上的事情，对于一个家族的存忘来说，并不重要。
<br/>　　李承乾之所以这样做，还是在为自己的未来上一层保险。
<br/>　　眼下长安城中老头子真正能依为心腹的人已经没有多少了，老秦是一个，老尉迟是一个，其他人基本上都是些会打小算盘的老鬼，李承乾并不怎么在乎。
<br/>　　也就是说只要他能稳住老秦，便等于又断了老头子的一条胳膊，凭老尉迟一个人就算是再厉害也翻不起大浪，到时候架空老头子的任务基本上也就算是完成了一大半了。
