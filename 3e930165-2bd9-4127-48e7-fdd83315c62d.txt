　　“咔咔咔咔……”
<br/>　　子午宿舍中除了云熙敲击键盘的声响外，只有偶尔从电脑音箱中传出来的警告声。
<br/>　　罗昊没有去打扰云熙，因为他清楚，云熙跟自己一样，当专注于一件事情的时候，也是一个疯子，根本不会在意自己身边发生了什么，就算是拿着一挺重机枪在她身边扫射，也不会影响到她的工作。
<br/>　　把放着子午那些专业书籍的纸箱进行打包后，离开了子午的宿舍。中午的时候，罗昊去食堂给云熙打了饭，带去子午宿舍的时候，云熙还在电脑前忙碌着，双手在键盘上飞快的敲击着。
<br/>　　罗昊把饭盒放在旁边的桌子上，等了半个小时，眼看着饭菜都要凉了，想要提醒云熙先吃饭的时候，云熙原本飞快敲打着键盘的双手突然停了下来，眼睛盯着电脑屏幕，眼中露出狂喜之色，“成功了，成功了！”
<br/>　　把端起的饭盒又重新放到桌子上，罗昊快步走到云熙旁边，问道：“什么成功了？”
<br/>　　“老师自己做的数据恢复程序。”云熙指着电脑上刚刚编写完成的程序，说道：“有了这个东西，就可以把硬盘里面之前被破坏的数据重新恢复。”
<br/>　　“能够完全恢复吗？”罗昊对云熙问道。
<br/>　　这个硬盘中的数据应该非常重要，要不然仙岛夫也不会要在启动了数据删除程序，再从试验基地中离开。
<br/>　　“这需要看数据的被破坏程度。”云熙说道：“如果是被完全删除掉的数据，那自然是没办法恢复的，但如果是遭到了严重破坏，但是并没有被完全删除的数据，这个程序可以把数据碎片进行恢复。”
<br/>　　“那还等什么呀，赶紧开始吧。”
<br/>　　这个时候，罗昊显得比云熙还要着急，他也很想看看，仙岛夫在这个硬盘中藏了什么宝贝！
<br/>　　“可是我现在饿了……”
<br/>　　云熙拖了一个长音，靠在椅子中，眼中露出狡黠之色。
<br/>　　罗昊把饭盒拿了过来，对云熙问道：“给你都准备好了，要不要我喂你？”
<br/>　　“不用！”
<br/>　　云熙想也没想，直接把罗昊手里的饭盒给抢了过来。她还能不知道自己面前这货在想什么？这喂着喂着就去做别的事情了，云熙都已经上过他好几次当了，这次如何会再上当？
<br/>　　云熙把一块鸡肉塞进嘴里一边啃着，一边对罗昊说道：“按下回车键，程序就会自己运行，恢复硬盘里的数据。”
<br/>　　罗昊按照云熙所说的敲下回车键，启动数据恢复程序，将硬盘中的程序进行恢复。
<br/>　　硬盘数据恢复是个很耗费时间的工作，电脑桌面上的百分比进度条，隔了有一分钟左右，才移动了百分之一。
<br/>　　云熙把一块鸡骨头吐在饭盒盖子上，对罗昊说道；“今天食堂是谁啊？这三杯鸡味道不错。”
<br/>　　“这哪儿是食堂做的，这是我去食堂特别给你做的。”罗昊回答道。
<br/>　　“真的？”云熙有些怀疑的看着罗昊，问道。
<br/>　　“当然是假的。”罗昊脸不红、心不慌的回答道。
<br/>　　“罗！小！昊！”
<br/>　　云熙咬牙切齿的瞪着罗昊，恨不得冲上去狠狠咬他几口来解恨。
<br/>　　两个人把子午宿舍里的东西都整理干净，又装了两个大纸箱，宿舍里只剩下了桌子上那台还在工作的电脑。
<br/>　　又等了差不多一个小时的时间，数据恢复程序完成了恢复工作，看着程序窗口百分之一百的进度条，罗昊心中有些许激动，原本以为这个硬盘被破坏的如此严重，里面的东西不可能再被恢复；可是子午留下的这个只差最后一小步就完成的数据恢复程序，将这个硬盘中的数据恢复了！
<br/>　　虽然无法将那些已经被彻底删除的数据进行恢复，但是罗昊相信，或许真的能从这个硬盘中找到一些有关于“暗月”不为人知的秘密！
<br/>　　就在罗昊准备动手打开硬盘的时候，云熙却是突然伸手抓住了罗昊握着鼠标的右手，说道：“既然数据恢复了，我们还是先离开这里吧。硬盘里的东西拿回去看也一样。”
<br/>　　罗昊知道在这里对云熙的触动很大，点了点头，从电脑上取下了硬盘，关掉了电脑，把电脑搬进一个纸箱子里，拿出宿舍，重新把门锁好后，离开了宿舍楼。
<br/>　　开车把子午宿舍中整理出来的东西带回家，放进杂物间后，罗昊和云熙回到他们的房间，把硬盘连上电脑，打算从里面找出一些有价值的东西来。
<br/>　　硬盘一打开，罗昊就看到了在最显眼的位置，有几个视频文件，刚把文件打开，白花花的肉体就出现在视频播放器中，紧接着就有着“嗯嗯啊啊”引人遐想的声音传出来。
<br/>　　云熙面红耳赤，伸手就把视频给关了，瞪了眼还盯着电脑屏幕的罗昊，骂了句“色狼”！
<br/>　　罗昊觉得自己给仙岛夫那混蛋背了锅，这东西明明就是仙岛夫留在电脑硬盘中，凭什么云熙骂他是“色狼”？！
<br/>　　罗昊移动鼠标，点开第二个视频文件，里面还是一样不堪入目的东西；第三个，第四个……，罗昊一连点开四个视频全都是类似的内容。
<br/>　　我勒个去！
<br/>　　罗昊心中有种要骂街的冲动，仙岛夫这货不会是在核心实验室的电脑里存的都是这种18X限制级内容吧，自己冒着被炸死的危险，拼命阻止数据删除程序，从电脑上拆下来的硬盘里面存的都是这种东西吧？
<br/>　　那特么的自己也太亏了，就为了这种在网盘上多得是东西，差点把自己的命都搭了进去！
<br/>　　“你来？”
<br/>　　罗昊松开鼠标，看了眼坐在自己身边，俏脸绯红的云熙。
<br/>　　云熙承认自己跟罗昊认识相恋后，被罗昊“带坏”了很多，可是刚才在视频里出现的东西，还是让云熙整张脸不住的发烧，这里面出现的东西，对于云熙来说，都是存在于传说之中的……
<br/>　　云熙摇头，她可不想点开视频以后，发现里面还是这种突破她认识的限制级内容。
<br/>　　最后，还是罗昊打开了硬盘中的第五个视频文件……
