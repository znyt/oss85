　　第591章
<br/>　　戎族王城，大殿之内。
<br/>　　“长老，遵照您的吩咐，我们将书取回来了。”
<br/>　　柳褐岩毕恭毕敬将《山水经》取出，双手捧给桌案后的老人。
<br/>　　对方头发花白，脸上有稀稀拉拉的胡须，从外表看上去，倒是猜不出有多大年纪。五十也像，说七十也有。
<br/>　　这位便是呼延族的大长老，算是戎族的国师，当然，戎族还没发展到建国的那一步，于是大家对他都以大长老相称。至于他本来叫什么名字，则是无人知道。不过，有小道消息说，此人并不是土生土长的戎族人，而是从帝国来的。
<br/>　　当然，这个消息没被任何人确认，这么说的人，渐渐也都消失了。于是，也就没人这么说了。
<br/>　　大长老将《山水经》翻开，帝国十三州一百单八城水文地理，便都在这本书上了。逐字用蝇头小楷书写，上面还绘着精细的小图，书院穷几十年人力物力，可以说书上的每个字，都是无数人心血凝成。
<br/>　　但如今，这本书不在书院，不在帝国长安，而是在戎族王城，一个来历神秘的老人手里。
<br/>　　“果然在这里了。”老人粗略翻过，轻轻将书合上：“你们做得不错，这件事属于机密，不需外泄，懂么？”
<br/>　　“明白。”柳褐岩道：“没有大长老吩咐，我们不会同任何人说。”
<br/>　　柳褐岩顿了顿，道：“还有一件事，属下要向大长老禀报。”
<br/>　　老人没有说什么，安静翻着面前的山水经。柳褐岩大着胆子，便将燕不归的事情说了，倒也没有什么隐瞒，不过仍有些提心吊胆，毕竟，那燕不归还有黑贞，都是大长老的亲传弟子。
<br/>　　也不知道他听到这个消息后，会如何反应。
<br/>　　“这样说……燕不归死了？”
<br/>　　“黑贞姑娘已遭了帝国人的毒手，燕先生拼着命要找书院报仇，怕已是九死一生。”柳褐岩提着小心道。
<br/>　　“你们下去吧。”
<br/>　　柳褐岩还欲再问，突然闭上了嘴。他心里想不明白，大长老是生气了还是没有生气呢，不过，大长老既然没有说责罚自己，那自己也不用多嘴讨罚。
<br/>　　他带着人静悄悄出了大殿，没敢再打扰大长老。但心里还是七上八下的，大长老的确值得认真对待，更可怕的，却还是生死不明的燕不归。
<br/>　　燕不归是什么脾气，大家都有了解。在戎族这个好勇斗狠的地方，他的脾气都是有了名的。柳褐岩倒是盼着燕不归死在书院手中，他如果活着，绝对不放过自己。
<br/>　　柳褐岩猜得不错，如此说来，他倒算是燕不归的知己。
<br/>　　燕不归从白沙岛跳海后，和程大雷推测得一样，肯定不会溺死在大海中。他被一条商船所救，一路顺风顺水的上了岸。之后，便扛着自己的重剑，气势汹汹往草原来了。
<br/>　　他回戎族目的就是杀人。
<br/>　　燕不归不善言辞，沉默寡言的人，往往用情至深。认定了一条路，便会一根筋走到黑。在他想来，自己的仇人都死得差不多了。张六被自己砍了，书夫子也死了，程大雷的女人也被自己杀了……剩下的，也就是柳褐岩一帮人。
<br/>　　黑贞死后，燕不归但觉天下人人皆可杀，更何况，还是柳褐岩这种有明显过错的人。
<br/>　　他在琅琊城耽误的时间不少，靠岸之后，便沿着柳褐岩走过的路奔来戎族。程大雷因为一路上遭遇书院的围追堵截，所以又跟在他后面。
<br/>　　而书院绝不会轻易放过程大雷，跋山涉水，在所不辞，所以他们又跟在程大雷身后。
<br/>　　这一日，燕不归过了青牛山，进入草原，算起来，这该是戎族北蛮部的地盘。
<br/>　　天快黑了，他点燃了一堆篝火，猎了几只野物裹腹。如今已到了草原，算是回了家，再有几日，便能回到戎族王城。
<br/>　　想起柳褐岩一波人，燕不归便咬牙切齿，等他回了王城，肯定一剑一个，全部把他们砍了。
<br/>　　忽然听得脑后一阵腥风，燕不归猛地回过头，见一头黑熊从密林中钻了出来。
<br/>　　燕不归晃了晃肩膀站了起来，咧嘴笑道：“好畜生，正好与我打打牙祭。”
<br/>　　天是越来越冷，人在屋里窝着不出门，畜生也是一样。到这个季节，黑瞎子往往窝在树洞里开始冬眠。看这头黑熊的样子，该是想把燕不归当食物，用以熬过这个冬天。
<br/>　　偏偏，燕不归也是个吃肉的主儿。
<br/>　　他双手握住重剑，盯着那黑熊的动作，黑熊突然扑了过来，来势汹汹。燕不归知道，这等畜生，经常在大树上磨皮，磨得皮糙肉厚。要么不出手，出手定要一击致命，否则，一头发狂的黑熊更加危险。
<br/>　　燕不归本身就是个半疯性格，他当然明白疯子的可怕。
<br/>　　当黑熊扑过来时，燕不归立刻跃起，几个纵跃，便立在一棵大树的树干上。惹得黑熊在树底下发出嗷嗷的咆哮，燕不归看了仔细，忽然从高处跃下，重剑扎的是黑熊的眼睛。
<br/>　　那黑熊发出一声痛苦的咆哮，猛地将燕不归扑在地上。燕不归心里一激灵，委实大意了，他一路风餐露宿，在琅琊城又受了不小的伤，他的力气，已经不如从前。
<br/>　　当真是敌得过天下英雄，敌不过一头黑狗熊么。
<br/>　　一柄羽箭凭空而至，嗖地一声，扎入黑熊另外一只眼睛中。在死亡关头，燕不归被逼出潜力，抓住剑，一剑将黑熊开膛破肚。
<br/>　　他浑身沾满腥臭的鲜血，立起来长舒了口气，刚才委实是在鬼门关走了一遭。
<br/>　　这时候他才想起那柄羽箭，抬头望过去，只见半山坡上，立着一个年轻人。模样倒也是平平无奇，只不过一只眼蒙着黑眼罩，看着有些诡异。
<br/>　　此刻，那只独眼就盯着自己，燕不归从小到大，确实是没怕过谁。但被这只眼睛盯着，心底却一缕缕冒寒意。
<br/>　　对方没有开口的意思，燕不归也算个深沉的人，此刻却被这沉默压得喘不过气来。
<br/>　　“谢了，你是那族的汉子？”燕不归率先打破沉默。
<br/>　　独眼年轻人慢吞吞开口：“你本事不错，愿意跟着我么？”
<br/>(本章完)
