        只见童筱颖再次回到他身边的时候，手里多了一个木制玩具。
<br/>
<br/>        “孔明锁？！”傅昼景诧异。
<br/>
<br/>        童筱颖随之将手里的孔明锁递给了傅昼景：“这是我给尚轩做的玩具，你帮我带给他。”
<br/>
<br/>        “很小巧也很精致。”傅昼景惊赞道，接过童筱颖递过来的孔明锁，看了看后，塞入了裤袋中。
<br/>
<br/>        “过了今晚，以后，我们都不要再见面了！”童筱颖将下巴抵在膝盖上，声音冷淡地说。
<br/>
<br/>        傅昼景脱下自己身上的西装外套，轻轻地披到了她肩膀上。
<br/>
<br/>        闻到属于他身上的这股熟悉的气息，童筱颖鼻头一酸，眼泪差点就快忍不住要掉了出来。
<br/>
<br/>        童筱颖突然站起身来，将肩上的衣服拿下，递给他，情绪上有些激动：“我想要的温暖，已经不是你能给予的了。不要再做一些无谓的事情。”
<br/>
<br/>        她不希望他死，但也不希望他在她的心死了后，对她做出这些关爱的动作。
<br/>
<br/>        傅昼景怔愣地接过衣服，沉默不语。
<br/>
<br/>        “我先去睡了。”
<br/>
<br/>        她淡淡地说，却没有再跟他说晚安。
<br/>
<br/>        “晚安”的拼音是“wanan”，也是“我爱你爱你”这句话的拼音首字母的缩写。
<br/>
<br/>        结婚后，她有每天都坚持跟他说晚安，不是短信就是电话，当面时，她会亲口说。
<br/>
<br/>        现在，她不说了。
<br/>
<br/>        傅昼景知道这意味着什么。
<br/>
<br/>        童筱颖下去后，他独自一个人坐了一会儿才从屋顶上下来。
<br/>
<br/>        这屋子里，就一张大床。
<br/>
<br/>        那张木板床，靠着墙壁，墙有多长，床就有多长。
<br/>
<br/>        小时候，他和她，还有外婆在那张大床上一起睡过。
<br/>
<br/>        现在，还是那张大床，很宽敞，只是物是人非。
<br/>
<br/>        他走过去的时候，她已经睡着了。
<br/>
<br/>        傅昼景小心翼翼地在童筱颖的身旁躺下来，然后拉黑了屋里的灯泡。
<br/>
<br/>        不知不觉中，他将她搂入了怀中，好像回到了他们在澳洲时生活的那段日子。
<br/>
<br/>        她身体很软，抱在怀里很香。
<br/>
<br/>        他依稀记得，曾经她身上有股淡淡的柠檬香气，后来去了澳洲后，那里牧场很多，家里每天都可以喝到牧民送过来的新鲜牛乳。
<br/>
<br/>        她的身上，也染上了牛奶的香气。
<br/>
<br/>        但现在，她的身上，让他怀念的那股牛奶香气，已经不复存在。
<br/>
<br/>        翌日，天才蒙蒙亮。
<br/>
<br/>        一阵急促的敲门声，将睡梦中的两个人惊醒。
<br/>
<br/>        两人还未来得及起床，门口便传来“哐啷”两声巨响。
<br/>
<br/>        童筱颖一脸迷蒙的揉着惺忪的睡眼，往门口望去，模糊的视野里，只见一个女人突然冲了过来。
<br/>
<br/>        “童筱颖，你这个贱女人！”季思妍先声夺人，疾步走到床边，扬起手来。
<br/>
<br/>        到她的手腕却在半空中被一一只强劲而有力的大手捉住。
<br/>
<br/>        “大哥！”傅昼景如梦惊醒，从床上坐起，震撼地看着傅夜沉抓着季思妍的手腕往旁边一甩。
<br/>
<br/>        童筱颖也彻底清醒过来，看到傅夜沉时，瞬间瞪大了眼睛。
<br/>
<br/>        “童筱颖，你这个贱女人！你这贱女人！你为什么要带走我的阿景！带走小景的爸爸，让他失去父爱！你这个女人到底是有多贱？离了婚，还能跟前夫上床！”季思妍再次踱步上前，哭得歇斯底里，指着童筱颖的鼻子破口大骂。
<br/>
<br/>        傅昼景几乎是本能地将童筱颖护在身后，担心季思妍会发疯伤着童筱颖。
<br/>
<br/>        童筱颖也趁机躲在傅昼景背后的阴影里，将目光挪到一旁，不敢直视傅夜沉的眼睛。
<br/>
<br/>        完了完了，她怎么又有一种被捉奸在床的感觉啊？
<br/>
<br/>        还有！
<br/>
<br/>        她昨晚明明是一个人睡的，傅昼景怎么就也跟着睡到她身边来呢？
<br/>
<br/>        “童筱颖，我要杀了你！”季思妍见傅昼景护着童筱颖，彻底失去了理智，伸出双手成钳子状，龇牙咧嘴地朝童筱颖扑了过去。
<br/>
<br/>        傅昼景立即从床上下来，抱住季思妍，阻止她再继续发疯：“思妍，别闹了！我跟你回去！”
<br/>
<br/>        “我不相信！我不相信！这个贱女人不死，你的心就永远不会在我这里！”季思妍双手紧紧地揪住傅昼景的衣领，布满血丝的眼睛，瞪得眼珠子都快要蹦出来了般吓人。
<br/>
<br/>        “大哥，帮我照顾一下筱颍。我先带思妍回去了！”傅昼景说着，将季思妍打横抱了起来，疾步离开了屋子。
<br/>
<br/>        童筱颖的目光，也追随着傅昼景的背影而去。
<br/>
<br/>        刚刚，那个女人，真的是她所认识的那个季思妍吗？
<br/>
<br/>        直到傅昼景抱着季思妍的身影消失在大门的转角，童筱颖才收回自己的目光。
<br/>
<br/>        “需要我替你放鞭炮吗？恭喜你，再一次被自己所爱的男人抛弃！”屋子里安静后，傅夜沉面无表情地盯着童筱颖，冷冷地说。
