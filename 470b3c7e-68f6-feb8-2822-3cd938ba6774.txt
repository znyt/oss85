　　道渊真人沉吟道：“应该不是，恕我直言，桃花山只是一个中等门派，无论是功法还是法器等各种方面，都没有办法与几大名山相比，风水也就那样，不大可能出现什么特别的强者。其次，桃花山既然敢发出法帖，召集天下同门，这么大的手段，就算出了强者，也得是灵仙牌位，我不相信桃花山能孕育出灵仙。”
<br/>　　毛小方也不假思索地回答：“这不可能。”
<br/>　　两个人一起看向叶少阳，叶少阳却是沉吟不语，半晌说道：“两位师兄，在以前，法术界有没有这个传统？”
<br/>　　两人一起摇头。
<br/>　　“别说没这个传统，首席弟子这个说法以前也是没有的，不知道桃花山为什么突然有这种提法。”
<br/>　　叶少阳道：“在我那个时代，是有的。兄弟我就是法术界首席弟子……虽然后来总被人否定，我自己也没当真。”猛然想到什么，一拍脑门，问道渊真人：“你说的那个陌生人出现在桃花山的时间，是多久之前？”
<br/>　　道渊真人想了一下道：“听说是有半个月左右，不到二十天。”
<br/>　　“不到二十天……”叶少阳喃喃道，“那不真是我来到这个世界的时间吗，怎么会这么巧，而且首席弟子这个说法，的确是有些超前……道渊师兄，桃花山在什么地方？”
<br/>　　“在杭州。”
<br/>　　“我要去杭州！”叶少阳斩钉截铁地说道，并且道出了自己的怀疑，他也摸不准具体是什么情况，但总感觉桃花山的这次行动、还有那个突然多出来的神秘强者，跟自己似乎有某种关系。
<br/>　　四个人一起商量了下，最后觉得，叶少阳去了根本没有：他在这个世界没有身份，估计根本见不到对方，而且山高路远，不是说去就能去的。
<br/>　　“还是我帮你走一趟吧。”道渊真人道，“我虽然在家族里不受重视，总算是龙虎山天师，有这身份，我去应该好说话一点。我帮你走一趟，调查一下这个人的身份。”
<br/>　　叶少阳犹豫了一下，道：“那多不好意思？”
<br/>　　道渊真人笑道：“你我是兄弟，有什么好客气的。”
<br/>　　叶少阳嘴角抽搐了一下，说道：“你是我老祖……”
<br/>　　道渊真人大笑，道：“你也不必多想，我也不光是为你去，我们龙虎山也想弄明白是怎么回事。等我调查之后过来找你，将来我们一起去参加龙华会。”
<br/>　　“龙华会什么时间？”
<br/>　　“下个月初八。”
<br/>　　叶少阳点了点头，又谢了道渊真人。
<br/>　　道渊真人到卧房里去睡了一觉，然后就走人了。叶少阳三人送他走的时候，在街上听到了爆炸性的消息：五通神被阴司抓走了。
<br/>　　叶少阳等人打听得知，一切都跟他们安排的差不多，五通神被阴司抓走的消息，是吴巧带回来的，当地人不信，去蛤蟆峰上找到陈老汉，结果得知了他被阴神托梦的结果……
<br/>　　很多老年人大哭大闹，跌足长叹，对未来的生活充满了恐惧。
<br/>　　叶少阳心中很是感慨，人类就是这样一种动物，很多地方没有山神土地保佑，也可以过的好好的，这地方因为一向有山神保佑，突然没有了，就不知道怎么过了。
<br/>　　美华之前刚收编了三足金蟾的手下们，要赶回去安排他们，暂时先跟叶少阳告别了，叶少阳跟毛小方一起在街上逛了一圈，回客栈的时候，在店堂里遇到两个人，指名要找叶少阳。
<br/>　　叶少阳跟他们聊了一下才知道，这两人是吴巧的父亲派来的，想着可能是吴巧跟她父亲提到了自己，也没什么坏事，于是让毛小方呆在客栈等着，跟着过去。
<br/>　　路上跟两人聊天，叶少阳才知道，吴巧家是镇上的士绅望族，很有钱，在镇上充当文书，也只是做个闲职，想为镇上百姓做更多的事。
<br/>　　吴巧家在镇上北边一座山坡上，一个非常气派的三层宅子，叶少阳进去之后，被带到客厅，一个男子立刻上前跟叶少阳握手。叶少阳之前见过他，知道他就是吴巧的父亲，自我介绍叫吴天乐，很热情地请叶少阳去客厅，然后招呼人倒茶，跟叶少阳闲聊起来。
<br/>　　叶少阳也不知道他找自己来的目的，就跟他应付了几句，过了一会，吴天乐支使走了佣人，让叶少阳稍等，自己去了内室，不一会把吴巧带了出来，然后把两边的门都关好，叶少阳还没明白怎么回事，父女二人就一起跪在了叶少阳面前。
<br/>　　“感谢叶先生救我女儿！”吴天乐十分激动，要给叶少阳磕头，叶少阳吓了一跳，赶紧过去把他们父女俩扶起来，看了吴巧一眼道：“我不是让你不要告诉家人吗？”
<br/>　　“那我忍不住……”吴巧红着脸喃喃说道。
<br/>　　吴天乐低声道：“叶先生放心，为了我女儿，真相我绝对不说。感谢叶先生为小女保存贞洁，大恩不言谢，我吴某人什么也不说了，但又所用之处，万死不辞！”
<br/>　　叶少阳扶起他，说道：“我听你说话也是读书人，不过之前带五通神选择姑娘的是你吧，结果选中的是你家女儿，如果选中别人，你这会儿估计幸灾乐祸呢。”
<br/>　　“幸灾乐祸不敢，不过……真的是只有论到自己，才明白其中痛苦……”吴天乐叹了口气，随即自嘲地笑了笑，“这次真是让我茅舍顿开，与其依靠山神，其实更应该依靠的是自己，叶先生，我打算买下那座山，建成一个新式的学堂，让镇上的孩子去上学，这些孩子有了文化，将来可以回报给大家的，会比一尊山神有用的多。”
<br/>　　叶少阳对他这个想法给与了相当的评价和支持。
<br/>　　晚上吴天乐留叶少阳吃饭，叶少阳推辞不得，只好让他派人去把毛小方也叫来了，三人把酒言欢，席间吴天乐问起他们两个婚配的情况，两人互相看看，都明白吴天乐的意思，叶少阳急忙说道：“我有对象了，毛师兄是孤家寡人，正愁着没人跟他来着。”
