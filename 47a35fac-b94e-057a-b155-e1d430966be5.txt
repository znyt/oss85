　　“咔嚓！”
<br/>　　天羽城的无数残垣断壁，都在这力量冲击之下，纷纷断裂。
<br/>　　两道身影，同时飞了出去。
<br/>　　楚风眠跟那盘使用，都是退后了三步，才稳定住了自己的身形。
<br/>　　“看来七大宗门中，倒是还有些人物。”
<br/>　　一招平手，楚风眠看向那盘师兄，也是自言自语了一声。
<br/>　　他自从度过生死大劫，实力暴涨，便是一路上还没有碰到敌手。
<br/>　　如今这盘师兄，算得上是第一个，足以让楚风眠动用全力的敌人了。
<br/>　　眼前这盘师兄的实力，面对什么夜天君，冷面魔君等人，简直都是可以秒杀。
<br/>　　七大宗门的底蕴，果然还是很强，除了楚风眠，也只有七大宗门中，才能出现如此天才。
<br/>　　楚风眠的眼神中，露出几分凶光。
<br/>　　不过，在天才的人，如今既然是招惹到了楚风眠，便只是一个死人。
<br/>　　这盘师兄的实力，就算是在强大十倍，今日一样得死。
<br/>　　“恩？”
<br/>　　同时，盘师兄的一双眼睛，也是在打量着楚风眠，眼神中似乎微微有些惊讶。
<br/>　　有些没有料到能够有人，就这抵挡得住他的一掌。
<br/>　　一般武者，就算是手持灵器，想要地挡下他的一掌，都不可能，唯有同样七大宗门的弟子，还要是七大宗门弟子之中的佼佼者，才能抵挡得住。
<br/>　　但是若是这样，徒手之下，抵挡住的。
<br/>　　九域之中，年轻一代中，还从未有过。
<br/>　　“小子，你很强，我不愿意杀你，现在你跪在地上，向我磕头臣服，我可以饶你一命，不然的话，今日你就要死在这归墟天中了。”
<br/>　　盘师兄看着楚风眠冰冷的开口道。
<br/>　　“你的实力，虽然不错，但是不管是你的剑术，还是你的身躯，都不可能是我的对手，不臣服，今日你必死！”
<br/>　　盘师兄开口，言语之中毫无感情，但是他整个人，都是散发出了一种高高在上的气势，一双眼睛，如同看向一个蝼蚁。
<br/>　　只要是这蝼蚁，稍不听话，就可以直接踩死。
<br/>　　“实力不强，口气不小。”
<br/>　　楚风眠淡然开口道。
<br/>　　“这句话，我也可以送给你，不过你们七大宗门的人，就算是跪下向我求饶，祈求当我的奴隶，我也不会收你们，你们没有这个资格，对你们来说，死亡，才是唯一的归宿。”
<br/>　　楚风眠的话，其中的嚣张程度，比起眼前这盘师兄话中的嚣张，更是不止一倍。
<br/>　　周围的武者，听到楚风眠的话，一个个都有些目瞪口呆。
<br/>　　这话，狂妄到了极点。
<br/>　　刚刚楚风眠放话，便是要将这归墟天中，七大宗门的所有弟子，都给斩杀。
<br/>　　如今更是说这盘师兄，就连做他奴隶的资格都没有。
<br/>　　敢在七大宗门弟子面前，说出这种话，整个九域，都从来没有过。
<br/>　　“好，够狂妄！”
<br/>　　盘师兄脸色阴沉到了极点，话语之中的杀意，已经是掩饰不住了。
<br/>　　这一个“好”字，听起来却根本不是在称赞。
<br/>　　而是如同“杀”字一样。
<br/>　　杀气冲天，要将楚风眠碎尸万段一般的杀意。
<br/>　　“一个小小蝼蚁，也敢跟我们七大宗门如此挑衅，很好，小子，你罪该万死，今日我就杀了你，将你的尸首炼制成一件灵器，让所有人都知道，敢挑衅我们七大宗门的下场！”
<br/>　　盘师兄的身上，金光闪耀，身上的气势都在疯狂攀升，显然也是在施展这圣王宗的秘术。
<br/>　　“圣王天怒！”
<br/>　　一瞬间，这盘师兄身上的气势，便是一直达到了定点，突然出手。
<br/>　　这盘师兄身上的气势，他的武道，将他整个人化为了一颗太阳一般。
<br/>　　他的两个拳头，便是如同两颗太阳一般，从天而降，向着楚风眠轰击过去。
<br/>　　空间都在颤动，一路上的空间，都要给点燃，比起火焰更加可怕的，便是这一拳的力量，足以粉碎一切的力量。
<br/>　　“这种圣王宗中的秘术，都给动用了，看来这小子，真的是嚣张到头了。”
<br/>　　“敢跟七大宗门挑战的人，都已经是死人了，这小子也快了。”
<br/>　　看着盘师兄出手，不少围观武者，都已经心中认定，楚风眠已经是一个死人了。
<br/>　　在这种力量之下，如何抵挡？
<br/>　　除非是超越生死的境界。
<br/>　　不然这种力量在，在强大的武者，都要被直接碾碎吧。
<br/>　　“圣王天怒？也没什么了不起的，圣王三书中，你不过是学到了一些皮毛罢了，还敢在我的面前耀武扬威？”
<br/>　　楚风眠抬起头，看着这盘师兄两个拳头，如同两颗巨大无比的烈日一般落下，他的脸色，一如既往的冷静。
<br/>　　只是从楚风眠的手心中，一道金光闪过，祖龙至尊剑，突然出现在了他的手心中。
<br/>　　这盘师兄，是楚风眠在这归墟天中碰到过的最强武者。
<br/>　　狮子搏兔也要动用全力。
<br/>　　这盘师兄，也的确有资格，要楚风眠动剑。
<br/>　　“破！”
<br/>　　楚风眠单单吐出一个字，这一个字的声音，却是传遍了整个归墟天。
<br/>　　随着这一个“破”字，楚风眠手中的剑锋动了。
<br/>　　剑光一闪。
<br/>　　他手中祖龙至尊剑，猛然一动，凌空而起，斩出了一道剑锋。
<br/>　　这一道剑锋，只有七尺之长，在空中，几乎算得上是毫不显眼。
<br/>　　“这样的剑锋，也想要跟盘师兄抗衡？”
<br/>　　“可笑，本以为此人的剑术可以达到什么地步，原来不过如此。”
<br/>　　“这样的剑锋，去跟这盘师兄抗衡，不是找死？”
<br/>　　看着楚风眠打出的剑光，那几名圣王宗弟子，都是不由的大笑起来。
<br/>　　单单是在这威势上，两者都简直不是一个层次。
<br/>　　盘师兄的两拳，如同两颗太阳一般，从天陨落，这等气势，将整个归墟天都给笼罩。
<br/>　　而楚风眠斩出的剑锋，却是如此的平平无奇。
<br/>　　“轰！”
<br/>　　这一道剑锋，跟盘师兄的双拳，凌空猛然的碰撞在了一起。
<br/>　　在一瞬之间，楚风眠的剑锋，几乎都被烈日吞噬。
<br/>　　“哼，不过如此……”
<br/>　　一名圣王宗弟子，看到这一幕，不屑的开口道。
<br/>　　但是他的话，还没有说完，他的表情，突然呆滞起来。
<br/>　　只见在空中，那两道烈日，居然是被直接切开，在这一道剑锋面前，被切成了两半。
