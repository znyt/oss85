　　小男孩舔了舔嘴唇，有些犹豫起来，嗫嚅道：“我们之前不是打过交道么，我是……通灵佛子花。”
<br/>　　什么！
<br/>　　叶少阳大惊，怔怔地看着这孩子。他早知道这孩子是邪物，邪物千变万化，这个倒是不假，但是叶少阳怎么也不能将那可怕的通灵佛子花，和面前这个孩子联系起来。再说……“通灵佛子花，不是被我烧了吗？”
<br/>　　“哦，我是花种子，你烧它的时候，我就已经离开了，不过我怕被你们发现，抓到我……你这阴阳镜能隔绝空间，所以我就躲进去了……”小男孩抬头看着叶少阳，可怜兮兮地说道，“求你不要杀我，我愿意给你当仆人，主人……”
<br/>　　“你妹的，你先别攀亲，我说，你是不是变成小男孩故意来装可怜，这不是你的本尊吧？”
<br/>　　“我的本尊，就是一株花草，不过我刚成形，化作人形，就是这个样子。主人，你就收了我吧。”
<br/>　　叶少阳瞪着他，说道：“你一个花种子，有这么强大的法力，可以幻变？”
<br/>　　花童道：“不是这等说，我从之前那株花上，继承了它的灵力，幻变还是可以做到的。”
<br/>　　叶少阳还是头一回听到这种事，一时间有些接受不了，怔然无语。
<br/>　　花童看他这样，于是解释道：“说起来，我也不算是种子，通灵佛子花，不是普通的树木花草，我也不是结出来的种子，说起来，我是它的一部分……”
<br/>　　花童描述了半天，叶少阳总算是听明白了。“我知道了，你是孢子。”
<br/>　　“包子？”小男孩眼睛眨巴了半天，道，“我不是包子啊，我是花……”
<br/>　　“我说的不是那个包子。”叶少阳突然想起来桌子上还真有之前没吃完的油煎包子，找来吃了两个，对小男孩说道：“我说的不是这个包子，嗯，跟你解释不清楚，反正我自己是明白了。”
<br/>　　通灵佛子花，虽然是邪物，但本质上也是植物——不过照小男孩的说法，应该是一种菌，通过孢子的房子繁衍后代，在自身遇到危险、被毁灭之前，会分裂出类似孢子一样的东西恶，生而为灵，成为新的一株通灵佛子花。这也算是一种生命的延续方式。
<br/>　　怪不得通灵佛子花被燃烧的时候，就那样让他们烧死，当时还觉得死得有点太快了，也没反抗什么的，原来它已经偷偷分裂出了孢子，让自身用另一种方式存活下去。
<br/>　　“要说你是孢子，我也能理解，可是你为什么会说人类语言，表现得也像个人类一样？”
<br/>　　“这个……之前那株花的记忆，我是可以继承的，所以他所掌握的东西，我也全都掌握了。”
<br/>　　“这样。那之前那株花，又是怎么知道这些的？”
<br/>　　花童犹豫了一下，说道：“只要是被它用幻象杀死的，不光血肉，连神识也会被部分吸收，就好像记忆一样。它杀过一些人，所以拥有一些他们的记忆，自然也懂得了很多事情。”
<br/>　　叶少阳顿时感到匪夷所思，这通灵佛子花，居然能从人的神识中吸收记忆和知识……这一点实在可怕。
<br/>　　“这么说，通灵佛子花之前杀人，你也有份了？”
<br/>　　花童听他这么说，立刻摆起了手。“不不不，在分裂出来之前，我只是它身体的一部分，也只有在分裂的一刻，我才拥有自己的神识，之前它所作所为，不能算在我身上啊。而且，它也是被人从西域移置到那里，沾染了尸气，才会对人的血肉产生兴趣，我虽然知道这一切，但我就是我。我不会照他那样做的，所以啊，我想认你当主人，用正修的方法来证道，希望主人成全呢。”
<br/>　　叶少阳心中一动，道：“为什么选择我？”
<br/>　　“因为，我的前身，之前那株花是死在你手上。”
<br/>　　“你不恨我？”
<br/>　　“不，它多行不义，死得其所。而且，如果没有它的死，怎么会有我的重生。”花童冲叶少阳笑了笑，“我还没说完呢，之前它用幻境困住你们，两次，都是你堪破幻境，我觉得你本事很大，跟着你，比较有前途。”
<br/>　　这花童说起话来，倒是挺接地气的。
<br/>　　叶少阳笑了笑，道：“我为什么要带着你？有什么好处？”
<br/>　　美华听到这，插了一句：“就是啊，你连我都打不过，要你有什么用啊。”
<br/>　　“我打不过你，那是因为在阴阳镜的空间里，我没法幻变成真身，手段没法施展……不然我们再来一次试试！”
<br/>　　“试试就试试，怕你不成。”美华说着就要打开他全身的禁制，突然想到什么，“不行，万一你个兔崽子跑了怎么办。”
<br/>　　“我不跑！”
<br/>　　“没事你松开，他跑不了。”叶少阳道，自己现在法力恢复了，就算这花童真的想跑，也绝对得逞不了。
<br/>　　美华只好给他解开禁制。
<br/>　　美华抱着双臂，道：“你先动手吧。”
<br/>　　花童爬到桌子上，抓起茶壶，咕噜噜灌了几口，一副十分满足的表情，然后往桌子上一坐，整个人迅速产生了变化，变成了一株草，垂下来的两条腿变成了根系，身体变成了花苞，两条手臂变成了叶片，轻轻扇动，对美华说道：“来吧。”
<br/>　　一股幽风，迎面吹了过来。
<br/>　　叶少阳立刻闻到了那股熟悉的香味，顿时警觉起来，往后退了几步，轻轻咬住舌头，打算万一有不对劲就用力咬下去。
<br/>　　“小兔崽子，我可来了！”美华冲到花童面前，伸手去抓他，一把就抓住了，运用修为，将它牢牢困在手里，狞笑道：“我看你怎么办！”
<br/>　　突然，手中的花叶枯萎，顺着指缝落了下去。
<br/>　　美华大惊，不会自己这一下把它抓死了吧？想来花童也不可能这么脆弱。美华怔怔的发了会呆，转头去问叶少阳：“老大，这是怎么回事？”
<br/>　　“不知道，可能跑了吧，不管他了，我好困，我要睡一会。”叶少阳说着走到床上去躺着，冲她笑了笑，“我肩膀有点酸，你能不能过来给我按按？”
<br/>(本章完)
