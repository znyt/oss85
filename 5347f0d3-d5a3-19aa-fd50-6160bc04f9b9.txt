　　几乎只是瞬息之间，巫森便大变了模样，变成了一个高大的人形妖兽。
<br/>　　只见巫森的身躯高达一丈有余，全身钢针一般的毛发漆黑如墨，寒光森森。
<br/>　　他的双手，已经变成了狼爪，十根尖锐锋利的指甲，长达一尺有余。
<br/>　　而巫森的面部，也已经大变了模样，虽然没有完全变成狼头，但也变得非常狰狞，恐怖异常。
<br/>　　“兽化？”苏莫双眸微眯，紧紧的盯着巫森，对方现在的模样分明就是兽化了。
<br/>　　不过，他知道，这绝对不是兽化这么简单，因为他感觉到，对方身上的气息在急速暴涨。
<br/>　　这是本质上的改变，对方的实力暴增了一个大层次。
<br/>　　“苏莫，现在本帝就让你看看，你与武帝之境的差别！”
<br/>　　巫森冷喝一声，随即虚空踏步，一步千丈，大步的向苏莫走去。
<br/>　　苏莫面色凝重，二十四柄本命灵剑在他周身盘旋飞舞，如同条条灵蛇，随时可以展开犀利的攻击。
<br/>　　咚！咚！咚！！
<br/>　　闷响连连，巫森每走一步，便是一声闷响，其脚下的虚空不断塌陷、爆裂，出现一个个巨大的黑洞。
<br/>　　不多时，巫森便接近了苏莫十里之内。
<br/>　　这个时候，苏莫便率先出手了，他可不会坐等对方攻击。
<br/>　　心神一动，二十四柄本命灵剑齐动，瞬间激.射而出，布成了二十四星宿剑阵，将巫森笼罩在了其内。
<br/>　　巫森面无表情，大步前进，仿佛毫不在意苏莫的星宿剑阵。
<br/>　　“杀！“苏莫厉喝一声，刹那之间，二十四柄灵剑光芒夺目，气势连成一片，齐齐向巫森暴刺而去。
<br/>　　唰！
<br/>　　巫森终于动了，身形瞬间下沉，向下方坠.落而去。
<br/>　　因为向下方而去，他就能避开灵剑的围杀，方便出手抵挡。
<br/>　　咻咻咻！！
<br/>　　二十四柄本命灵剑浩浩荡荡，如同剑气长河，裹挟玄妙的星宿之力，齐齐向下方的巫森当头刺去。
<br/>　　“破！”巫森一声暴喝，双爪齐出，连连挥动，刹那之间，漫天爪影爆射长空，向二十四柄灵剑狠狠的碾压而去。
<br/>　　瞬息之间，漫天爪影便和二十四柄本命灵剑相击在了一起。
<br/>　　轰！轰！轰！！
<br/>　　一声声爆响此起彼伏，每一声爆响声响起，便是一柄灵剑崩飞了出去。
<br/>　　嗖嗖嗖！！
<br/>　　爪影碾压而过，空间四分五裂，二十四柄本命灵剑，全部被击飞了出去，飞向四面八方。
<br/>　　“苏莫，你的死期到了！”
<br/>　　击飞了二十四柄本命灵剑之后，巫森身形一闪，划过虚空，瞬息见便临近了苏莫。
<br/>　　随即，巫森一爪狠狠的抓出，一只千丈爪影暴击长空，直击苏莫的脑袋。
<br/>　　“这么强？”
<br/>　　苏莫脸色凝重无比，二十四星宿剑阵，居然完全不是对手了！
<br/>　　眼见对方爪影急速袭来，他身形立刻暴退。
<br/>　　与此同时，苏莫立刻动用了大三宝术，抽取体内四成的精气神。
<br/>　　不动用大三宝术，今日根本不可能击败对方了，所以，不论什么代价，他也在所不惜了。
<br/>　　嗖！
<br/>　　苏莫身形急速暴退，他身上的气息却是疯狂攀升，无限暴涨。
<br/>　　只是半个呼吸的时间，苏莫身上的气息便暴涨了几个层次，狂暴的气势冲破云霄。
<br/>　　大三宝术，是一种非常强大的秘术，天下少有。
<br/>　　不过，苏莫发现，随着他修为的不断提升，大三宝术的增幅，已经大大减弱了。
<br/>　　比如，真玄境修为之时，他抽取四成的精气神，可以瞬间增加四倍的实力。
<br/>　　随后的时间里，他每一次动用大三宝术，增幅都稍稍有些减弱。
<br/>　　如今他提升到了武皇境四重境界，再抽取四成的精气神，则只能增加三倍的实力了。
<br/>　　这一点让苏莫不太明白，但他懒得去研究。
<br/>　　“给我碎！”
<br/>　　实力大增，苏莫立刻止住了身形，不再后退，一剑斩向迎面而来的爪影。
<br/>　　咻！
<br/>　　璀璨的剑光冲天而起，划过一道优美的弧度，裹挟无匹的力量，瞬间斩中了爪影。
<br/>　　嗤！
<br/>　　剑光锋锐无敌，所向披靡，力大无穷，直接便撕裂了爪影，一斩而过。
<br/>　　“什么？”
<br/>　　巫森见此顿时脸色一变，他如此强大的一击，居然被苏莫挥手破灭了！
<br/>　　“秘术！”巫森立刻看了出来，苏莫是动用了某种强大的秘术，所以才实力暴增。
<br/>　　“杀！”
<br/>　　怒喝声起，只见苏莫身形如电，直接向着巫森冲了过去。
<br/>　　帝皇剑剑光万丈，重重的一斩而出，惊天剑气如同一道三色的彩虹，横贯天地，向巫森怒斩而来。
<br/>　　滔天的剑威直冲云霄，犀利的锋锐之气，即便相隔数万丈，都让得下方的大地瞬间出现了一道巨大的剑痕。
<br/>　　而巫森面对剑气正面的攻击，承受的压力可想而知。
<br/>　　“灭！”
<br/>　　低沉的声音从巫森的喉咙中传出，他如同狼爪一般的利爪，急速探出。
<br/>　　锋利的利爪之上，弥漫着黑色的光芒，让得这一爪不仅更加锋利，力量更是恐怖至极。
<br/>　　轰！
<br/>　　惊天剑气和利爪硬撼在了一起，剧烈的碰撞，让得方圆数千里之内的空间，立刻湮灭成了虚无。
<br/>　　汹涌的冲击波扫荡八方，下方本已残破不堪的帝玄山脉，瞬间灰飞烟灭，变成了方圆数千里的大坑。
<br/>　　噗！
<br/>　　一口鲜血从巫森的口中冲天而起，随之立刻被气劲湮灭，转而化为了虚无。
<br/>　　咻！
<br/>　　而后，刺耳的破空声响起，巫森的身形如同一柄飞驰的利箭，急速倒飞了出去。
<br/>　　巫森整整倒飞数千里，才止住了身形，喉头一甜，再次张口喷出一口鲜血。
<br/>　　“可恶！”巫森眸中杀机冲天，他动用了人魂合一，完全发挥了战魂的威力，居然还不敌苏莫！
<br/>　　连连被苏莫击伤，他身上的杀机几乎凝为实质。
<br/>　　“武帝强者，不过如此！”
<br/>　　不屑的声音响起，只见苏莫脚踏本命灵剑，已经急速追了上来。
<br/>　　“秘术之威，也敢言勇！”巫森怒喝一声，身上的气息同样开始了暴涨，他的身躯再次膨胀了一圈，黑色的毛发更加的浓密，尖锐的利爪快速的生长，变得更长、更锋利了。
<br/>　　两人都动用了秘术，身上的气息如妖似魔，如同滚滚狼烟，震人心魄。
<br/>　　所有观战之人，心绪都是不断的震动，如同有大浪在不断的冲击。
<br/>　　这一场巅峰之战，战况真是瞬息万变，时刻都会改变啊！
<br/>　　现在，早已经没有人认为苏莫必败了！
<br/>　　因为以两人的战斗情况来看，任何一人都有胜利的可能，任何落入下风的人，随时都可能翻盘。
<br/>　　所以，最终谁能胜利，就看谁的底牌更强打了。
<br/>　　此刻，黄泉魔宗阵营，除了面色铁青的殷宗之外，还有一人脸色极为难看，那就是皇甫擎。
<br/>　　皇甫擎的脸色，几乎黑为锅底了，苏莫动用的秘术，他如何看不出来，正是他传授的大三宝术。
