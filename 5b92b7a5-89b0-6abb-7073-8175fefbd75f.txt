　　一路上自然不少丧尸，被车子碾碎，那些鲜血飚射到挡风玻璃上，林峰和陈勾没什么，习惯了已经，苏云却有些害怕。
<br/>　　不过也没有叫更没哭，忍得住。
<br/>　　“这血啊，多看几次，也就和番茄酱差不多的东西了。”
<br/>　　林峰道。
<br/>　　很快。
<br/>　　车子到了网吧门口。
<br/>　　林峰和陈勾下来，苏云跟在陈勾身边。
<br/>　　“胖子你保护老师，我打头阵。”
<br/>　　“得嘞，老师，若是你感到害怕的话，可以抱紧我，我不会在意的。”陈勾道。
<br/>　　“胖子，少占老师便宜！！”
<br/>　　林峰回头一个死亡凝视，吓得陈勾干笑一下。
<br/>　　这家网吧规模不大，在二楼，下面是个洗头房。
<br/>　　“你表哥还真是厉害，这网吧建在洗头房上面，上网累了，再下来做个大保健？”林峰道。
<br/>　　“嘿嘿，没错，这洗头房也是我表哥开的，他是我们家族里目前混得最好的，千万身家。”陈勾道。
<br/>　　“当然了，现在千万身价屁用没有，估计还没我混的牛逼。”
<br/>　　“别废话了，进去了。注意安全。”
<br/>　　林峰拔出武士刀就是来到了网吧内。
<br/>　　那网吧大门上贴着几个字。
<br/>　　未成年人不得入内。
<br/>　　除此之外，还有一大摊鲜血，看上去很恐怖。简直就是拍恐怖片的氛围。
<br/>　　透过门缝林峰可看到三四个丧尸。
<br/>　　推开门林峰冲了进去一刀把他们就给了解了。
<br/>　　一个网管小妹张牙舞爪的就是冲了过来。
<br/>　　“苏老师，你来试试。”
<br/>　　林峰解决掉其余丧尸，看就剩下这么一个就干脆交给苏云练手。
<br/>　　“试试就试试。”
<br/>　　苏云深吸口气，直接一个飞踢刷的一下，这网管小妹脑袋都被踢飞了。
<br/>　　“这么容易？”
<br/>　　苏云愣了。她自己都有些意外。
<br/>　　“厉害啊，腿可以抬这么高，这可以解锁多少姿势啊。练过跆拳道的就是不一样。”
<br/>　　林峰眼睛冒光。
<br/>　　吼吼。
<br/>　　又是个丧尸冲了过来。
<br/>　　“这个丧尸必须要爆头，不然他是不会死的，生命力非常顽强。”林峰道。
<br/>　　“知道了。”
<br/>　　苏云深吸口气旋即一个帅气飘逸潇洒的后踢踢爆了丧尸脑袋。动作潇洒飘逸。
<br/>　　“厉害！”
<br/>　　林峰伸出大拇指道。
<br/>　　“那是自然。当我跆拳道是假的吗？”苏云得意道。
<br/>　　“表弟，是你吗？”
<br/>　　忽然一个男人从一个房间内探出头来。
<br/>　　“表哥！你特么还活着！”
<br/>　　陈勾眼睛一亮。
<br/>　　“我去，真的是你。”
<br/>　　那人顿时冲了出来就是给陈勾一个虎抱。
<br/>　　“咦，表哥，你身上好臭啊。”陈勾一脸嫌弃。
<br/>　　“那个，这不，好几天了，裤子上可能沾了点屎。”陈勾表哥尴尬笑道。
<br/>　　“表哥，你不会是被吓出屎来了吧？”
<br/>　　陈勾一脸的嫌弃。
<br/>　　“林峰，这就是我表哥，吴三。”
<br/>　　“表哥，这是我同学，林峰，这是我英语老师苏云。”
<br/>　　“你们好你们好。幸会幸会。”
<br/>　　吴三瞥了眼苏云顿时眼中露出了一抹惊艳，显然被苏云的颜值给震到了，不过却没有任何的贪婪和淫邪。
<br/>　　“那个，既然人找到了就先回去吧。”林峰开口道。
<br/>　　陈勾拉着吴三就是一路上说了许多事情。
<br/>　　听的吴三一脸的愕然，时不时还扫一眼林峰。
<br/>　　“苏老师，我等下要去个地方，你就和陈勾回去吧。”
<br/>　　“嗯。”
<br/>　　苏云也没多问就和陈勾上了车。
<br/>　　林峰目送他们离去就独自一人朝C级空投飞奔过去。
<br/>　　“这林峰这么厉害？”
<br/>　　吴三听陈勾说起顿时愕然。
<br/>　　“所以表哥你可不要把末世前自己是千万富翁就很牛逼的那一套拿出来。”陈勾道。
<br/>　　“放心吧，你表哥是什么人，懂得分寸，这什么黑铁战士，有点意思，对了表弟，既然如此，干脆我们就去杀丧尸吧，你表哥我也想变厉害，不想当拖油瓶。”吴三道。
<br/>　　“我也想历练一下。反正时间还早。”苏云道。
<br/>　　“既然这样的话，那我们就清理丧尸吧，顺便找物资。”陈勾点头道。
<br/>　　话说林峰朝C级空投奔去。
<br/>　　此时此刻。
<br/>　　一个高墙围起来的院子里。
<br/>　　坐着十几个男人。
<br/>　　这十几个男人吃着肉串，这些肉串都是他们从超市里抢来的。
<br/>　　“妈的，吃完肉串大家抓紧点，组织上说好的交货目标，我们还差了不少。”
<br/>　　“这组织只知道下命令，不知道这年头这吃的喝的都很难找了吗？”
<br/>　　“没错啊，这附近超市就那么几家，全都给人搬空了。”
<br/>　　“看来只能去抢人家的了。”
<br/>　　“抢就抢呗，又不是第一次，顺便还可以抢几个女人来玩玩，你看队长不就现在在玩女人嘛，还是个学生妹，还挺漂亮。”
<br/>　　几个男人哄笑道。
<br/>　　此时此刻。
<br/>　　若有若无有一个女人的惨叫声。
<br/>　　这个惨叫声一开始很响。
<br/>　　但是到后面越来越弱。
<br/>　　因为在做的男人每个人都轮流干了一炮，每个人都最起码半个小时，把那个女学生给活生生的干的快死了。
<br/>　　“没意思，特么的，居然咽气了，草。”
<br/>　　一个两百五十多斤的大胖子穿着裤子走了出来。
<br/>　　“队长，你这种体型压都可以压死人家了。”
<br/>　　几个男人都笑道。
<br/>　　“不就死了个女人嘛，多大点事情，明天再抢几个。”
<br/>　　在他们胳膊上都画着一个狼头。
<br/>　　此时此刻坐在一起喝酒吃肉。
<br/>　　只不过外面若有若无的丧尸咆哮显得有些诡异和渗人。
<br/>　　林峰来到这里时。
<br/>　　地图显示就在这前面一个院子里。
<br/>　　“这尼玛可不好搞，居然掉到人家院子里去了。”
<br/>　　林峰也不知道院子里有没有人。
<br/>　　林峰发现有个铁门就去敲了敲。
<br/>　　“妈的，这个时候了谁来敲门。”
<br/>　　“难道是丧尸？”
<br/>　　“动动你的猪脑子，你们家丧尸会敲门吗？”
<br/>　　顿时里面传来一个男人骂骂咧咧的声音。
<br/>　　铁门打开了。
<br/>　　林峰就看到了一个大块头男人虎视眈眈的看着自己。
<br/>　　“小屁孩，干什么，要吃的我这里没有。”
<br/>　　林峰扫了眼院子内，还吃着烤肉，这群人看来生活的不错啊，要知道许多人都混的朝不保夕了。
<br/>　　林峰扫了眼他们，看上去也不像是什么正派人物。
<br/>　　“几个老铁，我有个东西掉在你们院子里了，能不能进去捡一下。”
<br/>　　这可是C级空投，都到门口了，甚至林峰都可以看到了，他自然不会就这样放弃了。
