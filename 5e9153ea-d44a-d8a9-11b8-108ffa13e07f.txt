　　“剑气化龙？”
<br/>　　剑祖荒看到楚风眠出手，冷傲开口道。
<br/>　　“看似不错，实则不堪一击，剑道可不是你这种花拳绣腿，在我的面前，你连一剑都接不下，给我死吧！”
<br/>　　剑祖荒猛然出手，他的剑锋在瞬间变得凌冽数十倍，猛然之间向着楚风眠斩杀而来。
<br/>　　似乎是要，一剑斩龙。
<br/>　　“花拳绣腿？”
<br/>　　楚风眠嘴角突然大笑起来，这剑祖荒，倒是真的不知天高地厚，敢在楚风眠的面前称其为花拳绣腿之人。
<br/>　　这已经不是狂傲了，根本就是无知。
<br/>　　“天剑宗，既然已经没落至今，万年之前，天剑宗虽然剑道微末，却也有些自知之明，没想到这万年之后，居然就连自知之明都没有了。”
<br/>　　楚风眠一边冷笑，一边突然出手，这一道剑气化龙，顿时咆哮起来。
<br/>　　随着这一声龙吟咆哮，漫天剑光，纷纷涌现。
<br/>　　“砰！”
<br/>　　一剑一龙，顿时相撞。
<br/>　　这剑的锋芒，在远古战龙面前，简直不堪一击，顿时破碎。
<br/>　　反噬的力量，让剑祖荒都是后退数步，脸色惊骇，眼神中露出一抹难以置信之色。
<br/>　　在这剑道交锋上，他跟楚风眠之间，退后之人，居然是他。
<br/>　　而楚风眠，则是站在原地，看起来甚至都是没有丝毫受伤的样子。
<br/>　　在剑道上，居然有人超越了他？
<br/>　　剑祖荒的脑海中，顿时一阵晴天霹雳，他剑祖荒，在天羽神域中纵横，剑道无敌。
<br/>　　就算是天剑宗的一些长老，一些超越生死的老古董，他们的剑道甚至都还不如剑祖荒。
<br/>　　在剑道上，剑祖荒从未想过自己会败，更何况是败给一个比起自己境界还低的武者。
<br/>　　楚风眠仅仅生死境四重的境界，在剑道之上交锋，便是战胜与他。
<br/>　　那些其他的天剑宗弟子，看到了这一幕，都也是一脸呆滞，这个结果，绝对是任何一人都无法想象过的。
<br/>　　剑祖荒的剑道，在九域年轻一代中，堪称无敌，近乎数十年来都无人能够与他争锋。
<br/>　　现在居然败在楚风眠的手中，怎么可能。
<br/>　　“剑道再强又如何？区区生死境四重的境界，怎么可能跟我抗衡，你若是跟我相同境界，这一战，我必败，但是你不过区区生死境四重罢了，在我们无数天剑宗弟子的面前，今日你依然得死！”
<br/>　　剑祖荒看向楚风眠，猛然怒喝一声。
<br/>　　“杀！”
<br/>　　所有天剑宗弟子，听到剑祖荒这一声号令，都是纷纷出剑，漫天剑光，直接凝聚出了一道长河。
<br/>　　组成这长河的，不是水，而是无数剑气，剑气如河般，向着楚风眠冲刷过去。
<br/>　　而就在这剑道长河之中，最为耀眼的便是剑祖荒的剑光，他的剑光被众星捧月一般，在这剑道长河之中，是要将楚风眠，一剑斩杀。
<br/>　　单打独斗不成，剑祖荒便是让所有天剑宗弟子联手之下，一同出手。
<br/>　　数十位生死境巅峰的剑修同时出手，凝聚的这剑气长河，简直是人挡杀人佛挡杀佛，谁也无法抵挡其锋芒，如今向着楚风眠，直接轰杀过来。
<br/>　　“哦？比拼力量？”
<br/>　　看到剑气长河逼近，楚风眠的脸色还是一如既往的平静，嘴角露出了几分冷笑道。
<br/>　　“你以为比拼力量，就凭你们这群蝼蚁，也足以跟我抗衡？”
<br/>　　要是楚风眠还没有度过生死大劫，面对这剑气长河，他自然不敢出其锋芒，只能后退。
<br/>　　但是现在，楚风眠度过了生死大劫，实力暴涨，再多的生死境武者，在他的面前，都是蝼蚁。
<br/>　　就算是联合起来，也一样。
<br/>　　突然之间，楚风眠再度拔剑，在他的背后，战龙虚影凝聚其后，漫天血气喷涌而出，将剑锋都给染红。
<br/>　　远古杀神，加持其身，除了这两股力量之外，更加显眼的，便是一道冲天剑意。
<br/>　　楚风眠的九域剑术，在度过生死大劫后，便是已经近乎大成，今日便是他真正展现剑术的时刻。
<br/>　　从楚风眠身上涌现出来的剑意，比起那无数天剑宗弟子一同联合凝聚而起的，都要恐怖的多。
<br/>　　“嘶！”
<br/>　　无数剑光，从楚风眠的身上展现出来，全部凝聚在剑锋上，一股剑意，笼罩在那剑气长河之上。
<br/>　　“死！”
<br/>　　楚风眠手中灵剑一动，直接斩在这剑气长河之上，突然之间，如同天地崩塌一般，这剑气长河，被猛然撕裂开来。
<br/>　　在这一剑之下，被一剑斩断，一分为二，无数剑气，纷纷瓦解。
<br/>　　这无数天剑宗弟子，在加上剑祖荒全力之下，凝聚而出的剑气长河，如今在楚风眠的一剑下，彻底毁灭。
<br/>　　“怎么可能？这是什么手段？”
<br/>　　剑气长河被毁，剑祖荒等人，都连连退后，一个个口吐鲜血起来。
<br/>　　他们联手之下的剑气长河，居然都无法奈何的了楚风眠，被楚风眠这样一剑破开。
<br/>　　“妖孽！妖孽！怪不得小魔主要联系其他宗门要斩杀此人，此人在天羽神域中，已经几乎无敌，不杀他，我们都得死！”
<br/>　　剑祖荒的脸上都是露出几分悔意，早知道他便是提前告诉其他七大宗门的弟子，来一同围攻楚风眠。
<br/>　　可是现在后悔，已经迟了。
<br/>　　今日，楚风眠绝不可能放走他们。
<br/>　　“剑破天地！”
<br/>　　楚风眠一剑斩破这剑气长河，没有任何停息，再度一剑斩出，直冲天际，无穷剑锋，从天而降斩杀而下。
<br/>　　那一尊尊天剑宗弟子，在这漫天剑锋之上，无数人都是身体停滞，身中剑气，在下一刻便是直接砰的一声爆炸开来，化为了一团血雾，一身精血，随着剑气，被楚风眠直接收取。
<br/>　　“天剑宗弟子，不过如此。”
<br/>　　楚风眠身形一动，回到宝车上，坐在宝车之上，俯视着众人，天剑宗弟子，已经是死伤大半，仅仅只剩下五人。
<br/>　　除了剑祖荒之外，其他四人，也都是如今天剑宗弟子中的强者。
<br/>　　只有他们，从这漫天剑锋中，勉强抵挡了下来，但是一个个也都是身受重伤，灵力亏空。
<br/>　　不过楚风眠，却是突然停下手来，他的目光，突然看向远处的冰谷之中。
<br/>　　“不用躲躲藏藏的，都滚出来吧。”
<br/>　　突然之间，楚风眠冷喝一声，他的声音，如同一声龙吟一般，响彻整个冰裂谷。
