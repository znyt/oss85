　　“什么？”
<br/>　　所有人闻听玄净尊者之言，均是心中剧震，瞬间瞪大了眼眸，面现不可置信之色。
<br/>　　四生战魂？
<br/>　　开什么玩笑？
<br/>　　这也太扯了吧？
<br/>　　对方若说苏莫是三生战魂，众人勉强还能相信，毕竟，三生战魂虽然万年难得一见，但终归是有这种天才。
<br/>　　比如，三神宗的姜神夜，就是三生战魂的天才，整个古灵星河只此一人。
<br/>　　但是四生战魂，这未免太不切实际了，在古灵星河的无尽历史上，都没有出现过这种人物吧！
<br/>　　经过短暂的震惊之后，整个拍卖会场，再次的沸腾了起来。
<br/>　　“玄净前辈，你老开玩笑的吧？”
<br/>　　“这绝对不可能！”
<br/>　　“刚才的战斗，苏莫好像一直都没有释放战魂，就算动用了不同的战魂之力，但好像只有两种吧！”
<br/>　　“四生战魂？这未免太夸张了，谁会相信啊！”
<br/>　　“……”
<br/>　　众人七嘴八舌，声浪嘈杂不已，但所有人言语的意思，基本上都是一样的，根本没有人相信。
<br/>　　刚才的影像之中，若是苏莫释放出了自己的战魂，直观的展现出来，那样的话众人不信也要信。
<br/>　　但现在，空口说白话，谁能信服？
<br/>　　尤天韩沉默不语，眸中闪过沉吟之色，不知道在思忖着什么？
<br/>　　巫人王面无表情，眸中精光不断的闪烁，同样不知道在思索着什么？
<br/>　　“众位安静！”
<br/>　　玄净尊者再次朗喝一声，声音如同雷霆炸响，直接在整个拍卖会场炸开，将所有人都震的有些发懵。
<br/>　　随即，他便继续解释道：“刚才的战斗，众位应该发现了，苏莫动用了两种战魂之力，其中一种那磅礴浩荡的剑威，自然便是剑之战魂，另外一种便是秒杀数十名武者时，突然出现的吞噬力量！”
<br/>　　“这种吞噬力量便是一种特殊的战魂，目前没有人知道这是什么战魂，有可能是吞灵战魂、有可能是七彩吞天蟒战魂、有可能是天噬战魂，也有可能是吞天圣魂！”
<br/>　　“总之，无论这是什么战魂，我们不予深究，反正这是一种强大战魂，是板上钉钉的事情！”
<br/>　　“我要告诉众位的关键，便是苏莫的剑之战魂，根据我方商会两位武圣的观察，这绝不只是一种剑之战魂，因为就算是完美的剑之战魂，在武帝境四重之时，也远远不可能发挥出如此逆天威能，甚至不可能发出这种威能的两成。”
<br/>　　“根据苏莫身上所爆发的剑道威能，和剑之战魂的力量，我们可以肯定，苏莫有三种剑之战魂，而且全部都是完美战魂！”
<br/>　　玄净尊者口中不停，加快了语速，想要尽快将此事解释清楚。
<br/>　　毕竟，将苏莫列为真圣榜第一，这简直就是一次冒险，搞不好就会成为笑柄。
<br/>　　当然，他们既然如此做了，自然是有足够的底气。
<br/>　　其实，本来他们商会的众强者，猜测苏莫可能有更多的剑之战魂，因为就算是三种剑之战魂，都不可能发出如此强的剑之威能。
<br/>　　但是这个猜测，连他们自己都不愿意去不相信，最后才统一了观点，认为苏莫有三种剑之战魂。
<br/>　　当然了，三种剑之战魂，跟本不可能有这么强的威能，这也有些解释不通，他们只能认为是苏莫的三种剑之战魂全部是完美战魂，以及苏莫精神力强大。
<br/>　　众人闻听玄净尊者之言，脑中回想之前影像中的画面，这影像不仅仅是画面，连双方气势、攻击威势等等，全部都展现了出来，如同身临其境。
<br/>　　的确！
<br/>　　苏莫的剑，简直凌厉到了令人发指的地步，那磅礴浩瀚的剑威，形成一道巨大的虚幻剑影，剑威无可匹敌。
<br/>　　这是剑之战魂才能发出的气势，一种剑之战魂，甚至是两种剑之战魂，根本不可能有这么强悍的威势。
<br/>　　貌似，只有三种剑之战魂，才有可能发挥出这么强的威势吧！
<br/>　　众人心惊不已，难道苏莫真的有三种剑之战魂？真的是四生战魂的盖世妖孽？
<br/>　　这未免也太夸张了吧！
<br/>　　先前，众人只是觉得苏莫的剑凌厉到了极致，剑威强大的不像话，倒是没往剑之战魂上多考虑，现在则是越想越心惊。
<br/>　　不过，这终究只是猜测，一切都是不确定的，毕竟没有人真正见过苏莫有几种战魂。
<br/>　　虽然玄净尊者说的句句在理，但也有例外啊，若是苏莫有某种玄妙宝物，能像战魂一样，爆发出强大的剑魂力量，也不是不可能的事情。
<br/>　　众人以及有些不敢置信，实在是四生战魂这种事情，太过荒诞了！
<br/>　　坐席上，苏莫一直在聆听，心中是又好气又好笑，这分析的还真是有些道理。
<br/>　　只不过，他的剑之战魂不是三种，而是多到数不胜数。
<br/>　　当然，他所吞噬的剑之战魂，并没有太过强大的，都只是较为普通的剑之战魂，但胜在数量多。
<br/>　　中央拍卖台上，仿佛是知道众人的想法一般，玄净尊者话语不停，继续说道：“战魂的事我们暂且不谈，我们再来说说特殊体质！”
<br/>　　“想必众位都看到了，苏莫身上有特殊体质，非常的强悍，大家能否看出这是什么体质？”
<br/>　　玄净尊者没有直接说出苏莫的体质，而是反问起了众人。
<br/>　　苏莫彻底无语了，他这是变成怪物了，被人不断的研究分析，这种感觉让他觉得有些荒唐。
<br/>　　试问，就在他的面前，有无数的人正在研究分析他，分析他的战魂、分析他的体质，这是多么的古怪！
<br/>　　偏偏分析的不对，还自认为很正确。
<br/>　　这时，听闻玄净尊者的话，不少人纷纷开口了。
<br/>　　“这种体质很少见啊！白光中还留露出琉璃色的光芒！”
<br/>　　“画面中苏莫周身的空间都扭曲焚化了，显然是有恐怖的高温，很像烈日圣体！”
<br/>　　“烈日圣体不可能，应该是大日神体！”
<br/>　　“但是也不对啊！大日神体若是一施展，整个人都化身成了一个小太阳，这明显弱很多啊！”
<br/>　　“最主要的是，大日神体之中，怎么会有琉璃色的光芒？”
<br/>　　众人七嘴八舌，但并不是回答玄净尊者的问话，而是在各自分析，分析苏莫是什么体质。
<br/>　　不过，玄净尊者没有给众人多久的时间分析，仅仅是几个呼吸之后，便朗声道：“老夫告诉你们，这是因为苏莫有两种特殊体质！”
<br/>　　玄净尊者的话音不大，但却是比朗声大喝，效果还要好的多，顿时将所有人震的楞住了。
