　　大伙怔怔地看着这一幕，不明所以。
<br/>　　星月奴双手凌空，面对钟馗的方向作法不停，钟馗按不住青州鼎，让它飞了回去，青州鼎一路飞回到了星月奴身下，星月奴坐了进去，上面只能看到旋转的水柱和翻腾的白汽，看不到她的人了。
<br/>　　“这是什么意思！”叶少阳纳闷。
<br/>　　钟馗走过来，说道：“这是天地同修，是轩辕山最高层次的一种修炼法门，将神识外化，以器为身，最大程度地吸收天地之气……”
<br/>　　“天师你讲得好深奥啊，能不能通俗点，我没什么文化。”小马凑上来说道。
<br/>　　钟馗瞪了他一眼，说道：“这口青州鼎，现在就是她的识海，正在吸收的海水，八成就是九星叠气阵所吸纳的愿力而产生的幻象，她而今在修炼，等这些海水都吸干，便是表示她完成了修炼……”钟馗抬头朝四周看了看，说道：“我猜，这里头之前应该遍布海水，已经被她炼化到只剩这么一些了。”
<br/>　　“不会吧，这里这么多海水，这得炼化到什么时候？”小马感觉不相信。
<br/>　　美华之前试图冲破小岛的结界束缚，调集海水攻击（毕竟她是水鬼，更擅长用水），一直都在小岛边上徘徊，寻找结界薄弱的地方。
<br/>　　听见小马的话，她回头说道：“是这样，水位一直在下降，速度很可怕，我估计要不了多久就会被她吸干了！”
<br/>　　叶少阳彻底明白了：星月奴还是没有托大，之前双方战斗，对她来说只是一种试探，然后她发现凭自己现在的实力（哪怕她无限接近证道、又有青州鼎、会拆魂术），仍然架不住他们这么多强者的围攻，当她确定这一点之后，毫不犹豫地用自己二魂七魄和四宝一起组成结界，自己躲在后面继续修炼最后一波……
<br/>　　没什么好说的，叶少阳跟大家一起尝试打开结界，一番折腾之后，总算明白了星月奴为什么敢放心去修炼了：
<br/>　　大凡结界形态的阵法，不惯是什么元素构成，核心都是一样的，都有阵眼所在，阵眼被破，阵法也就不存在了。
<br/>　　而眼下这个结界的阵眼……就是四宝！
<br/>　　别的位置，不管怎么样进攻，打开缺口之后，水流总是能够从四方涌来，将缺口填补。一般结界阵法，总是故布疑阵，让人找不到阵眼所在，但眼下这阵法，阵眼却是直接暴露在了外面。
<br/>　　原则上来说，只要破了阵眼，结界会立刻粉碎。但是……四宝现在被洗脑了，在地上打坐，将金身罗汉的灵力附着在了自己周身，如同定海神针一般，稳定地维持这道水元素的结界。
<br/>　　想要破阵，就得重击四宝，而一旦阵破，他作为阵眼，阵法的力量会一瞬间压迫在他身上，他必须无疑……
<br/>　　星月奴这一手，实在狠到极点。
<br/>　　“四宝，四宝你特么的醒醒！”大伙一个劲地叫喊，试图唤醒四宝。
<br/>　　假如四宝真的昏迷那还罢了，关键他现在十分清醒，还不断劝说众人：“你们不要执迷不悟，等前辈修成万劫不磨之身，将你们渡化到轩辕山，到那时，你们与我同去婆娑净土……我而今看上去做了坏人，但我这么做，都是为了你们好……”
<br/>　　“你特么的怎么跟搞传销似的！”小马大骂，四宝只当做没听见，仍旧苦口婆心地劝他们。
<br/>　　试了很多办法，对他都没用。
<br/>　　最后大伙只能放弃，朝叶少阳看去。
<br/>　　“小天师，你要抓紧时间决断了……”钟馗倒不表态，只是在边上提醒。
<br/>　　叶少阳当然知道他这话是什么意思……
<br/>　　九面猫走过来，对叶少阳说道：“你们毕竟兄弟一场，你下不了手，让我来！”
<br/>　　说完朝四宝走过去。
<br/>　　叶少阳猛然回过神来，冲上去挡在四宝面前，恶狠狠地望着她，“谁都不许动他！”
<br/>　　小马、吴嘉伟也第一时间站到他身边，一致对外。
<br/>　　九面猫怔了一下，无奈笑道：“好啊，那我们就一起死！叶少阳，你是讲义气的，为兄弟而死，自然没话说，但为了他一人，牺牲你这么多兄弟，值得吗？”
<br/>　　“我们都不怕！”瓜瓜大声喊道。
<br/>　　叶少阳的心却在滴血。
<br/>　　一旦星月奴成功证道……被关在这九星阁中的他们，差不多真的就是死路一条……
<br/>　　九面猫继续说道：“好，你们都讲义气，愿意跟兄弟一起死，但是……外面那么多的人，希望可都在你们身上呢，你也让他们跟着一起陪葬，哦对，还有张无生，他为了救你们，自我牺牲了，难道你让他也白死不成？”
<br/>　　叶少阳心头一沉。
<br/>　　九面猫最后这句话，击中了叶少阳心中最脆弱的地方，他转过身，望着四宝，一步步走到他跟前去。
<br/>　　“小叶子不要啊！”小马过来挡在前面，吴嘉伟也跟上来。
<br/>　　叶少阳双手搭在两人肩膀上，喃喃道：“那你们告诉我，还有什么办法？”
<br/>　　“这……”
<br/>　　两人互相看看，也是说不出话来。
<br/>　　叶少阳转头看着林三生，道：“你也技穷了吧。”
<br/>　　林三生沉思不语。
<br/>　　“所有人的希望都在我们身上，我们……只能这么做，不要你们动手，所有我一个人扛着。”叶少阳转头看着四宝，“再问你一遍，你真的不走开是不是！”
<br/>　　四宝还是一脸平静地看着他，缓缓说道：“我不入地狱谁入地狱，杀了我，能让你找回本心，你动手便是。”
<br/>　　“滚你妈个蛋，杀了你，老子再自杀，陪你一起做鬼！”
<br/>　　“兄弟，上路！”
<br/>　　叶少阳猛地一把推开小马和吴嘉伟，拔剑而起，朝四宝面门刺过去。
<br/>　　“少阳！”
<br/>　　“老大！”
<br/>　　叶少阳暴起一剑，大伙想要阻止，已经来不及了。
<br/>　　七星龙泉剑剑锋抖动着，刺中四宝眉心处。
<br/>　　四宝有金身护体，剑锋刺在上面，仿佛刺在了钢铁上。
<br/>　　“上路！”叶少阳大吼一声，带着满腔的绝望和愤怒，用力刺下去……
<br/>　　“呛！”
<br/>　　一道金光飞来，却是一朵金莲，落在四宝头顶，一时间金光普照，生生地剑锋弹开，叶少阳退了一步，大惊失色，转头看去，一朵祥云，从空中飞来，悬空停住，云朵上站着一个人，一个和尚，头顶有神光，一看就修为不凡。
