　　厚土塔，第十层。
<br/>　　苏莫就在第十层的入口处，缓缓的盘坐了下来。
<br/>　　庞大的重力，让得他体内的玄力几乎无法运转，费了老大的力气，玄力才开始缓慢的在体内流转。
<br/>　　但这个玄力流转的速度，却是比平常下降了无数倍。
<br/>　　不过，虽然慢了很多倍，但对于玄力的淬炼效果，却是大大的增加了。
<br/>　　玄力每运行一个大周天，便更加的凝炼、更加的凝实。
<br/>　　就这样，苏莫抛却了心中所有的杂念，全身心的投入了修炼之中。
<br/>　　所谓修炼无岁月，时间的修炼之中悄然流逝，一日又一日！
<br/>　　转瞬之间，一个月的时间，便悄然而过。
<br/>　　厚土塔第十层内。
<br/>　　苏莫体内玄力不断的流转，经过了一个月的时间，他已经适应了第十层的重力。
<br/>　　他体内玄力的流转速度，比刚踏入第十层时，已经快了十几倍。
<br/>　　“还是不太够！”
<br/>　　苏莫低声自喃，他感觉体内的玄力虽然颇为凝实了，但距离一举冲破境界瓶颈，还是有点差距。
<br/>　　随着他修为的不断提高，玄力的淬炼，需要的时间也越来越长。
<br/>　　这还是在厚土塔的重力领域之中，若是在外界修炼，估计起码需要小半年的时间。
<br/>　　随即，苏莫继续修炼，又修炼了半个月之后，他才停止了下来。
<br/>　　而后，意念一动，他整个人便消失无踪，进入了虚界神图之中。
<br/>　　神图空间之中。
<br/>　　苏莫在一片光华平坦的岩石地面上盘坐，目露沉吟之色。
<br/>　　随即，他手掌一翻，那颗精气所剩不多的伪龙珠，便出现在了他的手中。
<br/>　　苏莫准备先吞噬这颗伪龙珠，将肉身实力再提升一番。
<br/>　　毕竟，肉身实力越强，他吞噬灵气就能更加的狂猛，不会担心因吞噬的灵气太多，而出现身体承受不住的情况。
<br/>　　“吞！”
<br/>　　苏莫没有释放出吞噬武魂，就这样在体内催动了起来。
<br/>　　霎时间，伪龙珠中的精气，顺着他的手掌，疯狂的向他体内涌去。
<br/>　　苏莫身上金光闪耀，他的肉身实力再次开始暴涨。
<br/>　　他的体内发出噼里啪啦的爆响声，全身的血肉也不断的震颤了起来。
<br/>　　苏莫的肉身实力，早已达到了堪比武皇境五重武者实力的地步，开始向更高一层次冲击。
<br/>　　伪龙珠的精气已经所剩无几，但苏莫全部吞噬完之后，肉身实力依旧提升了一个层次，达到了堪比武皇境六重武者的地步。
<br/>　　随后，苏莫准备冲击武皇境四重境界。
<br/>　　手掌不断的挥动，各种灵石、灵草、灵药不断的出现，堆积在苏莫的四周。
<br/>　　片刻之间，苏莫几乎被无尽的修炼资源掩埋。
<br/>　　他的四周，堆积的上品灵石和极品灵石，加起来不下百亿之多，各种级别的灵草灵药也有数万株。
<br/>　　这些修炼资源，全部是诛灭殷家所获，他的储物戒内还有不少。
<br/>　　“一举成功！”
<br/>　　苏莫深吸了口气，他向东海龙王等人保证，两个月之内突破，而现在已经过去了一个半月。
<br/>　　这一次，他要一举成功。
<br/>　　“吞！”厉喝一声，苏莫立刻释放出了吞噬武魂。
<br/>　　庞大的吞噬武魂，如同一个巨大的黑色漩涡，悬浮在虚空之中，强大的吞噬之力，横扫八方。
<br/>　　轰轰轰！！
<br/>　　嗡嗡嗡！！
<br/>　　霎时之间，无数的灵识纷纷爆开，无数的灵草灵药急速枯萎，其内的药力飞速流逝。
<br/>　　药力混杂着灵气形成滚滚浪涛，急速向苏莫身上涌去，瞬间将苏莫的身形淹没了。
<br/>　　苏莫盘坐之地，方圆数千丈之内，已经形成了浓郁的灵气之海。
<br/>　　那浓郁的灵气，已经粘稠的接近液态，非常的惊人。
<br/>　　庞大的灵气之海上方，突兀的塌陷了下去，出现了一个漩涡。
<br/>　　仿佛地漏一般，灵气通过漩涡不断的向下方流去。
<br/>　　轰隆隆！！
<br/>　　苏莫的体内，早已波动震天，巨响连连。
<br/>　　澎湃的灵气如同长江大河，在他的体内横冲直撞。
<br/>　　大混沌五行神诀早已运转到了极致，大量的灵气被炼化成五种属性的玄力，分别汇入五枚属性神丹之中。
<br/>　　这种情况的修炼非常惊人，整个苍穹世界，估计从来没有人像苏莫这样修炼过。
<br/>　　当然，苏莫修炼所需要的灵气，也是别人的数十倍，甚至更多。
<br/>　　苏莫的修为，在天殷城之时，吞噬了大量的殷家族人，早已达到了武皇境三重的巅峰，现在灵气入体，他立刻便开始冲击武皇境四重境界。
<br/>　　轰隆隆！
<br/>　　海量的玄力汇入五枚神丹，五枚神丹立刻剧烈的震颤了起来，发出夺目的光芒。
<br/>　　神丹剧烈的震颤，引起滔天的波动，让得苏莫的身躯都不听的颤.抖。
<br/>　　这是因为他吞噬的太猛了，灵气和玄力都太过澎湃。
<br/>　　苏莫不管不顾，要紧牙关，继续冲击境界瓶颈。
<br/>　　时间流逝，庞大的灵气之海不断的缩小，两个时辰之后，便缩小了一半。
<br/>　　四个时辰之后，灵气之海只剩下了方圆五丈左右。
<br/>　　就在此时，一股浩瀚的气息，从灵气之海中陡然升起，而后如同秋风扫落叶一般，横扫八方。
<br/>　　轰隆隆！！
<br/>　　气息扫荡而过，空间不断的炸裂，所剩不多的灵气之海，也被扫荡的灰飞烟灭。
<br/>　　但是，紧紧是一个呼吸的时间，这股庞大浩瀚的气息便全部收敛，消失无踪。
<br/>　　呼！
<br/>　　盘膝而坐的苏莫显出了身形，他长长的出了一口气，浊气如箭，爆射十数丈。
<br/>　　“武皇境四重境界！”苏莫眸中精光璀璨，呢喃一声之后，便缓缓的站了起来。
<br/>　　他终于突破到了武皇境四重境界，如果到目前，黄泉魔宗的宗主，没有突破到武帝之境的话，那整个苍穹世界，基本上没有人是他的对手了。
<br/>　　现在，苏莫甚至有信心，与真正的武帝强者一战而不败。
<br/>　　当然，他只是有信心，他心中也不清楚真正的武帝，到底有多强。
<br/>　　“是时候诛灭黄泉魔宗了！”苏莫心中杀机升腾，立刻便离开了神图空间，而后走出了厚土塔。
<br/>　　刚刚走出厚土塔，苏莫蓦然发现，厚土塔的大门之前，居然盘膝坐着一人。
<br/>　　“钱副阁主！”
<br/>　　苏莫凝神一看，此人居然是上次遭受重创，险些身死的天涯海阁钱副阁主。
<br/>　　“苏莫，你终于出来了！”钱副阁主见到苏莫出现，急忙站了起来，面现焦急之色。
