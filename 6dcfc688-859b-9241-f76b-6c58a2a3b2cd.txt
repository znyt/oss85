　　望着空间皱褶，慕清霄最清楚，这是斗尊级别才有能力布置的绝强结界。
<br/>　　除非本人亲自开启，或者拥有特殊的开启手段，最后就是强行破开，否则根本闯不进去。
<br/>　　整座迦南学院，有能力布置空间结界的，绝对只有那位行踪不定的半圣级别院长了。
<br/>　　楼阁下方，只见两名灰袍老者自袖袍中探出两双干枯的手掌，旋即快速结印...
<br/>　　结印过程极其迅速，且颇为繁杂，哪怕记忆力在好的人，看一遍也根本记不住，当然，过目不忘除外。
<br/>　　随着两人的结印，慕清霄能够清晰的感觉到，两股无形雄浑的波动，自他们的手掌中扩散而出。
<br/>　　无形的波动宛如浪潮一般，随之扩散，最后与空间皱褶接触在了一起。
<br/>　　两者接触的瞬间，慕清霄等人面前的空间，便是犹如一块水做成的镜面一般，其上不断泛起一圈圈涟漪。
<br/>　　片刻后，涟漪停止，一扇透明的空间门，仿佛被一双无形大手，给缓缓的撕裂开来。
<br/>　　“都跟上来。”见空间门开启，琥乾挥了挥手，率先走了进去。
<br/>　　前脚刚踏进空间门，视线明显比刚才清明许多，仿佛眼前的透明色帘幕掀去，显然，他们已经置身于领域空间中。
<br/>　　“劳烦二老。”
<br/>　　站在阁楼外，琥乾朝着盘坐在地上的灰袍老者微微行礼。
<br/>　　“无妨，任务而已。”
<br/>　　全身都包裹在黑袍中，两位神秘老者连身体都未曾动一下，嘶哑的声音缓缓响起，听不出有任何一丝情绪波动。
<br/>　　“现在，你们自行进入藏书阁吧，记住，不管你们想要得到什么，都不可强求。”
<br/>　　说罢，琥乾目光在全员身上扫过，话语中的意思，不言而喻。
<br/>　　“里面所有的东西，都被加上了能量层，若是你们的手能穿过能量层，便能拿走其中的东西，当然，不管你能拿多少，最终能够带出藏书阁的，只能是一件，千万不要贪心，否则，到头来只会竹篮打水一场空。”
<br/>　　“是！”
<br/>　　“嗯。”
<br/>　　琥乾微微点头，指着藏书阁，接着说道：“若是你们不能穿过能量层，那就放弃吧，凭你们的实力，还破不开能量层，每年都有学员进入藏书阁，最后空手而归的也并不在少数，一切随缘，得不到的东西，切勿强求。”
<br/>　　话语刚落，慕清霄就拉着小医仙，缓缓向藏书阁走去，身后学员回过神来，也迈步跟了上去。
<br/>　　安静的山谷中，只有细微的脚步声响起，慕清霄与小医仙速度很正常，而先前的五名学员早已去往藏书阁下。
<br/>　　至于古薰儿，一直走在最后，与慕清霄保持着五米的距离，速度不快也不慢，眼睛盯着他，冷不丁的问道：“你是元古八族中，哪一族的？”
<br/>　　走在前面，慕清霄并没有回应她，他当然清楚，古薰儿估计是将他当作了远古八族的族人了吧？
<br/>　　见他不理会自己，古薰儿本就清冷的脸色，变得更加冷漠，眼中不停思索，猜测着慕清霄的身份。
<br/>　　片刻后，慕清霄拉着小医仙，穿过楼阁面前的草地，最后踏上那被岁月摧残的坑坑洼洼的青石阶梯。
<br/>　　青石阶梯上布满一层厚厚的青苔，一脚踩上去，有些湿滑，不过对他们并没有什么阻碍。
<br/>　　小医仙回头看了一眼古薰儿，随后牵了牵他的袖口，道：“...你们认识？”
<br/>　　听到她幽幽的话语，慕清霄脚步一顿，瞥了眼古薰儿，倾国倾城，但小医仙一点也不差，两女可以说是各有千秋吧。
<br/>　　古薰儿高贵清冷，小医仙柔美至极，慕清霄更喜欢后者，伸手捏了捏她小巧的鼻子，道：“我们也快进去吧。”
<br/>　　说着，慕清霄就再次迈开脚步，小医仙美眸瞪了他一眼，显然知道自己想多了，于是快步跟了上去。
<br/>　　她虽很想独占，奈何后者实在太优秀，若是三妻四妾，她也完全能够接受。
<br/>　　顺着青石阶梯走了一会后，便来到藏书阁面前，抬头望着古老的匾额，一股荒凉的气息扑面而来，缠绕在心间，久久不散。
<br/>　　失神片刻，慕清霄收回目光，心中不免有些惊叹，拉了拉小医仙，一股清凉席卷上她的心头，使得她从失神中反应过来，美眸中尽是震惊。
<br/>　　见慕清霄面色如常，门口处的两位灰袍老者，袍服一颤，嘶哑细微的声音再次响起，道：“你小子，古怪的很啊!”
<br/>　　“咳咳...”
<br/>　　说着，嘶哑的咳嗽声忽然响起，将失神中的学员惊醒，五人皆是一愣，视线连忙从匾额上挪开。
<br/>　　“都进去吧，藏书阁大门只会开启一个小时，一个小时后，不管有没有得到需要的东西，那也必须出来。”
<br/>　　当学员全部来到藏书阁门口，灰袍老者袖袍随风而动，旋即伴随着一阵嘎吱声响，紧闭的大门缓缓开启，露出一条漆黑的通道。
<br/>　　通道中一股微风迎面而来，携带者一股古老而苍凉的气息，刚才失神的学员们都面露凝重。
<br/>　　若是在陷进去，没有灰袍老者提醒，估计一个小时结束，他们出来多半要空手而归了。
<br/>　　“走吧。”
<br/>　　说着，慕清霄就拉着小医仙，消失在漆黑的通道中。
<br/>　　在全学员都进入通道后，宽敞的大门，再次响起一阵嘎吱声响，最后合拢，连一丝缝隙都没有留下。
<br/>　　望着紧闭的大门，琥乾轻轻松了口气，微笑道：“希望这些小家伙能够得到想要的东西吧，能够进入迦南学院的藏书阁，可是不可多得的机缘啊。”
<br/>　　走在漆黑的通道中，小医仙紧紧跟着慕清霄，约莫五分钟后，终于穿过通道，尽头就有一抹光晕投射而来。
<br/>　　来到通道的尽头，映入眼中的，是一个极为宽敞的房间，在房间四周墙壁上，有着将近数十个能量罩。
<br/>　　此时，这些能量罩正缓缓吐缩着光芒，将整个房间照耀的犹如白昼。
<br/>　　来到房间门口，其中一名男学员盯着小医仙看了一眼，随后目光落在慕清霄身上。
<br/>　　“两位，副校长刚才说，你们是特招生，特招生难道不需要参加选拔赛吗？”
<br/>　　慕清霄微微一笑，道：“当然不是，后面不是还有第二轮选拔赛吗？”
