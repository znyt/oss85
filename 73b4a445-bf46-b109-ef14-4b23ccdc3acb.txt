　　滋滋滋~
<br/>　　下一刻，
<br/>　　叶尘明显感觉到，丹田内被封印的真元，已经可以缓缓释放出来一丝，而他的修为，也从普通人，恢复到了炼气一重境界。
<br/>　　“继续！！”
<br/>　　这个细小的变化，顿时让叶尘大为振奋，当即开始寻找下一目标。
<br/>　　一晃眼，
<br/>　　大半个小时过去了，叶尘已经斩杀了三十多头二阶妖兽和五头三阶妖兽，而他真元实力，此时也已经恢复到了炼气六重！
<br/>　　而随着真元力量的恢复，叶尘斩杀妖兽也变得越发得心应手，即便是三阶妖兽，也能在一分钟之内解决战斗。
<br/>　　而斩杀一头三阶妖兽，所恢复的真元量，足足抵得上十头二阶妖兽！
<br/>　　这就像是滚雪球一样，斩杀的妖兽越快，恢复的真元越多，而接下来斩杀妖兽的速度就会更快，优势越滚越大，真元恢复的也越来越快。
<br/>　　不难想象，这第一关过去之后，肉身强度具有优势的考验者，必定能够取得更大的优势！
<br/>　　很快，一个时辰的时间到了，
<br/>　　叶尘一共斩杀了一百多头二阶妖兽，和近三十头三阶妖兽，真元实力也恢复了炼气境巅峰，里筑基境只有一步之遥。
<br/>　　轰隆！
<br/>　　就在叶尘斩杀完最后一头妖兽的时候，周围空间忽然一阵颤动，一道璀璨的白光自九天之上照射下来，笼罩在叶尘的身上。
<br/>　　下一刻，
<br/>　　叶尘只感觉周围空间一阵变幻，似乎进入了一条空间通道之中。
<br/>　　白光过后，叶尘已经消失不见，显然已经离开了第一重星域，进入到了第二重星域之中。
<br/>　　等到周围的空间再次稳定下来，叶尘已经来到了一颗新星球之上，
<br/>　　“也不知道，这第二重的考验，又是什么？”
<br/>　　叶尘满怀期待，同时开始观察周围的环境。
<br/>　　只见，眼前是一座宏伟壮阔、高耸入云的巨型宫殿，
<br/>　　宫殿的大门，虽然是敞开着的，却被一道无形的禁制所阻隔，所以也就无法看到宫殿之内是什么。
<br/>　　这时，
<br/>　　叶尘的脑海之中，再次蹦出了一条信息：
<br/>　　“眼前宫殿之中，一共有十道禁制，以两个时辰为限，每破解一道禁制，便可恢复一定量的真元，若能全部破解，另有额外奖励……”
<br/>　　叶尘没等将这道信息读完，便直接身体一晃，来到宫殿大门之前，
<br/>　　这第二重考验，和第一重考验的模式，显然是相似的。
<br/>　　只不过，第一关考验的是肉身修为，而这第二关，则考验的是闯关者的精神修为！
<br/>　　当然了，破解禁制阵法，除了精神修为尤其重要之外，对阵法之道的理解，同样十分的重要。
<br/>　　不过，话又说回来，一般心力修行者，几乎都会钻研阵法之道，破解禁制阵法，肯定比寻常人要有很大的优势！
<br/>　　若是前世的叶尘，在这个方面，相比于其他各大天才，肯定不具备任何优势的。
<br/>　　可是，重生之后的他，不但有着前世八百年的记忆，在阵法之道上的造诣，早已经登峰造极。
<br/>　　而且，当初为了保住曦月的残魂，叶尘苦修神念力量，精神力量相比于十大天骄，也肯定丝毫不差，甚至还犹有过之！
<br/>　　轰！
<br/>　　不到五分钟，第一道禁制，便已经被叶尘成功破解。
<br/>　　而与此同时，伴随着一道光芒没入叶尘的体内，被封印的真元力量，也顿时又恢复了一丝。
<br/>　　轰隆！
<br/>　　而叶尘的修为，此时也终于，一举突破到了筑基境。
<br/>　　不过此时，叶尘却顾不得去体会这种修为恢复的快感，而是身体一晃，直接冲向下一道禁制。
<br/>　　第二层的禁制力量，竟然比第一重强大了十倍都不止！
<br/>　　叶尘足足用了近二十分钟，方才将第二重禁制完全破开。
<br/>　　而来的第三重禁制前，叶尘顿时眉头紧皱起来，
<br/>　　第三重禁制的强度，竟然比第二重禁制又强大了十倍以上！
<br/>　　而这一次，叶尘足足用了半个多小时，方才完全破解！
<br/>　　尽管，这种级数的禁制阵法，对于叶尘来说，并不算太难，在半个多小时之内破解，已经算得上是极快了。
<br/>　　可是时间不等人啊！
<br/>　　仅仅只是前三道禁制，叶尘就已经耗费了近一个小时！
<br/>　　两个时辰是四个小时，也就说，叶尘此时只剩下三个小时的时间，而后面却还足足有七道禁制！
<br/>　　而且更重要的是，下面每一道禁制的难度，都要比前一道禁制强大十倍！
<br/>　　按照这个速度下去的话，叶尘恐怕最多也就只能止步于第六道禁制而已。
<br/>　　“难道，这真的是一个无法完成的考验吗？”
<br/>　　叶尘紧皱着眉头，来到第四道禁制之前，虽然心里郁闷不已，可是手上的动作却并没有停止。
<br/>　　很快，便已经破解了第四道禁制的一角。
<br/>　　就在这个时候，叶尘似乎突然想到了什么，猛然一拍自己的脑袋，
<br/>　　“我真是太蠢了！规则并没有规定，一定要完全破开禁制，我只要先破开禁制阵法的一角，然后以我如今的修为，完全可以以力破法，生生撕开一道裂缝，只要我人穿过去不就行了？”
<br/>　　一念及此，叶尘顿时豁然开朗，当即将全部的真元灌入双手之中，然后用力一撕，
<br/>　　刺啦！！
<br/>　　那道已经残破的禁制光幕，顿时被叶尘硬生生撕开了一条裂缝，然后他整个人双脚一蹬地，便已经穿梭而过。
<br/>　　而这一次，前后只用了不到十分钟的时间，就闯过了第四道禁制！
<br/>　　而与此同时，叶尘体内的真元，也再次恢复了一丝，此时他的真元修为，也已经恢复到了筑基三重境界。
<br/>　　这也足以说明，叶尘的判断是对的，只要人通过了禁制，就算是闯关成功，根本不需要将整道禁制阵法完全破解！
<br/>　　改变了闯关方式之后，叶尘的闯关速度，也顿时有了质的飞跃。
<br/>　　又过去一个多小时，叶尘已经闯到了第八道禁制，而此时离两个时辰的限制，还有足足一个半小时的时间！
<br/>　　只不过，等叶尘闯到第九道禁制前，此时眼前这道禁制阵法的难度，已经达到了极为变态的地步！
