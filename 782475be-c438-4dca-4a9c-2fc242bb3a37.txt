　　鲁克森地区，一处相对落后的地区。这里密林丛生，物产丰富，一个个部族团体汇聚于此。
<br/>　　一处陡峭山崖之上，十多个人站在一起。一个小村落就在山崖下不远处，炊烟渺渺充满平静。
<br/>　　“侠客，是这里吗？”领头的是一个年轻男子，幻影旅团团长库洛洛。一头黑发梳理的整整齐齐，双眼睛平静幽深，最显眼的是纹刻在额头的十字架，华丽而神秘。
<br/>　　“团长，就是这里。火红眼塔库鲁一族，就隐居在这个村子里。”侠客一头金发，脸上带着开朗笑容，好似一个阳光大男孩。
<br/>　　“团长，下决定吧。我的拳头已经迫不及待了！”窝金满脸的狂热笑容。两米高的大汉，满身都是岩石般坚硬肌肉。头发根根直立，裂开满是尖牙的大嘴，笑的格外狰狞。
<br/>　　“目标，火红眼，所有火红眼。”库洛洛平静道。
<br/>　　“哈！”窝金一声大喝，直接从山崖扇一跃而下。
<br/>　　轰！
<br/>　　地面猛的一震，一圈烟尘扩散开来，一个大坑出现在地面上。
<br/>　　“你们是什么人？”村子里顿时被这惊天动静惊动。
<br/>　　烟尘散尽，窝金半蹲土坑中，猛然抬头，满脸狞笑：“杀你们的人！”
<br/>　　砰！
<br/>　　一个染血的拳头，从开口问话的村民背后冲出。
<br/>　　瞬间，整个村子陷入血色的混乱。
<br/>　　……
<br/>　　旋律觉得这些天的经历，就像是挥之不去的噩梦。只是想一想，就会让她痛苦万分。
<br/>　　无意中得到暗黑奏鸣曲，打定主意不演奏。却在酒精刺激下，最终没压住心中好奇。
<br/>　　当第一个音符响起的瞬间，所有人都堕落了。无法自拔的沉浸其中。等她再次清醒，所有事情都变得不同了。
<br/>　　自己浑身长满青绿鳞甲，身材矮胖、面容大变。她对着镜子看了一天，却从里面找不到任何熟悉的地方。
<br/>　　她再也没见过当初的同伴，旋律已经猜到了结局。
<br/>　　她绝望了！
<br/>　　即便因此获得了神秘的能力念，却依旧无法让她释怀。她曾经自杀，但却被阻止了。
<br/>　　那个把她救出来的少年，冷酷的像一团冰，即便她想死都办不到。
<br/>　　旋律绝对不会认为对方是好心，直到今天，她仍任记得第一次自杀的场景。
<br/>　　对方轻易掰断自己手中刀子，然后冷漠的盯着自己道：“你不能死，我要从你身上研究暗黑奏鸣曲的奥秘。如果你再敢自杀，我会打断你的四肢，割掉你的舌头。让你失去自杀的能力。”
<br/>　　那毫无情绪波动的声音，那古井无波的眼眸，让旋律心底发寒。那不是在看一个人，好似在看一块石头、一株小草。
<br/>　　呼！
<br/>　　一阵狂风突然吹来，旋律的宽大衣服咧咧作响。
<br/>　　抬头眺望远方密林，绿色的光影如同海浪一般席卷上空，晶莹光芒美丽无比。
<br/>　　点点绿光汇聚成溪，快速向中央汇聚。一眨眼的功夫，漫天的绿光消失干净。原地只留下一堆枯树。
<br/>　　刚才还生机勃勃枝繁叶茂，眨眼间就枯萎衰败，完全死亡！
<br/>　　“这才是真正的魔鬼！”旋律眼中带着丝丝惊恐。
<br/>　　一个月以来，她不止一次见到这种场景。每次看到，都让她不可遏制的恐惧。
<br/>　　吸收树木生命，几个呼吸的功夫，就让周围树木全部死亡。这种能力，如妖似魔！如果要是用这种能力对付人……
<br/>　　只是想一想，她就忍不住的打了个寒颤。
<br/>　　吼！
<br/>　　一身洪亮且残暴的咆哮在脑中回荡。苏重睁开眼睛，眉头微微皱起。
<br/>　　自从他开始利用黑烟尝试练体，脑中就一直存在那种吼叫声，而且越来越大。
<br/>　　低头看向手臂。手臂皮肤微黑，却不是以前那种太阳照射所造成的黑。而是带着晶莹剔透的感觉，好似放着微黑光泽的玉一般，一种内敛的黑色。
<br/>　　手指弯曲起来，弹了弹手臂。
<br/>　　铛铛铛！
<br/>　　一种金属碰撞的声音响起。苏重满脸怪异：“这不会练成了钢了吧？”
<br/>　　忍不住用手捏了捏，却入手柔韧绵软，一点也没有金属的脆硬感。
<br/>　　“暗黑奏鸣曲，到底是什么东西？”一个月以来，苏重不止一次的升起这个疑问。
<br/>　　旋律的身体遭到了极大的损害，但苏重多次探查却发现，黑烟侵蚀旋律的身体，并不是一味的掠夺生命力，而是在进行一种体质改造。
<br/>　　“可惜，旋律只是普通人，身上生命力没有得到开发，不能够支撑完全改变。”
<br/>　　而使用黑烟练体的苏重，也在进行着某种转变。
<br/>　　皮肤防御更强大，身体更强壮，除了那个不知名的吼叫声，苏重没发现任何异常，所以并未停止利用黑烟修炼。而且现在黑烟和道种融合，发生了某些奇妙变化，他不用也没办法。
<br/>　　“你什么时候放我走。”旋律瞪着走过来的苏重，愤怒道。
<br/>　　苏重看了一眼旋律，思考了片刻：“你现在就可以走。”
<br/>　　旋律一怔，现在就能走？她几乎怀疑自己的耳朵。自己每天都要问这个问题，每次的答案都是拒绝。没想到这次竟然让自己走。
<br/>　　“你真让我走？”她心中忐忑，死死盯着苏重。这家伙不会是被自己问烦了，要杀人灭口了吧？
<br/>　　苏重毫不在意旋律的质疑：“你已经没用了，我自然会放你走。”
<br/>　　旋律闻言一滞，旋即长长松了一口气。
<br/>　　在觉醒念之后，她获得了一种能力。他能听到别人的心跳声，能通过心跳知道对方的情绪变化，也能辨别对方是不是撒谎。
<br/>　　刚才苏重心跳声，一如既往的平静，或者说冷漠。不过，自己真的自由了！
<br/>　　看着苏重，旋律倒退着往后走，走出十多米，见苏重盘坐在地闭目养神，真的没有杀她的意思，这才转身离开。
<br/>　　可刚走了两步，浑身一僵，停在了原地。
<br/>　　暗黑奏鸣曲赋予了旋律独特的念，使她的念和声音息息相关。
<br/>　　就在刚才，清风送来了一缕声音。一缕充满了恐惧、不甘、怨恨、愤怒的声音。那声音中冲天的负面情绪，让她整个人呆愣当场。
<br/>　　苏重睁开眼，疑惑的看着旋律。她站在不远处双拳紧握，浑身颤颤发抖，眼睛死死的盯着天远处边发。
<br/>　　那里发生了什么吗？他知道旋律的能力。借助声音，旋律能够探查到很远处的地方。比苏重的圆还要厉害。
<br/>　　苏重下意识的把圆放出，两公里之内一切纤毫毕现。可苏重并未发现什么异常。
<br/>　　心中一动，圆的形状立刻变形。从半球状不断变细变长，探知范围瞬间扩大。
<br/>　　然后一双眼睛涌入心田，带着血色的眼睛！
