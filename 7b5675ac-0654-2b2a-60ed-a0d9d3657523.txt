        季雅姗抬眸看了童筱颖一眼，回答道：“你今后的婚姻生活可能会不太顺！”
<br/>
<br/>        “可没有谁的婚姻生活是百分百顺心的啊！”童筱颖会心一笑。
<br/>
<br/>        季雅姗随即将牌收好，一脸无所谓的样子，反驳道：“这种东西，信则有，不信则无。你不信就对了！”
<br/>
<br/>        然而，下一秒，季雅姗又变脸了，不知从哪儿掏出一条红色的编绳来，放在了童筱颖的面前。
<br/>
<br/>        童筱颖看着这条编绳，手工非常精致，花纹繁复编得特别好看，便问道：“这条手绳很漂亮，你在哪里买的？”
<br/>
<br/>        “我自己编织的，而且还去教堂请神父开了光噢！专门解霉运的，既然你婚姻不顺，要不买一条回去破霉运？”季雅姗眯眼一笑，大大咧咧地问。
<br/>
<br/>        其实，上次她的救命恩人的那条青丝手链也是她自己亲手编织的，只不过，她觉得，自己亲手做的不上档次，怕救命恩人嫌弃，所以才谎称是知名手艺师傅编织的。
<br/>
<br/>        童筱颖拿起这条手绳，直接戴在了左碗上，问道：“多少钱？”
<br/>
<br/>        “不如，这顿你请我吃。这条手绳，就送你好了！”季雅姗微微一笑。
<br/>
<br/>        童筱颖忍俊不禁道：“看在你的手绳这么漂亮的份上，我就请你吃这餐好了。”
<br/>
<br/>        “你就这么好骗啊？”季雅姗嘴角一敛，打趣地说。
<br/>
<br/>        童筱颖将手绳戴好后，举起手来在季雅姗眼皮子底下晃了晃，笑容灿烂：“看在你的手绳很漂亮的份上，我心甘情愿地被骗！”
<br/>
<br/>        其实，她也不知道为什么，会和这个女孩格外投缘。
<br/>
<br/>        而季雅姗打从她俩见面的那一刻开始，就从童筱颖身上闻到了一股气味非常非常淡的香气，而这个香气，和救命恩人的那条青丝手链上的香气很像。
<br/>
<br/>        “我们交个朋友吧！你好，我叫季雅姗，今年十七岁半！”季雅姗随即一边自我介绍着，一边向童筱颖伸出了手来。
<br/>
<br/>        童筱颖看了看季雅姗伸过来的手，毫不犹豫地握住，微笑道：“你好，我叫童筱颖，比你大一岁！”
<br/>
<br/>        “你们乡下来的女人，是不是结婚都比较早啊？”季雅姗大大咧咧地问。
<br/>
<br/>        童筱颖倒是一点都不介意季雅姗“用乡下来的”来形容自己，因为，第一，她不觉得自己来自乡下就低人一等，第二，她并不觉得季雅姗在歧视她。
<br/>
<br/>        季雅姗说话只是性子直，大大咧咧了而已，心肠并不坏。
<br/>
<br/>        童筱颖心宽度量大，释然一笑地回答：“算是吧！但不一定，个人因素决定。”
<br/>
<br/>        “如果，你缺钱用的时候，你会委屈自己，被有钱的长得挺帅的已婚男人包养吗？”季雅姗突然黯然神伤地问。
<br/>
<br/>        童筱颖怔愣了一下，看着季雅姗那张稚气未脱，有点儿婴儿肥，且肌肤吹弹可破的脸，不知怎么的，就心直口快地回答道：“不会！”
<br/>
<br/>        她确实有过缺钱用且急需钱用的经历。
<br/>
<br/>        只不过，那个时候，她选择的是相亲，把自己嫁出去拿点彩礼钱回来给童爸童妈还债。
<br/>
<br/>        那是她走得最极端的一条路了，但是，如果那个时候，真有这么一个已婚有钱又长得帅的男人出现说要包养她的话，她会毫不犹豫的拒绝。
<br/>
<br/>        或许，每个人对待同一件事情的方法、观念以及背景不同。
<br/>
<br/>        这对于季雅姗来说，就是如此。
<br/>
<br/>        那个时候，她只有那一个选择，也只有那一条捷径可选。
<br/>
<br/>        是她的亲生父亲，以及她亲生父亲的那位“正宫娘娘”将她逼上这条绝路的。
<br/>
<br/>        “如果，当时，你妈妈病了，急需救命钱，再晚就没命了。而且，那种情况，只有那个男人会一下子就给你那么多钱，你还会拒绝吗？”季雅姗步步紧逼，眼神里晃动着难以言喻的黯光，就像是在叙说自己内心的痛苦。
<br/>
<br/>        童筱颖怔了怔，看着季雅姗的眼睛，态度不再像刚刚那么果决：“我、我不知道……但是……我觉得……应该可以谈……可以跟那男人借，不一定非得被他包养才行……”
<br/>
<br/>        季雅姗眸光微抬，对上了童筱颖的目光后，淡然冷笑：“呵，如果有得谈，对方就不会提出那样的要求了。”
<br/>
<br/>        童筱颖垂下眼帘，瞬间沉默了。
<br/>
<br/>        “以后常联系，好吗？”季雅姗画风一转，忽然间不再那么黯然神伤。
<br/>
<br/>        童筱颖再次看向季雅姗，微笑着点了点头。
<br/>
<br/>        于是，她俩最后还交换了电话号码。
<br/>
<br/>        吃过午饭后，童筱颖陪着季雅姗在餐厅里休息了一会儿，才顺路送季雅姗回学校。
<br/>
<br/>        临别前，季雅姗一脸认真地对童筱颖说道：“你是第一个愿意跟我交朋友的人，我会好好珍惜你的！”
<br/>
<br/>        “嗯，我也会珍惜你这个朋友。”童筱颖挥了挥手，欣慰地笑着。
<br/>
<br/>        季雅姗眉开眼笑，背着书包，转身跑进了校门内。
<br/>
<br/>        童筱颖目送季雅姗走进学校，同时也感慨，这滨江一中真的比她那个西镇上的高中要大很多。
<br/>
<br/>        人各有命，但她不会因为自己命不好而自暴自弃。
<br/>
<br/>        她也有过人生的低谷时期，但是她没有就此放弃自己。
<br/>
<br/>        傅氏集团旗下地利置业大厦。
<br/>
<br/>        总经理办公室里，傅昼景得知大哥傅夜沉将一个五个亿的项目交给堂叔冷岩峰去负责后，心里十分不是滋味。
<br/>
<br/>        换做以前，总部那边如果有什么上亿的项目，都会先让内部的子公司的总经理带着自己的团队公平竞选，中标获胜者才有资格去负责那个项目。
<br/>
<br/>        但是，大哥这还没正式上任，就直接将项目委派给堂叔，是什么意思？
<br/>
<br/>        傅昼景心里不服，但又无处发泄，只能在心底暗自抱怨。
<br/>
<br/>        他不知道是不是自己产生了什么错觉，总感觉，自从大哥退役从商后，在交接之际，傅氏集团旗下子公司的实权正在被大哥一丁点一丁点地收了回去。
<br/>
<br/>        傅氏集团旗下，有三个大子公司：天时、地利、人和，分别为傅氏集团负责商界各领域的业务。
<br/>
<br/>        而三大子公司的总经理，就类似于封地的王爷，各自握有自己所负责的子公司的权利。
