　　所以说尉迟宝琪完全可以说是自投罗网，大食人围攻李二的事情原本跟他没有任何关系，但是他却一头撞了进去。
<br/>　　不过也正是因为有了这小子，李世民的舰队才能够坚持下来，没有被那些以大食人为主的番邦彻底困住。
<br/>　　可就算是这样，他们也没有办法继续在这一片水域鏖战下去了，淡水的补充是一回事儿，在武器装备方面，他们同样面临着弹尽粮绝的困境。
<br/>　　热武器的应用固然是大大的提升了大唐军队的战斗力，可同样也使得大唐军队对后勤的依靠越来越重。
<br/>　　老头子当初出海的时候，李承乾也没有想过这老头儿那么能折腾，竟然直接跑到了澳洲那片大陆玩了一圈，又跟各个地方的土著打了好几场小规模的战争。
<br/>　　所以尽管当时给老头子装了很多的弹药，但到了最后的这段路程，李世民的舰队实际上已经没有多少战斗力了。
<br/>　　……
<br/>　　是夜，一切战略战术都已经布置完成，李世民拖着疲惫的身体回到了属于他的船舱。
<br/>　　长孙皇后已经等了他一天，见他回来立刻迎了上来，一边吩咐侍女去泡茶，一边嘘寒问暖的问道：“二哥，怎么样？商量出什么结果了没有？”
<br/>　　“明日我们就会转移，向着东南方的一座岛进发，只要到了那座岛，我们基本上也就安全了。”李世民握着老婆的手，笑着说道。
<br/>　　“嗯！”长孙皇后点了点头，想了想又说道：“只是不知道乾儿怎么样了，我们被困在这里，如果他知道消息一定会急坏了吧！”
<br/>　　“那小子不会收到我们消息的。”李世民拍了拍长孙后的手说道：“小尉迟来的时候已经说了，我们那些驻守海岛的军士已经全都被人给杀了，所以我们的消息根本传不出去，那小子现在能得到的，应该只有我们在某个地方失踪的消息。”
<br/>　　“怎么会这样？如果是这样的话，那岂不是说……岂不是说我们连援兵都不会有？”长孙皇后也不是什么都不懂，听了李世民的话很快就意识到会发生什么。
<br/>　　“听天由命吧，如果那小子足够聪明，他应该可以猜到我们这边发生了什么，到时候他自然知道应该怎么做，如果他猜不到……，观音婢，朕……”李世民说到这里顿了一下，接下来的话没有说出口。
<br/>　　但长孙皇后已经听出他话里的含意是什么，微微一笑轻轻揽住他的胳膊，将头放到李二的肩膀上：“二哥，你我夫妻一场，无论生死，妾身都会陪在你的身边。”
<br/>　　“观音婢，现在说这些还言之过早！”李世民心中微微发酸，抽出被长孙皇后抱在怀里的胳膊，将她揽入怀中：“朕不会轻易放弃的，朕曾经贵为天子，这个世界上想要朕这条命的人不知凡几，但朕不是依旧还活的好好的。”
<br/>　　“嗯，妾身相信您一定会吉人天相，这一次我们一定可以平安回到大唐的。”
<br/>　　“回大唐？”李世民沉默片刻：“这一次若是他们不能要了朕的命，朕便会要了他们的命，若是不将他们斩尽杀绝，朕誓不为人！”
<br/>　　李世民的声音虽然不大，但语气却异常坚定，在长孙皇后的眼中，自己的男人似乎又回到了二十多年前征战沙场时的样子。
<br/>　　……
<br/>　　寂静的长夜缓缓过去，随着第一缕天光透过窗子照进船舱，李世民已经换上了昔日的战甲，带着长孙皇后的嘱咐来到了战舰的甲板之上。
<br/>　　“准备的怎么样了？什么时候可以发起进攻。”看着已经在般头集结的长孙无忌、秦琼、独孤青云等人，李世民沉声问道。
<br/>　　“只等您的命令了！”独孤青云作为这次行动的总指挥，来到李二的面前说道。
<br/>　　“那就开始吧，别让我们的敌人久等了。”李世民嘴角微微一挑，露出一个曾经让所有人都万分熟悉，但却有二十一年没有见过的微笑。
<br/>　　当初玄武门的时候，他的脸上露出过同样的笑容，那是孤注一掷又信心百倍的笑容，现在，这笑容再一次回来了，不管是长孙无忌，还是秦琼、尉迟还是独孤青云，所有人都是心中微微一凛。
<br/>　　但无论如何，仗还是要打的，得到命令的独孤青云用力一抱拳转身而去，时间不大，三团代表着反击开始的信号烟升上天空，
<br/>　　一时间，所有被围在钢铁战舰中间的那些木质的补给战船动了起来，向着东南方向鱼贯而出，每四艘战船一排直直向着远方而去，而在这些木质战船的后面，十二艘钢铁战船排成两列纵队紧随而上。
<br/>　　大唐舰队的动静很快便引起了那些大食人的注意，早有人将消息飞报到了旗舰之上。
<br/>　　阿里·伊本作为这次行动的总指挥，对于那位大唐太上皇的动静自然是时刻注意，听到手下的报告之后，立刻也拿出海图。
<br/>　　只是大食人的海图标注的并不怎么清楚，他在上面找了半天也没有看出东南方向到底有什么。
<br/>　　但虽然他不知道为什么大唐的舰队没有向北而去，而是反其道而行，可这却不耽误他做出拦截的判断。
<br/>　　作为敌对双方，敌人想做的，就是自己必须要破坏的，所以不管大唐那位太上皇想要干什么，他都必须将他拦下来。
<br/>　　于是命令一道又一道的发出，大食人和一些其他势力的战船开始集结，阻拦到了大唐舰队的前方，一片又一片渔网被丢下水，然后他们开始后退。
<br/>　　不过与以往不同的是，大唐战船虽然已经看到了大食人的动作，但却依旧没有停下来的打算，反而加快速度冲了上来。
<br/>　　而且大唐的战船之间距离很近，近到只要大意一点点，很可能就会撞到一起的程度。
<br/>　　这使得阿里·伊本愈发的搞不清楚大唐舰队的目的，因为这根本就不符合海战的套路，而是骑兵冲锋的布局。
<br/>　　把战舰当成骑兵，老实说，阿里·伊本还是第一次见到，所以他的心立刻提了起来，对大唐舰对的关注也开始谨慎起来。
<br/>　　大战在这一刻一触即发！
