　　第471章
<br/>　　樊梨花怔了怔，有些奇怪程大雷的问题。但程大雷问得如此认真，她也不好回答。
<br/>　　“好像没说什么话吧。”
<br/>　　“那就好，那就好。”
<br/>　　程大雷拍着胸脯，长长的松了口气。要杀薛丁山的心情如此强烈，一是因为这薛丁山委实有些恶心人，二来，人家可是官配，万一二人一见面，针尖对麦芒，天雷勾地火怎么办。
<br/>　　幸运的是，这个恶心人的货终于被被自己砍了。程大雷踢了踢地上薛丁山的尸体，只感觉神情气爽。
<br/>　　嘟，恭喜你击杀薛丁山，奖励抽奖次数1 。
<br/>　　程大雷眼睛一亮，原来击杀薛丁山，还能够爆东西的。这薛丁山，妥妥的属于boss级别了。不过，抽奖也不知道会抽出来什么，假若抽出一个绝世魔星，系统再附赠自己一个绝世正义使者……一个薛丁山就如此难缠，再出来一绝世等级的，程大雷当真不知该如何是好了。
<br/>　　先放下这次抽奖机会，以后再用，目前收拾眼前的战局要紧。
<br/>　　程大雷和樊梨花并肩厮杀，这场战斗很快便结束了。蛤蟆城大获胜利，几乎全歼帝国军。同样的兵力比，蛤蟆城却占有绝对优势。
<br/>　　战斗结束后，刘悲、关鱼、张肥、赵子龙、高飞豹、高飞虎、秦蛮等同时聚在程大雷身边。
<br/>　　“大当家。”诸人齐声道。
<br/>　　“现在情形怎样？”
<br/>　　“帝国军已被击溃，逃走了一些。”刘悲道：“另外还有些俘虏，愿意投降我们，大当家看怎么处理。”
<br/>　　击溃万余人，斩敌三五千，另外还留下一两千俘虏。
<br/>　　在程大雷来到他们身边时，这些俘虏跪在地上，大呼饶命。
<br/>　　“程当家，我们愿意投降，只求大当家刀下留情。”
<br/>　　尤其是陆承喊得最响：“程当家，我知道相府的情报，徐军师，我们关系一直不错，您为我求求情。”
<br/>　　诸人目光看向程大雷，询问他该如何处理。
<br/>　　程大雷的手扬起，又重重落下，口中吐出一句话。
<br/>　　“杀，一个不留。”
<br/>　　没有人违抗程大雷的命令，留下的俘虏统统丧命在长矛短刀之下，空留几句惨叫，在草原上回荡。
<br/>　　程大雷冷冷看着这一幕，表情没有丝毫动容。
<br/>　　刘悲啧地叹了一声：“杀降不详，杀了他们无足轻重，可对我们名声有损呐。”
<br/>　　刘悲刚才没有反对程大雷，但并不代表他认同程大雷的作法，此刻，只有二人时，他还是说出了自己的想法。
<br/>　　程大雷没有反应，只是吐出一句话：“林少羽死了。”
<br/>　　刘悲点点头，却又摇摇头，并没有说什么。
<br/>　　事后，秦蛮几人来到程大雷身边，留下的俘虏一个不留，全部被杀光。
<br/>　　“大当家，我们是特意来接应您回去，一切都是我自作主张，要怪您就怪我，莫责怪其他人。”秦蛮。
<br/>　　“秦老大，怎么能说是一个人的主意呢，明明是我先想到的。”张肥。
<br/>　　“嗳，二位也不必争了，明明是我的主意。”高飞虎。
<br/>　　程大雷无语的看着这一幕，蛤蟆城虽然颇有自由散漫的风气，但军令执行甚严。这次他们私自出兵，虽然解了程大雷的围，但回蛤蟆城后，定然是一顿责罚少不了。
<br/>　　“好了，你们的事先记下，回去后一个都跑不了。”程大雷话锋一转：“我们先不走了。”
<br/>　　“大当家，咱们不回蛤蟆城了么？”秦蛮问。
<br/>　　“少羽死了。”程大雷看着他的眼睛道。
<br/>　　秦蛮一怔，随后扣紧了牙关，重重的点了点头。
<br/>　　“我的兄弟不能白死，死的人已经死了，活着的人总要为死了的人做些什么。”
<br/>　　程大雷念叨这句话时，脑海中又想起林少羽惨死在自己面前的一幕，不仅仅是如此，更多是回忆起曾经。初见时的狼狈，这狼狈不仅仅是林少羽，也包括自己，随后一次次攻上山的可笑……再然后，便是越来越偏执，越来越疯狂，直到，惨死在自己身前，那时，距离自己那么近，自己只能看着他血肉无存。
<br/>　　这个坎儿，程大雷至今没有过去。只不过带着残兵逃脱，他必须冷静下来，压抑自己崩溃的情绪。
<br/>　　但现在，他有了人，有了不远万里从蛤蟆城过来，自己亲手调教出来的一万精兵。
<br/>　　养兵千日，用在一刻，用在，这一刻。
<br/>　　其实程大雷很多事情都想不明白，关于这个扑面而来的乱世，关于未来该何去何从，但他有一件事，总是想得明白。那就是，总要为死去的人做些什么。
<br/>　　与程大雷想象得截然相反，在知道暂时不回蛤蟆城后，这帮人并没有任何不满情绪，个个磨刀霍霍，似乎就等着出刀收割人头。
<br/>　　二百义军的尸体被找到，包括白平在内，义字军全体阵亡，到现在，林少羽带出来的这支起义军，死得一个不剩。不得不说，疯子一般的林少羽，带出来一支疯子般的队伍。
<br/>　　“高飞虎，你带一千人去青牛山。”程大雷下令。
<br/>　　“呃，去青牛山做什么？”
<br/>　　“有点东西，现在时候到了，也该取出来了。”程大雷。
<br/>　　高飞虎临走前，白起不是很情愿，他也想为林少羽多杀几个帝国军。
<br/>　　秦蛮几人聚到程大雷身边，问：“大当家，下一步该怎么办，您下令吧？”
<br/>　　程大雷昂起头，看到蛤蟆城的兵都立在自己面前，人人表情肃穆，目光像坚硬的石头。
<br/>　　“自由行动，每支队伍各自为战，做，你们想做的事情。”
<br/>　　“是！”
<br/>　　所有人一起答应，秦蛮、关鱼、张肥、高飞豹、赵子龙各自带兵出发，这片草原，将会成为他们的猎场，猎杀敌人，也可能会被敌人猎杀，程大雷花三年磨一把刀，这把刀究竟锋不锋利，这次就可见到效果了。
<br/>　　只有刘悲、徐神机、银眸、单雄信陪在程大雷身边，他们看到程大雷一个走远，嘴里轻轻念叨着一句话。
<br/>　　“正义之路被暴虐之恶人包围，以慈悲与善意为名引导弱者，那些胆敢荼毒残害我同伴之人，我将向他们大施报复。”
<br/>(本章完)
