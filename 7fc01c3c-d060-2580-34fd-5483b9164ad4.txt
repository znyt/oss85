<br/>　　白允儿对步纱的古怪脾性也不在意，相对来说她是最古怪的那个才对，没有任何停留的意思便直接转身离开。
<br/>　　在沉默中缓了好一会儿劲的白羽并没有第一时间跟上白允儿的脚步，而是先深呼吸了几下，而后猛然大吸一口气吼道：“好你个美阎罗你给我等着！等我到十阶！我一定回来报仇！”

<br/>　　白允儿嘴角微不可查的一勾，脚步不停的快速消失在这片山谷里。
<br/>　　小屋内的步纱眉眼冷光一闪而过，带着嗔怒出声：“小屁孩就是烦！”

<br/>　　此时小屋内的一个角落里传来一阵咳嗽声，伴随着丝丝水滴溅落在地的声音一起。
<br/>　　步纱转身望去，便看到被她救下的那个人已经醒了，正在咳血。

<br/>　　“命挺大，这都没死。”步纱面无表情的出声，更是随意的找了张椅子坐下，就这么毫无情绪波动的看着那个伤员在不住咳血，凄惨无比的样子不能让她有哪怕一丁点的怜悯之情。

<br/>　　此时的他浑身是血，有干涸的黑色血迹，也有因为刚才咳血而崩裂伤口溢出的新鲜血液，总之整个人都看上去随时会死，但他还是一只手捂住胸口，努力让自己能开口说话。
<br/>　　“你，你就是美阎罗步纱？”江凌轩的声音里满是震惊，也不知道是在震惊自己竟能见到步纱的真容，还是在震惊步纱呈现在他面前的冷漠无情样子。

<br/>　　刚醒过来的江凌轩状态还算好，至少不会咳血，毕竟伤口大多都已经闭合，整个身体也都在缓慢的恢复中。

<br/>　　这一切都是拜了那‘美阎罗’三个字所赐！
<br/>　　美阎罗，竟然是美阎罗！
<br/>　　这个在猎者联盟内无比神秘的女人！
<br/>　　要知道美阎罗的来历身份甚至一切，都没人知道，她一直都是猎者联盟中占据着最高机密档案的那一个，甚至还有人怀疑美阎罗压根不是女人。
<br/>　　如此神秘的人在江凌轩差点死掉后苏醒的第一时间就遇到，如何能让他不惊讶？
<br/>　　步纱的表情怪异了一秒，有些好笑的开口：“是，见到我有这么奇怪么？”
<br/>　　江凌轩回了回神，终于将情绪稳定了下来，苦笑道：“有生之年能见到猎者联盟内最神秘的猎者真容，我也是值了。”

<br/>　　“一周？！”江凌轩大惊，霍然而起之时，更是牵动了伤势。
