　　第695章
<br/>　　蛤蟆城诸多人，对程大雷来说，其实也由远有近。论起来，从青牛山走出来的第一批人，关系自然亲近些，都是从死人堆里打滚爬出来的。之后再加入的，关系便略远些，有些人连程大雷的面都没见过。
<br/>　　手下有精兵三万余，文武官员也有不少，许多人程大雷自己都识得。但偏偏这孙小毛，程大雷却还当真知道。
<br/>　　年轻并不大，跟着程大雷时，才不过十七八岁，这几年算是长成了人。他是祖传的山贼，父母爷爷都是山贼，其父是死在青牛山与戎族的战斗中。
<br/>　　程大雷紧握的拳头慢慢松开，之后长长叹了口气，心里才开始觉得这件事不太好处理。
<br/>　　年轻气盛，拼着脑袋打仗，在破城之后玩个把女人在这里实在不算什么。但他又的确违反了程大雷定下的军纪，骚扰女眷者，斩立决。
<br/>　　杀了其也有好处，让凉州城的百姓看看，本当家言出法随，爱民如子。如此，可以好好收割一波人心。
<br/>　　而问题是，程大雷也不觉得这件事如何，他手下山贼居多，走投无路的亡命徒居多，虽然程大雷从严治军，将他们训得令行禁止。但很难在道德方面对他们有过多要求。
<br/>　　毕竟，破城之后，就连程大雷也占了崔白玉这样的无瑕美人，张肥高飞豹这等人也都参与分赃。如孙小毛这样的小兵，当然不会有他的份儿，可总也不能只许州官放火，不许百姓点灯。
<br/>　　是杀了他，换来人心。还是放过他，为了他死去的父亲。
<br/>　　程大雷忽然感觉为难起来，他也并非圣人在世，有兼济天下的胸怀。本身善恶观其实也很模糊，可不杀他，如何服众，如何安抚凉州百姓。
<br/>　　程大雷长叹一声，忽地发觉自己也变得如同枭雄一般，要在人心和情义间做出某种选择了。
<br/>　　苏樱伸出手，抚平他紧皱的眉头，道：“很难办？”
<br/>　　程大雷苦笑一声，轻轻点了点头。
<br/>　　“我有一个法子，却不知当说不当说。”苏樱。
<br/>　　程大雷咳嗽一声：“这里又没有外人，你我也不是外人，有什么不能讲的。”
<br/>　　苏樱缓缓道：“你如果觉得难办，就对这件事装作不知，让吴用当这恶人，也不损你们兄弟情分。”
<br/>　　程大雷握住苏樱的手腕，道：“我只是没想好要怎么做，要杀，或者不要杀。可假若我决定要杀，这个恶人就不能让别人替我做。”
<br/>　　苏樱抿起下唇，拍了拍程大雷的手背，道：“是我说错话了。”
<br/>　　程大雷为难的并非是要不要做这恶人，而是他过不去心中这个节。今次孙小毛犯错可以杀，但假若犯错的是高飞虎、高飞豹或者徐神机这等人呢，看这帮人平时为人处事，实在不是什么好人，做出这等龌龊事也不稀罕。到时候自己还舍得杀么……
<br/>　　他们对自己当然很重要，有同生共死的情分在。可那被糟蹋的女子，对她的家人也是很重要。
<br/>　　程大雷手中的兵将猛虽猛矣，但实在不是什么善良之辈。孙小毛的事情可以缓些处理，但兵员素质问题必须着手解决了。
<br/>　　程大雷抄起桌上笔，重重写下两个字。
<br/>　　练兵。
<br/>　　笔走龙蛇，依旧如鬼画符一般，可苏樱已读懂了其中的杀气。
<br/>　　外面有人通禀，吴用求见。程大雷手下的文武，都不可擅入城主府后院，这才是张肥七拐八拐，让奥黛琳娜过来说情的原因。
<br/>　　程大雷在书房见到了吴用，此子十分狼狈，披头散发，脸上挂着淤青。
<br/>　　“什么事，搞成这般样子？”程大雷心情本不好，说话也板着一张脸。
<br/>　　“那个……张将军想要将孙小毛抢走，我顶撞了其两句，便是这般样子。”
<br/>　　“什么？”程大雷眼皮直跳，阴沉着一张脸：“带我去看看。”
<br/>　　张肥是想要保孙小毛的，吴用是要杀的。如今孙小毛是看押在城主府的大牢中，张肥还是想着关进自己营中更保险些。负责的吴用自然不肯，于是双方就动起手来。
<br/>　　程大雷气冲冲奔赴大牢，心情实在糟糕到了极点。倒也不仅仅是因为张肥的胡闹，而是一些其他事情。
<br/>　　蛤蟆城的势力越来越大，是已经不是当初活过今天没明天的日子。而武将与文官的矛盾也便越发明显，张肥敢这么做，未必没有居功自高的意思。而吴用搞得这么狼狈，也未必是张肥造成的，说不得特意打扮得可怜些，激起自己的怒火。
<br/>　　都是人心，都是算计，都……特么的是混蛋。
<br/>　　城主府距离大牢并不远，程大雷没走几步就到了，见大堂闹成一锅粥相仿。
<br/>　　几个武将和几个文吏吵得不可开交。
<br/>　　“老子拼死打仗的时候你们在哪里，没死在敌人手里，却死在自己兄弟手中，那里的道理。”
<br/>　　一群莽汉乱糟糟大叫，都是血性汉子，嗓门很大，尤其是其中的张肥，更是杀机赫赫，威风八面。
<br/>　　突然，所有声音都消失，大堂内变得十分安静。已经有人发现了门口的程大雷，垂着头颤巍巍发抖。
<br/>　　程大雷目光冷冷扫过，随之轻轻拍了拍手掌：“好威风，好杀气，张将军要在此地杀人么？”
<br/>　　张肥一激灵，道：“属下不敢。”
<br/>　　程大雷迈步从人群中间经过，在书案后坐下。目光打量着堂内的情景，武将自知犯错，垂头不语。文官笔吏自以为有了靠山，挺胸抬头，就连吴用嘴角也翘起半分。
<br/>　　程大雷的手已经举起，不知该如何落下。若是罚得太重，便助长了文官的气焰，如果罚得太轻，却又会让张肥等人变本加厉。
<br/>　　都需要算计，都需要把握尺度。
<br/>　　一种深深的疲惫感从心底涌上来，程大雷抬起的手无力的挥了挥道：“散了吧，散了吧。”
<br/>　　拳头高高举起，轻轻落下，张肥等人面面相觑，但也察觉到了程大雷的不开心，识趣的退后。
<br/>　　吴用等人也是一头雾水，程大雷难道真的是忌惮武将势力，连一句重话都不给。
<br/>　　只有程大雷一人窝在椅子上，身边人很多，却像孤家寡人。
<br/>(本章完)
