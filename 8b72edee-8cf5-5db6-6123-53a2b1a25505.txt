　　叶少阳离她还有一段距离，都明显感觉到了火焰的炽烈。
<br/>　　“啊……”芮冷玉站起来，朝叶少阳看过来。
<br/>　　叶少阳这才注意到，她的眼睛变了，看上去一片血红，却不是那种完全的，而是不满了红色的血丝，表情盛怒，看上去像是一个来自地狱的魔女。
<br/>　　她缓缓抬起手，松纹古定剑自动升起来，落在她手中，一抹血色的煞气，从剑柄的地方开始往下蔓延，很快就蔓延到了整个剑身，这把龙虎山镇山之宝，此刻却是充满了一种邪恶的气息。
<br/>　　芮冷玉抓着宝剑，对叶少阳劈砍过来。叶少阳急忙挥剑抵挡，惊愕的发现，芮冷玉比之前又强大了好几倍，自己……几乎不是对手了。
<br/>　　“啊！”
<br/>　　芮冷玉大喝一声，剑锋迸发出了数道光华，组成了剑阵，将叶少阳封锁了起来。
<br/>　　倒不是叶少阳真的那么菜，而是战斗双方的心态不一样：芮冷玉被邪性蒙蔽，把叶少阳当成了追杀自己的人，一个劲地要置他于死地，更不要说她刚觉醒，体内无尽的煞气，强烈地想要释放出去，自然而然选中了叶少阳。
<br/>　　叶少阳却不想伤她，心中有顾虑，再加上被芮冷玉弄伤。两人在气势上首先就不对等，这才造成了叶少阳的落败。
<br/>　　在用剑阵困住叶少阳之后，芮冷玉没有任何停留，直接挥剑对着他面门刺了过来。
<br/>　　叶少阳无法反抗，眼看着长剑贴近自己面门，突然数道红光袭来，打在芮冷玉胸前，将她震飞出去，紧接着一个巨大的身影将她扑倒在地上，张口就要咬下去。
<br/>　　“小九！”
<br/>　　关键时刻，是小九冲了过来，救了叶少阳一命。叶少阳生怕芮冷玉被她杀了，情节中叫了一声。
<br/>　　小九听见叶少阳的呼唤，咬下去的动作迟钝了一下，芮冷玉猛然出剑，一剑刺进了小九的喉咙！
<br/>　　鲜血如注，从伤口汩汩流出。
<br/>　　松纹古定剑是九段光的法器，与妖血一旦相遇，立刻焕发出了强大的灵力，在小九体内游荡，撕扯着她体内的妖气。
<br/>　　一切发生在电光石火之间，就这么……发生了。
<br/>　　“小九！”叶少阳扑了上去。
<br/>　　小九抬起爪子，想要拍下去，如果这一下拍下去，立刻能杀了芮冷玉，但是她听见叶少阳的喊声，再一次犹豫了，然后，她一口咬住了芮冷玉的脖子。
<br/>　　叶少阳猛然站住了。
<br/>　　林三生等人也站住了，停止了打斗，转头朝这边看过来，一脸的不可置信。
<br/>　　黑白无常等人也停止了打斗，一个个都朝这边看了过来。
<br/>　　女魃的目标本就不是道风，根本不想跟这个强者纠缠，早就飞上了山崖，去杀黎山老母等人。摇光仙子和几个空界的强者跟了上去，将她挡住，女魃看没机会，立刻退走。
<br/>　　摇光仙子留下几个强者保护伤员。自己下来独斗道风，她与道风的实力也在伯仲之间，在芮冷玉和小九出事之时，十娘带着一帮人冲了进来，告诉摇光仙子几大门派的援兵到了，太阴山残余的人马已经退走，援军接住了尸族大军，正在激战，她带着几大门派的宗师前来保驾。
<br/>　　这几大宗师是刚刚赶到，一个个都还生龙活虎的，看了一下形势，立刻选择围攻道风，将他困在中间。摇光仙子却是轻松下来，回到山顶去保护伤员们，一边俯瞰山谷中的战局。
<br/>　　瓜瓜眼看着小九叼住了芮冷玉的脖子，急得大叫：“小九，不要伤害主母！”
<br/>　　所有人都以为小九被芮冷玉重伤，情急中要杀死她，然而，小九叼起芮冷玉之后，却朝着叶少阳飞了过去。
<br/>　　“少阳，快困住她的元神……我快坚持不住了。”小九用神识对他说道。
<br/>　　“你……”叶少阳二话没说，一口气画了三张定魂符，贴在芮冷玉身上。
<br/>　　小九这才吐出芮冷玉，浑身颤抖，幻化出了人身，倒在地上。
<br/>　　“小九！！”叶少阳上前抱住她，看了一眼她的脖子，芮冷玉那一剑，几乎造成了贯穿伤，滚热的血不断流出，落在叶少阳手上。
<br/>　　这一剑，怎么会伤成这样？
<br/>　　叶少阳其实也明白，松纹古定剑是九段光法器，最克制一些邪物，小九虽然是妖王，但是被刺中要害，身体内部又被宝剑的灵力进行了一番剧烈的冲击，现在的她，已经虚弱至极，只剩下一口气。
<br/>　　“少阳，我不辱使命……”
<br/>　　“不要说了，什么都不要说。”叶少阳拿出阴阳镜，想要把她收进去养伤。
<br/>　　小九想说什么，突然瞳孔一缩，猛然推开了他，挺身而上。
<br/>　　一只手，打在了她的天灵盖上。
<br/>　　“啊！”
<br/>　　小九惨叫一声。这只手却没有丝毫怜悯，反手抓着她的脑袋，直接丢了出去，叶少阳这才反应过来，一把抱住小九，结果感到到一股庞然大力，拖着他跟小九一起以失重地姿势横飞，最后撞在一个坚硬的东西上，叶少阳眼前一黑，差点昏过去，定睛看去。两人是撞在那尊宝塔的顶端，正在往下坠落。
<br/>　　小九撞的比他要重的多，但还是凭着近乎本能的反应，一只手抓住了他。
<br/>　　一道黑影，狂掠过来，是李浩然！
<br/>　　刚才那一掌就是他击出的，目的是杀叶少阳，结果被小九挡了一下，在丢起他们、撞在宝塔上之后，他再度飞来，从腰间拔下长剑，挺剑刺来。
<br/>　　他跟小九没仇，他要杀的是叶少阳。
<br/>　　小九几乎濒死，急切之中，毫不犹豫地吐出了内丹，然后……用妖力融化了内丹。
<br/>　　“什么！！”
<br/>　　摇光仙子惊叫起来。
<br/>　　黎山老母等人本来坐着静静调息，俯瞰见这一幕，也是惊的站了起来，一个个瞪大眼睛，望着小九。
<br/>　　小九几番身受重伤，但是只要内丹完好，就算是肉身完全损毁，也能一点点修复，内丹，是一切妖精的根本，一身修为也都蕴藏在内丹之中，一旦损毁，其中的修为会在瞬间释放出强大的能量，然而……内丹是不可再生的，一旦毁了，就再也无法修复。
