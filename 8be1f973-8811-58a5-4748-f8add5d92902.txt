　　叶尘自然察觉到两人跟了过来，不过他也懒得和两只蝼蚁计较，继续慢悠悠的朝教堂的最深处走去。
<br/>　　忽然，
<br/>　　绕过一条小路后，前方出现了一条甬道，
<br/>　　唰！唰！
<br/>　　两道黑影在空中闪过，陡然出现在叶尘跟前，拦住了他的去路。
<br/>　　只见这两人，身穿灰色的教士长袍，身上透着一股阴森的气息，
<br/>　　“前方乃教廷禁地，闲杂人等，不得入内！”
<br/>　　其中一名灰袍男子，阴测测开口道，说的是英语，不过却透着一股中世纪的语气风格。
<br/>　　叶尘也懒得和他废话，直接开门见山道：
<br/>　　“我要见你们的教皇！”
<br/>　　那两名灰袍男子对视了一眼，另外一人开口道：
<br/>　　“教皇大人每个月月初都会在圣彼得大殿举行礼拜仪式，你若是想见他老人家，下个月月初，直接去圣彼得大殿即可！”
<br/>　　说完，两人便要转身离开。
<br/>　　叶尘一脸淡然，再次开口，
<br/>　　“我要见的，是柯莱特那个老家伙！”
<br/>　　“放肆！”
<br/>　　那两名灰袍人，几乎同时一声暴喝，身上的气势猛然变得强大暴虐，似乎随时都能将叶尘吞噬。
<br/>　　就连一直远远跟在后面的，王硕和韩颖玉两人，也被两人身上的气势所惊，不由得纷纷脸色大变，
<br/>　　“这两个人，好强的气势！恐怕至少也是圣境之上的高手吧？”
<br/>　　“早就听说，圣彼得大教堂，乃是西方教廷的总部，内部高手如云，看来传言果然不虚啊！”
<br/>　　两人都是出自华夏最顶级的家族，知道的东西，自然比普通人要多很多。
<br/>　　“哼！这小子还真是不知道天高地厚，竟然敢直呼那位老教皇的名讳，待会有他受的！”
<br/>　　王硕反应过来之后，脸上顿时露出一抹快意，似乎已经看到眼前这个神秘少年，被西方教廷轰走的场景。
<br/>　　“小子！我不知道你是什么人，不过如果你不想死的话，就速速离去，否则可就要休怪我们不客气了！”
<br/>　　两名灰袍人，袖袍无风自鼓，身上的气势也攀升到极点，想要用这种方式，让叶尘知难而退。
<br/>　　然而，
<br/>　　他们这点手段，在叶尘眼中，实在太微不足道了。
<br/>　　面对两人的气势压迫，叶尘一脸平静，目光直接越过两人，望向后方的大殿深处，轻轻吸了一口气之后，暴然开口，
<br/>　　“柯莱特老教皇！你给我，滚出来！滚出来！出来！来！”
<br/>　　声音犹如平地一道巨雷，浩浩荡荡的朝那深处传递而去，久久回荡不息。
<br/>　　轰隆隆！
<br/>　　后方的那座雄伟大殿，都被震得簌簌颤抖，似乎随时都要倒塌一般。
<br/>　　蹬蹬蹬！
<br/>　　那两名灰袍人，直接便被震飞了出去，并且狂喷了一口鲜血，生死不明。
<br/>　　而后方，原本还等着看叶尘笑话的王硕，也顿时脸色惨白，简直无法相信眼前所发生的一切，口中忍不住失声喃喃道：
<br/>　　“他怎么会……怎么会这么强？”
<br/>　　王硕丝毫不会怀疑，倘若方才这个神秘少年，动用真正的实力，他此刻恐怕早就死翘翘了。
<br/>　　……
<br/>　　与此同时，
<br/>　　坐在教皇位置上的老教皇，听到这道声音，也霍然站起身来，老脸瞬间变的惨白无比，
<br/>　　“是他！他来了！难道他已经知道了我的计划？”
<br/>　　下方的那位红衣大主教，同样脸色大变，
<br/>　　人的名，树的影，当初叶狂仙一脚踩下米国的视频，他看过不下十次，连百万吨级的核弹，都奈何不了此人，此人的实力恐怕已经远远超过老教皇！
<br/>　　哪怕是顷他们整个西方教廷之力，都未必能拦得住此人。
<br/>　　“老师，我们这该如何是好？要不，还是赶紧逃命吧！”
<br/>　　老教皇颓然长叹了一声，
<br/>　　“没想到他竟然来的这么快！如今想要逃跑，恐怕已经来不及了！”
<br/>　　……
<br/>　　另外一边，
<br/>　　叶尘平地一声雷，顿时惊动了整个教廷中的高手。
<br/>　　唰唰唰！
<br/>　　一眨眼的功夫，一道道浑厚的气息，自四面八方包围而来。
<br/>　　王硕和韩颖玉两人，脸色顿时变得越发难看。
<br/>　　“不好！教廷的高手似乎全部被吸引了过来，这下连我们恐怕也走不掉了！”
<br/>　　他们能清楚的感觉到，这些第一时间赶过来的教廷强者，个个实力都远远超过他们，如果想要杀他们的话，简直比捏死一只蚂蚁还要简单！
<br/>　　“哪里来的小子！竟然敢在我教廷放肆，找死！”
<br/>　　其中一名穿着黑色主教长袍的老者，锁定了叶尘的气息之后，直接大喝一声，便朝他扑杀了过来。
<br/>　　王硕双眸一凝，忍不住失声道：
<br/>　　“这人的资料我见过，好像是黑衣主教哈达威，据说此人的实力，在黑衣主教中也是顶级，就算比红衣大主教，也差不了太多，恐怕至少也是神级以上的实力！”
<br/>　　韩颖玉听到这话，不由得望向那道瘦削的少年身影，心中暗叹，
<br/>　　“完了！这小家伙不知道天高地厚，竟然孤身一人，独闯西方教廷，这简直就是在作死啊！”
<br/>　　然而，
<br/>　　下一刻的所发生的一切，顿时让她目瞪口呆。
<br/>　　面对一位神级强者的一击，那名神秘少年竟然没有挪动分毫，似乎只是随手一弹，
<br/>　　“嗖！”
<br/>　　一道紫色剑芒自他的指尖发出，直奔那黑衣主教哈达威爆射而去。
<br/>　　“彭！”
<br/>　　哈达威人在空中，连惨叫都没有来得及发出，脑袋竟然被那道紫色剑芒，活生生打爆，化成了一团血雾！
<br/>　　“什么！”
<br/>　　所有人看到这一幕，顿时尽皆骇然色变。
<br/>　　那些原本正要冲上来的教廷高手们，更是尽皆暴退，一时间不敢再轻举妄动。
<br/>　　“阁下到底是什么人？为何要擅闯我教廷？”
<br/>　　最终，一名红衣大主教站了出来，两眼死死盯着叶尘，缓缓开口道。
<br/>　　叶尘淡淡瞥了那人一眼，
<br/>　　“我是什么人，你们这群废物还不配知道，我再说最后一次，让柯莱特出来见我，否则我今日，便踏平你们西方教廷！”
<br/>　　轰！
<br/>　　叶尘这话一出口，众人顿时炸了。
<br/>　　西方教廷统治欧洲大陆一千余年，在鼎盛之时，几乎统治了全球，连东方诸国都沦为了它的殖民地。
<br/>　　这千年以来，还从未有人，胆敢当面挑衅教廷的威严。
<br/>　　“你找死！”
<br/>　　那名红衣大主教顿时暴怒，
<br/>　　“大家一起出手，宰了这个狂徒！”
