        冷逸峰的目的已经显而易见。
<br/>
<br/>        傅夜沉深沉地凝望着身边的童筱颖，看着童筱颖脸上的笑容，内心也很复杂。
<br/>
<br/>        如果再去探寻真相，万一得到的结果是他无法接受的怎么办？
<br/>
<br/>        冷逸峰看出了傅夜沉的心思，又看了一眼童筱颖，抬起手来拍了拍傅夜沉的肩膀，接着说道：“我已经确定了，她确实是你的女人，不是我的。这下，你可以放心去查了吗？”
<br/>
<br/>        “你要我从何查起？”傅夜沉反问。
<br/>
<br/>        “就查，为什么筱颖身上会有婴宁的影子？”这是冷逸峰心底的疑惑。
<br/>
<br/>        傅夜沉犹豫了许久后，才勉强地点头答应了这件事情。
<br/>
<br/>        两个大男人在一旁窃窃私语，这边季雅珊也没闲着跟童筱颖重新认识。
<br/>
<br/>        傅夜沉警告过她，不希望她再跟童筱颖交朋友。
<br/>
<br/>        所以，季雅珊也只是跟童筱颖做了一下很简单的自我介绍，让她知道，这世上还有她这么一个人，仅此而已。
<br/>
<br/>        而童筱颖对季雅珊，感觉上像是第一次见面，但她却有一种认识了很久的错觉。
<br/>
<br/>        比起对于苏漫雪的厌恶感，童筱颖对面前的这个女人，不厌恶，反倒有点亲切之情。
<br/>
<br/>        傅夜沉和冷逸峰聊完后，也到了拜别的时候。
<br/>
<br/>        当傅夜沉搂着童筱颖先行离开时，冷逸峰突然转过身去，对童筱颖的背影大喊了一声：“夏玄烟！”
<br/>
<br/>        这么久以来，这是他第一次呼唤她的真名。
<br/>
<br/>        童筱颖听到这个名字的时候，本能地回过头去四下张望了一下。
<br/>
<br/>        但很快的，她又回眸看向身边的傅夜沉，眼神里对他透着爱慕的笑意。
<br/>
<br/>        季雅珊惊愕地看着这一幕，同时偏头看向一旁的冷逸峰时，只见冷逸峰的脸色僵硬，黑色的瞳孔紧缩，就像是面临着什么难以置信的事情，让他无法接受。
<br/>
<br/>        她以为这么久了，他已经忘掉婴宁了，但是……
<br/>
<br/>        “阿十，我们回去吧！”季雅珊黯然神伤地劝道。
<br/>
<br/>        而此刻，冷逸峰却在不经意间地握紧了双拳。
<br/>
<br/>        他的心思，她猜不透，所以，她只能在这个时候，抬起手来捧住他紧握成拳的手，告诉他，无论发生什么事情，她都会一直陪在他的身边。
<br/>
<br/>        玩了一天，童筱颖玩累了，回到私宅后，洗完澡，早早就睡下了。
<br/>
<br/>        傅夜沉在童筱颖睡下后，独自去了书房，给连华生打了通电话。
<br/>
<br/>        那个秦烈靠近筱颖，是因为筱颖长得像夏玄烟，而事到如今，就连十爷冷逸峰都说，筱颖的行为举止和夏玄烟几乎一模一样，那么在筱颖的身上就真的存在很大的问题。
<br/>
<br/>        连华生听完傅夜沉的阐述后，给傅夜沉说出了自己的意见：“你要不抽个空，带她来我这里。我找个心理医生，给她催眠试试。”
<br/>
<br/>        “为什么要催眠？”傅夜沉难以理解。
<br/>
<br/>        连华生条条是道地解释：“她既然经常做梦，梦到的又是过去发生在童筱颖身上的事情，并且，她的潜意识里还听命于另外一个她自己都不知道的人。如果排除她被人替代的可能性以及她有双重人格的病症，那么剩下的这种迹象表明的是，她一直在做梦。简而言之，就是她现在处于一种被催眠的状态。”
<br/>
<br/>        “你的意思是，要心理医生将她从催眠中唤醒？”傅夜沉微微蹙起了眉头。
<br/>
<br/>        连华生心里也很没有底地回答：“自从童筱颖回来后，关于她身上的事情，一直都是我们在揣测以及摩挲，我们从来就没有得到一个真正的真相不是吗？”
<br/>
<br/>        “你有没有问过你父亲？”傅夜沉淡定地问。
<br/>
<br/>        连华生不禁唉声叹气起来：“我可是明里暗里，直接或间接地询问过我父亲关于童筱颖的事情。可结果，我父亲是那种明里暗里，直接或者间接都不想告诉你真相的老顽童！”
<br/>
<br/>        “如果催眠筱颖，有是什么结果？”傅夜沉眉头深锁。
<br/>
<br/>        连华生犹豫了一会儿，才老实交代：“如果，她现在正处于被催眠的状态，强行将她唤醒后，她确实会变成原来的样子。比如说，她原来如果是昏迷状态，她就会进入昏迷状态，她原来如果是清醒的，那么她就会是清醒的。如果，她没有被催眠，而是因为‘解药’的副作用产生的两个人格。说不定，催眠后，她能找回原来的那个自我。”
<br/>
<br/>        “你的意思是说，催眠筱颖，无法保证筱颖百分百安全？”
<br/>
<br/>        “没错。”
<br/>
<br/>        “那我放弃！”傅夜沉毫不犹豫地说。
<br/>
<br/>        连华生也不打算说服傅夜沉。
<br/>
<br/>        因为他知道，童筱颖好不容易活着回到傅夜沉的身边，让傅夜沉再次放手的可能性几乎为零。
