        &lt;h3&gt;第三卷第二百八十九章&lt;/h3&gt;
<br/>
<br/>        易倾秋接替了陈子霖的工作，江俞轩已经离职，陈子霖这几天都忙着家事，公司的大小事情她都得来处理。她本来是李秀推荐的，现在的身份竟然比李秀还要高一个职位，这让她见了李秀有点难为情。
<br/>
<br/>        “师傅，倾秋有什么事情你得提醒我，要不是你我哪有今天啊。”易倾秋找到李秀说。
<br/>
<br/>        “你有这个能力，陈总任人才是用，别气馁，好好带领大家一起干，我永远都支持你。”李秀看着易倾秋，易倾秋胜在情商高，而且忠诚，这两点大概就是陈子昂看上她的原因吧。
<br/>
<br/>        “谢谢师傅。”易倾秋心存感激，一个新人到公司如果没有老人带着，其实是真的很难出头露面的。
<br/>
<br/>        “别师傅师傅的叫着，你这样叫我还怪不好意思的。”李秀笑着说“还有，我怕九儿柠檬精上身，到时候我吃不了兜着走。”
<br/>
<br/>        “哈哈哈，那我就九儿师娘她就不会柠檬精上身了。”易倾秋开心的笑着。
<br/>
<br/>        “九儿师娘，你能不能来我办公室一趟？”易倾秋直接就拨打了谭九儿的内线电话，叫来了谭九儿。
<br/>
<br/>        “倾秋，你个小怪物，什么时候如此调皮了？”谭九儿推开易倾秋办公室的们，发现李秀也在。
<br/>
<br/>        “师娘在上，倾秋这厢有礼了。”易倾秋笑着说。
<br/>
<br/>        “你什么时候有师傅了？师傅是谁？”谭九儿看了一样李秀。
<br/>
<br/>        “我也不晓得什么时候有师傅什么时候有师娘，好像从我进了这个公司的时候就有了？”易倾秋调皮的笑着。
<br/>
<br/>        “你这越来越调皮了，说叫我们来干什么？”谭九儿看见李秀也在，知道易倾秋绝对不是认师傅师娘这么简单。
<br/>
<br/>        “是这样的，最近新城的项目已经启动了，我们面临着三家强大的对手，这三家有夏颜作为后盾，我想知道你们的想法，是进还是退。”易倾秋不再开玩笑，直接投入到工作上。
<br/>
<br/>        “陈总怎么说？”谭九儿问，陈子昂最近在西城，是不是应该问问她的意见？
<br/>
<br/>        “西城是总部的一个分支，但是也有独立运营的能力，如果事事都让陈总来做决定，那么我们这些中高层就是一个摆设，她要我们来是为她分担的，不是来添乱的，江总走了，我想我们更要为她尽一份力，如她所说，我们要做到离开任何人公司都能正常运转，不但能运转还要运转的更好。”
<br/>
<br/>        李秀和谭九儿看了一眼，点点头。
<br/>
<br/>        “我们要进，但是也要做好退的准备，夏颜针对我们公司不是一天两天了，在她手中的项目我们基本很难拿到，如果你说的那三家要是联手的话我们没有胜算。”李秀说。
<br/>
<br/>        “我们可以声东击西，莲余的项目也已经启动了，我们表面上去竞争新城的项目，实际上去拿莲余的项目，让他们放松警惕。”谭九儿微笑着说。
<br/>
<br/>        “好，那就这样决定了，九儿你负责莲余的项目，李秀负责新城的项目。我们就来个声东击西，一举拿下莲余的项目。”易倾秋笑眯眯的说。
<br/>
<br/>        易倾秋下午开了一个公司动员会，动员大家一定要努力，将重心放在新城项目上：“这个是我上任的第一个项目，倾秋拜托大家和我一起努力，尽心尽力拿到这个项目，倾秋给大家鞠躬了。倾秋虽然年龄小，请相信倾秋会带着西城公司走向更加辉煌的地方，我们要让总公司看到我们西城分公司不但是总部的分支更是一个独立的个体，离开总部我们照样能够做得更好。大家有没有信心？”
<br/>
<br/>        “有信心！”这一段时间西城分公司经过了刘清影等人的叛变，江俞轩的离职，每个人心里都像是压着一块石头，经过年轻的易倾秋的一番鼓舞，大家也热血沸腾，齐声呐喊。
<br/>
<br/>        是啊，他们都是公司的一部分，他们是公司的一个整体，离开谁他们都能正常运转。
<br/>
<br/>        .......
<br/>
<br/>        陈天天到了冯睿的家里，冯睿说她身体不好，有些感冒，陈天天来陪她。她们本来也离得不远，陈天天将自己的笔记本摔坏了之后并没有重新买笔记本，她到了冯睿的家里，用的是冯睿的电脑。
<br/>
<br/>        冯睿的电脑里有陈子建的资料，但是冯睿将有陈子建资料的那个盘符加密了。
<br/>
<br/>        陈天天原本不想动冯睿的电脑，看着有加密的盘符，她还是有点好奇：“妈，这个盘里是什么啊，你还加密了？”
<br/>
<br/>        “也没有什么，就是一些照片啊什么的。”冯睿有点支支吾吾。
<br/>
<br/>        “啊，什么照片啊，你还加密？”陈天天好奇之心不减。
<br/>
<br/>        “我和你爸年轻时候的照片，唉，我都不想看了，看了我就伤心。”冯睿唉声叹气。
<br/>
<br/>        “妈，你和我爸到底是什么原因离婚的？”陈天天不知道冯睿和陈子建离婚的原因。
<br/>
<br/>        “因为陈子昂啊，有一次陈子昂带着江俞轩来我们家吃饭，你爸给我的一个手镯我送给了陈子昂的婆婆，就是张函的母亲，想着张家能不能对陈子昂好一点，但是他们都不领我的情，江俞轩眼巴巴的看着你爸爸打我，他都不拦着。他们还在旁边阴阳怪气的说我的坏话，天天你说妈妈的心里苦不苦？我可是为了陈家好啊。”冯睿的眼泪下来了。
<br/>
<br/>        “他们为什么会这么对你？”陈天天不理解。
<br/>
<br/>        “你爸娶我的时候花了他们家里的一些积蓄，他们就排斥我，上班工作之后我也尽心尽力的帮衬着家里，可是无论如何都得不到他们的心，再说，那个时候家里都穷啊，陈子昂没有考上大学和我有什么关系呢？他们就都认为是你爸爸娶了我让家里变穷的。”冯睿是一把辛酸泪，说得天花乱坠。
<br/>
<br/>        “原来如此，那他们也太不是人了，这都过去了多少年了。”
<br/>
<br/>        “天天，他们连外人都不如，你看夏颜，还经常隔三差五的来看看我，给我拿东拿西，你看陈家人有一个人来看我吗？再怎么说，我也给他们生了俩个孩子吧？天天，妈妈现在只有你了。”冯睿泪眼婆娑。
<br/>
<br/>        “妈，你别哭了，我以后会对你好的。”陈天天看不得人哭，尤其还是她的亲生母亲，血缘是奇怪的东西，无论如何，你都斩不断和它的联系。
<br/>
<br/>        易倾秋开完会，给陈天天打电话：“天天，你在哪里？”
<br/>
<br/>        陈天天看见易倾秋的电话，迟疑了一下，她现在的心思已经慢慢的靠近冯睿了，对陈子昂产生有了一点点的隔阂，所以看见易倾秋的电话犹豫了一下。
<br/>
<br/>        “我在我妈妈家。”陈天天看了一眼冯睿。
<br/>
<br/>        “你妈妈还好吧？你有没有时间出来我们一起吃饭？”易倾秋试探着问。
<br/>
<br/>        “我妈妈感冒了，我得照顾她。”
<br/>
<br/>        “那好吧，等你有时间了再说。再见！”易倾秋叹了口气。
<br/>
<br/>        “谁给你打电话？”冯睿问陈天天。
<br/>
<br/>        “易倾秋，陈子昂公司最近新晋的副总，七叔现在是公司的老总了。”
<br/>
<br/>        “你们俩人关系很好？陈子昂很相信易倾秋？”
<br/>
<br/>        “关系还可以，陈子昂好像很喜欢易倾秋，一回来就将易倾秋提拔了。”
<br/>
<br/>        “她约你干什么？”冯睿不经意的问。
<br/>
<br/>        “约我吃饭呢，我们经常一起吃饭。”
<br/>
<br/>        “那以后就叫她来家里吃啊，我给你们做饭吃，天天去外面吃饭怎么行呢。”冯睿其实是做得一手好饭菜的。
<br/>
<br/>        “真的？”陈天天看着冯睿。
<br/>
<br/>        “傻孩子，你喜欢的人妈妈都会喜欢的。叫过来吧。”
<br/>
<br/>        “啊，那谢谢妈，我给倾秋说。”陈天天满心欢喜，其实她也很想让易倾秋知道她的妈妈对她其实是很好的。
