　　如果仅仅是这样也就算了，虽然少年给他安排的职位是个闲职，福利算不上好，但最起码还算安全。
<br/>　　可就在不久前，他上了自己顶头上司的老婆，事迹败漏，被发配到搜索队里任职。
<br/>　　那可是每天都把脑袋别再腰上过日子的地方，他心中自然有万般不愿，但苦于没钱没势的，他也只能认命。
<br/>　　“整天给我惹事，朱厅的霉头你也敢触，要不是我从中活动，你早就死了！”
<br/>　　“就搜索队，爱去不去，要么我就把你交给朱厅发落。”
<br/>　　“行了行了，你先去呆一段时间，等朱厅的气消了，我再给你调动就是了。”
<br/>　　想到少年当时的语气，孙强心中有些恼怒，这个小王八羔子，他的命都是自己给救的呢！
<br/>　　那个时候，他端茶倒水的，怎么不用这样嚣张的语气和自己说话？
<br/>　　心中有些忿忿，但这些话，他也就只敢在心中想一想了，纵然心中牢骚不断，他也只能冒着风险带队外出。
<br/>　　有几年没有面对那些可怕的丧尸了，这几次出来搜索物资，都是心惊胆战的，不过幸好，他们队伍中的幸存者都是老手，一直也没有出现什么意外。
<br/>　　“孙队长，前面有个车子，好像是被人故意给藏在那里的。”就在孙强分神的时候，前方探路的斥候跑了回来，向孙强禀报道。
<br/>　　听到这个消息，孙强眼睛一亮，像是寻找到了猎物一般，兴奋了起来。
<br/>　　像他们这种官方的搜索队，虽然任务只是外出搜索无主资源，但打家劫舍的事情，同样也没少干。
<br/>　　只要将胸前代表官方的徽章给摘下来，他们摇身一变，立即就会成为一伙训练有素的土匪。
<br/>　　搜索队在外找到的资源，百分之三十都是归个人所有，没人会拒绝多捞一笔外快。
<br/>　　而搜索队猎取到的晶核，也全部归为他们的队员，基地并没有要求强制上交。
<br/>　　孙强不缺晶核，在闲职上干了那么久的时间，他也攒下了一些晶核，虽然不多，但也足够他在基地中的日常花销了。
<br/>　　但他缺乏安全感。
<br/>　　或者说，他极度怕死，故此每次外出时，他都会找各种借口，来摆脱那些危险的行动。
<br/>　　这样的行为，自然会引起队伍中一些人的不满，虽然他是小队的队长，但也害怕手底下的那些队员突然反叛。
<br/>　　故此，他要想一直保持这样现状的话，唯一的办法就是让手底下的队员，从他这里得到从别的队伍中的更多的东西。
<br/>　　就比如，像这种偶遇的幸存者，如果将其成功猎杀的话，他都会将对方身上的晶核给大家平分下去，而他分文不取。
<br/>　　这样一来，大家就对他十分爱戴了，甚至在他的刻意维护之下，队伍中成员之间的关系，都要比别的队伍好上许多。
<br/>　　“嘿嘿，老大，快递来了呢，我们谁去拿一下？”孙强的身边，一个矮小的男子开口，脸上的笑容显得他有些猥琐。
<br/>　　虽然长相不招人喜欢，但他可是队伍中最重要的火力手，一个同为四阶的火系异能者。
<br/>　　但火系异能可不像速度异能，因为火系异能强大的杀伤力和破坏力，无疑是一支队伍中最核心的地方。
<br/>　　“不要大意，敌人的人数和实力都不明确，我们先埋伏起来，先看看情况再做决定。”孙强沉吟了一下，开口吩咐道，随后带着众人隐藏在四周。
<br/>　　大概等待了半小时的功夫，一道带着鬼脸面具的身影从远处晃悠了过来，让众人惊奇的是，这个家伙竟然毫无遮掩的走在马路上，周围的丧尸就好像没有发现他一般。
<br/>　　从对方干净整洁的衣物上看来，这明显不是一头丧尸，其身上也没有那些令人作呕的烂肉。
<br/>　　难道是，特殊异能者，或者是基地之前没有记录过的高阶丧尸？
<br/>　　虽然大部分的高阶丧尸为了保持自己的战斗力，都没往人类的外貌特征方面进化，但像是极少数的特殊丧尸，譬如精神类的高阶丧尸，依旧选择了人类的形态。
<br/>　　基本上保持人类形态的高阶丧尸，到了八阶之后，外边就和正常人类没什么区别了，只是在一些很细微的地方，才能辨认出它们丧尸的身份。
<br/>　　但是，这个小镇不太可能出现这样高等阶的丧尸吧，而且基地官方，也没有任何记录。
<br/>　　心中带着疑惑，孙强看了一眼同样惊异的众人，打了一个继续潜伏的手势，这个家伙有些诡异，他们并不想与之交手。
<br/>　　从小镇中出来的人，正是刘哔，因为脸上带着鬼脸面具的缘故，他也就没有隐藏自己的行踪，反正以他的实力，一些普通的幸存者小队也不会是他的对手。
<br/>　　顺着有些干燥的地面，刘哔回到自己藏车处的附近，突然，脚步放慢了下来，他在附近，感应到几股陌生的气息。
<br/>　　这几道气息的实力都不算高，其中最为强大的，也不过只是筑基初期的样子，此时这几个人就隐藏在他的车子附近，衣服不怀好意的样子。
<br/>　　之所以认为是人，是因为像丧尸或者爬行者那样的生物，没有到达一定等级之前，是不会隐藏起来进行埋伏的。
<br/>　　如果进行埋伏的是丧尸，那这几股气息不可能这样弱小，同样也不会这般轻易就能让他探查出来。
<br/>　　脸上浮现一抹不屑的笑容，刘哔装作若无其事的样子，继续往停车的地方行走着。
<br/>　　二十米……十米……五米……
<br/>　　距离越来越近，似乎是感应到紧张的气氛，空气逐渐开始焦灼了起来，孙强的额头，不断向下滑落着冷汗。
<br/>　　他此时有些后悔自己先前做下的决定了，这个戴着鬼脸面具的男人，明显不是一副好惹的样子，光靠他们这几个人，很难将他留在这里。
<br/>　　周围的队友都没敢有任何动作，他们自然也发现了其中不对的地方，纷纷向孙强传递着眼神，询问下一步的行动。
<br/>　　苦笑一声，孙强轻轻摇了摇头，安全起见，他打算放弃这次的行动了。
