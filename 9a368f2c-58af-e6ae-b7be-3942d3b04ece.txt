　　约莫半刻钟后，在附近逛了一圈，慕清霄找到一家露天茶摊，找了个位置坐下，道：“老板，上一壶好茶，一盘糕点。”
<br/>　　“好嘞，客官您稍等。”
<br/>　　很快，摊主就端着一壶热茶与一盘糕点走了过来，热情微笑道：“公子是出来游玩吧，琴川的景色肯定不会让您失望。”
<br/>　　“阳春三月，江南景色确实颇为不错。”端起茶杯，轻轻抿了一口，慕清霄语气平淡的说道。
<br/>　　“公子真是性情中人呐。”
<br/>　　摊主很热情，他也不着急，边品茶边闲聊。
<br/>　　半刻钟下来，一盘糕点吃完，两杯茶水见底，慕清霄站起身来，取出一枚银子放在桌上，转身离去。
<br/>　　摊主上前，望着桌上的一枚银子，刚想看口说钱给多了，但客人早已不见踪影。
<br/>　　离开茶摊后，慕清霄就顺着运河，向琴川城西侧走去，他来琴川可不光是为了看风景，而是另有深意。
<br/>　　十多分钟后，站在一座小宅院前，慕清霄陷入沉思，小宅院的左侧则是一座府邸，门口的匾额上有着‘方府’两字。
<br/>　　望着小宅院紧闭的大门，慕清霄神识释放而出，将小宅院里里外外查探了一遍，其中丝毫没有人住的迹象。
<br/>　　“看来，欧阳少恭并未回来，只有在等上几年了，先去碧山吧。”
<br/>　　微微摇头，叹息一声，说罢，周围空间细微波动一下，身影就自原地消失不见。
<br/>　　原本，慕清霄的目标就是自闲山庄的另一半玉衡，中途想到，欧阳少恭在近期内可能会出现在琴川。
<br/>　　于是，慕清霄便顺路来到此处，遇到方家姐弟倒并不意外，没有遇到欧阳少恭，多少令他有些失望。
<br/>　　...
<br/>　　...
<br/>　　两日后的清晨，旭日东升。
<br/>　　此时，映入慕清霄眼中的是一座山野村庄，村庄占地面积并不大，也就几十亩地的面的，其中估摸着也就住的下几十口人。
<br/>　　村庄完全偏离了附近的城镇，与世隔绝，外围则由一米多高的竹栏给围了起来，完全没有安全感可言。
<br/>　　这种高度，最多也就防御一些普通野兽，如果真有妖魔鬼怪作祟，全村的人恐怕无一人能够幸免。
<br/>　　所以，妖魔鬼怪纯属无稽之谈，否则，别说派人去城镇内报信，能有活人都尚未可知。
<br/>　　说来也怪，翠萍村坐落在这碧山山脚下，山中不可能没有野兽，可别说是妖魔鬼怪，慕清霄一路走来，就连只普通的野兽都不曾看到。
<br/>　　但村内的青年却频频怪异的死去，其中事有蹊跷。
<br/>　　青年们的死去，令得村内的居民都惶恐不安，甚至有再度搬迁的意思。
<br/>　　最终，由于村内的老人不愿放弃村子离去，年迈的老村长只好向附近城镇里的人求救。
<br/>　　城主不愿摊上此事，但他多少有些门道，只好以书信的方式，向传说中的修真门派求援。
<br/>　　望着眼前的小村庄，慕清霄不禁抬头看了一眼碧山顶端。
<br/>　　哪怕现在是白日青天，烈阳高照，这里都显得死气沉沉，周围连鸟叫声都没有，气氛怪异至极。
<br/>　　收回目光，慕清霄就迈动脚步，推开村口的竹栏门，向村庄内部走去。
<br/>　　没走多久，视线中就出现一位老农夫，年龄六十岁左右，正拿着锄头在一块小田中耕种，似乎是在种植蔬果幼苗。
<br/>　　慕清霄看了一眼老者走上前去，轻声道：“老人家，请问这里可是翠萍村？”
<br/>　　话语刚落，老者就抬起头来，露出一张满脸皱纹的脸颊，浑浊的眼眸透着迷茫。
<br/>　　愣神片刻，老者放下手中的锄头，道：“娃娃，你不是村里的人吧？”
<br/>　　“对，在下是村外来的。”
<br/>　　老者虽上了年纪，但并不妨碍两人沟通，只是听到娃娃两字，慕清霄眼角忍不住抽了抽，感觉有些怪异。
<br/>　　“没有到会有人来这种荒凉的地方，不过，最近村里不太平啊，老头子劝你快些离开吧。”
<br/>　　似乎想到了什么，老者眼中闪过一丝恐惧，开始劝说慕清霄离去。
<br/>　　“老人家，在下是城主请来，帮你们解决村庄内的怪事，还请将事情的来龙去脉跟我说一说。”
<br/>　　“你......你是仙人？”
<br/>　　听到慕清霄的话语，老者一激动，城主请来的可不就是传说中的仙人么，甩下手中的锄头，就要给他跪下磕头。
<br/>　　此举令慕清霄有些无奈，连忙将他搀扶了起来，道：“仙人不敢当，只是有些小道行的普通人罢了，老人家不必如此。”
<br/>　　“仙人，我们半截身子已经入土，您可要救救村里那些可怜的孩子啊。”想到村子里的青年一个个死去，老者脸上就露出痛苦之色。
<br/>　　“在下来此就是为了解决此事，还请老人家将村内发生的事情叙述一遍。”
<br/>　　“事情是这样的，我们翠萍山一向相安无事，不知为何，在一个月前......”
<br/>　　经过老者的叙述，慕清霄得知，在这一个月中，村内共有三名青年男子死去，身上没有任何伤势，莫名其妙的就死了。
<br/>　　而且，在死之前，这些男子都有一个特征，浑浑噩噩，甚至半夜在村内游荡的。
<br/>　　先前，村名们都没当回事，直到死去两名青年后，方才开始重视起来，到现在前后死去的男子，总共三人。
<br/>　　简单的了解事情的经过后，慕清霄思索片刻，从老者口中得知村长家的位置后，便向村庄内部走去。
<br/>　　几分钟后，顺着老者所指的路线，直线行走，很快就看到一座小型四合院，此处就是村长的住处了。
<br/>　　相比村内的土屋，这座四合院，倒是与村外城镇中的房屋没有多大的差别。
<br/>　　村长是一名年过九十的老者，头发洁白，看到慕清霄后，毕恭毕敬，甚至吩咐老伴准备茶水与糕点。
<br/>　　“茶水糕点就不必了，先带我去看看死者的躯体。”
<br/>　　“死者的尸体就安放在后院，请仙人跟我来。”
<br/>　　跟在村长身后，来到后院，慕清霄目光一凝，放眼望去，在院落中一颗枝繁叶茂的愧树下，摆放着三具黑色的棺椁。
<br/>　　愧树繁茂的枝叶正好遮住了倾洒而下的阳光，使得院落中的气氛显得有些压抑。
