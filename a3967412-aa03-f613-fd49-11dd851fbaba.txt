　　第226章
<br/>　　杨龙停最近过的不是很好，至少没有程大雷看上去那么好。
<br/>　　他一来到长安城，就察觉到一种不安的气氛。帝国首府，繁华长安，可在繁华中似乎潜藏着某种怪物，随时会扑过来，向自己亮出牙齿。
<br/>　　这也是为何，在他的住处周围永远部着重兵。
<br/>　　他不喜欢长安，他想回家。在幽州，自己可以只手遮天，呼风唤雨，然而在长安，身份比自己高的人比比皆是，必须夹起尾巴做人。
<br/>　　可想回去也不是简单的事情，明帝要自己在武科结束的日子出现，在天下士子面前接受封赏。
<br/>　　杨龙停一天天推算着日子，心情越来越烦躁。
<br/>　　咚咚咚。
<br/>　　伴随着急促脚步声，李善言来到房内，看没有其他人，他道：“主公，刚才宫里派人来打听一个人，问我们有没有听说过？”
<br/>　　“谁？”
<br/>　　“程大雷。”李善言吐出三个字。
<br/>　　杨龙停猛地明白到自己的不安感从何处而来。
<br/>　　“主公，怎么回他们？”李善言。
<br/>　　杨龙停皱眉思索着，道：“你觉得他还活着么？”
<br/>　　“从那样的悬崖跳下去，自然不该还活着，只是我们的确没有找到他的尸体。”
<br/>　　杨龙停有些失望，他想从李善言这里得到的是一个确定，确定程大雷已经死了，像落叶般被扫去，拥有不会在世间出现。可惜李善言没能给他这个消息。
<br/>　　他沉默片刻，道：“好了，我知道该怎么回他们。”
<br/>　　……
<br/>　　咿咿呀呀的曲调，是锣鼓轻动的声音，一唱三叹，曲调哀婉动人。
<br/>　　当李婉儿到的时候，就看到这样一幕，程大雷坐在屋檐下，面前一群戏子正在哼唱。
<br/>　　“嚯，姓程的，没想到你也开始养戏子了。”李婉儿抿抿嘴，长安的达官贵人中间，有许多喜爱男风的，在家里豢养着俊俏的小戏子。在这京城算不得什么大事，甚至可以说是名士的风流雅事，却是没想到，程大雷刚抖起来没几天，就也爱上了这一套。
<br/>　　程大雷一激灵，顿时站起来，拿着一块毯子盖住头。
<br/>　　“有话好说，你想干嘛？”
<br/>　　“瞧把你吓的，你在做什么，调教戏子？”
<br/>　　上次的事情，的确给程大雷留下阴影，发疯的女人的确很恐怖，孔老夫子不是都说过么：唯女汉子和熊孩子不可惹也。
<br/>　　见李婉儿没有动手的意思，程大雷才稍微放下心，他冲戏班招招手，道：“青衣，你过来。”
<br/>　　楚青衣，戏班内的红角。
<br/>　　看着面前的女子，或者是男子走过来，李婉儿明显怔住，一时有些移不开视线。
<br/>　　对方穿着女装，身材修长，却柔弱的像是柳条，一笑一颦，皆有倾城风景。
<br/>　　就算是李婉儿也被对方的美色吸引，而对方不是以男子的身份吸引他，而是以女子的妩媚。
<br/>　　这不禁令人不寒而栗，浑身起鸡皮疙瘩。
<br/>　　“世间当真有如此祸国殃民的绝色。”
<br/>　　李婉儿情不自禁感慨一声。
<br/>　　“是不是感觉自惭形秽，我觉得应该让青衣好好教教你怎么做女人。”
<br/>　　从一个女人的角度，楚青衣的确甩李婉儿十八条街，就连李婉儿也不得不承认这一点。
<br/>　　“程大雷，你堕落了，养女人就算了，你竟然，养男色！”
<br/>　　“姑娘，请自重。”
<br/>　　楚青衣忽然道了一声。
<br/>　　李婉儿一怔，却听程大雷道：“李董，你误会了，人家是卖艺不卖身的。”
<br/>　　楚青衣的确有些女儿貌，可娘娘腔和龙阳好是两回事，楚青衣喜欢的的确是女人。当初就是有某大官贪慕楚青衣的美貌，将他陷在大牢，楚青衣宁死不从，最后是程大雷救了他。所以他才会对程大雷言听计从。
<br/>　　程大雷挥挥手让他退下，问道：“李董，你找我什么事？”
<br/>　　“嘿，你的好事到了，明日父王就要在天下士子面前封你，我听说了，至少是四品官，激不激动，兴不兴奋？”
<br/>　　明日就是武科结束的日子，一边是封赏武状元，一边是封赏程大雷，最重要的是在天下人面前封赏杨龙停，就连明帝和这次戎族的谈判团也会露面。
<br/>　　“兴奋，当真是兴奋。”程大雷垂头默默道了一句，忽然道：“李董，明日那么重要的日子，肯定有歌舞助兴，你能不能把青衣他们安排进去。”
<br/>　　“捧一个戏子你可够费心思的，我为什么要帮你？”
<br/>　　“你难道不想知道你六哥的消息？”程大雷道：“有人在青州见过他，还活着。”
<br/>　　……
<br/>　　青州，
<br/>　　一望无边的青纱帐。
<br/>　　“抓住他，抓住他，抓住那两个偷鸡的。”
<br/>　　一群村民提着锄头，木棒，口中发出阵阵吼声。
<br/>　　在他们前面两个人跑得气喘吁吁，卖命的向前狂奔。
<br/>　　“不就是偷只鸡么，这架势偷头牛也不至于啊。”一个裹着羊皮大袄的人道。
<br/>　　“我们偷得好像不是一只鸡，是把他们一村的鸡都偷光了。”穿着麻衣的人说。
<br/>　　“跑呀，他们追上来了。”
<br/>　　“追上他们，别让他们跑了。”
<br/>　　“这两个混蛋，把我家的老母鸡偷吃了，还正下蛋呢。”
<br/>　　“追呀，追上了打死他们。”
<br/>　　二人跑得跌跌撞撞，终于将村民甩开。
<br/>　　夜来，二人在野地里点了一堆火，围着篝火暖身子。
<br/>　　“该再去偷件衣服，穿着这大袄实在太像个戎族了。”
<br/>　　福德勒扫他一眼，不是很想搭理他，低头看着面前飘忽的火苗。咕咕，他肚子不争气的发出响声。
<br/>　　“哈哈，你是不是饿了。想不想吃东西？”
<br/>　　“说的好像你不饿似的，你又没吃的。”
<br/>　　“嘿嘿，谁说我没有。”男人从怀里掏出一个被破纸包着的鸡腿：“想不想吃啊，想不想吃？”
<br/>　　“你为什么还有，我明明看见你吃光的。”
<br/>　　“哈，我能什么都告诉你。”
<br/>　　福德勒夺过来，不顾一切的狼吞虎咽，年轻人看着他的样子，道：“你慢点吃，记得那纸给我留着，我还要擦屁股使。”
<br/>　　“呃……”福德勒有狠狠抽他的心情。
<br/>　　突然，年轻人过来揽住他的肩膀。
<br/>　　“喂，你以后就跟我混吧，跟着我吃香的喝辣的。”
<br/>(本章完)
