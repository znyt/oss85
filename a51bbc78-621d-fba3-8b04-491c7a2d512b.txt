　　李孝恭臭着一张脸不再说话了，一直到了大殿上，都没有再理会李承乾，看样子是郁闷的够呛。
<br/>　　如今东征已经定局，高句丽作死的行为连出兵的理由都直接送给大唐了，可见东征讨伐高句丽那是天意使然，若不出兵才是
<br/>　　李二坐在高高的御阶之上，精气神那是高的不行，整个人全身上下满是战意，和前段时间上朝时蔫蔫的表现一比，就像是打了鸡血一般。
<br/>　　“玄龄，粮草、物资可有齐备？大军出征后勤十分重要，莫要误了事情。”不等执事的太监宣布早朝开始，李二就第一个开口了，而且第一句话就和东征有关。
<br/>　　“陛下，粮草物资正在进行二次清点，不日便可运往边境。”老房当年就跟着李二打了不少的仗，押运物资、粮草也不是第一次了，自然不会在这方面有什么差错。
<br/>　　“无忌、克明，军械呢？”
<br/>　　“陛下，出征的十三卫人马军械已经经过检查，没有任何问题！”长孙无忌走出朝班应道。
<br/>　　“好！”李二点点头，将目光转向殿下众人，扫视一圈之后说道：“今天下大定，唯辽东未靖，朕欲亲取之，不遗后世之忧，望尔等众志成城，同心协力……”。
<br/>　　“臣等遵旨！”出征之事已定，自然不会再有反对的声音。
<br/>　　武将一侧更是一个个红着眼睛等着老李那一句：三日之后，校军场点兵出征。
<br/>　　不过此间却有两人早就已经踏上征途，一个是统领水师的张亮，此时这家伙已经带着他的五百假子（干儿子），顶着沧海道行军大总管的名头，领军出征。
<br/>　　另一个则是李绩，这位有儒将之称的大唐将军此时已经离开定襄城向幽州进发，在幽州与东征的先锋大军汇合，领辽东道行军大总管之职，兵进辽东征讨高句丽。
<br/>　　正是有了这两路先锋军，老头子这边才能紧不慢的安排调度，安安总计六路大军，十一卫三十余万人马（不算辅军）枕戈待发。
<br/>　　不过尽管朝堂之上场面热烈，讨论之声不绝于耳，但是李承乾不管怎么看都觉得老头子这次东征很有问题，似乎更像是出去旅游，而是出去打仗。
<br/>　　但是管它呢，反正老头子给他安排的任务是留守长安，在李孝恭、杜正伦等人的辅助下好好监国。
<br/>　　监国啊，这不是第一次了，上次差点被累死，不过好在现在已经有经验了，该偷懒的时候绝不手软便是。
<br/>　　正走神的时候，李承乾发现李孝恭正在拿手不断的捅自己，忍不住扭头看过去，却发现这老李头儿正不断的给自己打眼色，似乎在向老头子那边看。
<br/>　　正琢磨呢，老头子的声音就传过来了：“太子，想什么这么入神？朕唤了你三声都没有听到，莫非是有什么好的进攻计划？”
<br/>　　“呃，父皇……”
<br/>　　毛的计划啊，刚刚不过是想着可以有很长时间睡到自然醒而已，怎么就被老头子发现了呢。
<br/>　　“怎么？莫非是真有？若是真有好计划便拿出来说说，也让你的这些个叔伯们给你参详参详。”老头子似乎并不打算轻松放过李承乾，死死的扣着他走神的这一段不放。
<br/>　　没办法了，编吧，古今中外听过那么多的战例，不管实用不实用，先拿出来一个把眼前的事情应付过去再说。
<br/>　　想到这里，李承乾从自己的位置上站起来，走到大殿中间：“父皇，儿臣就是在琢磨着，我们能不能再出一支偏师，以此偏师断了高句丽的后路呢？”
<br/>　　“哦？不想吾儿真有高见，你且说来，朕来听听。”李二开始只是想给李承乾一个教训，让他不要在自己讲话的时候走神，可没想到这小子竟然真的能想出办法来，这也引起了他的一丝兴趣。
<br/>　　“父皇可记得马訾水？就是我们现在叫鸭绿水的那条水道？”李承乾估计老头子应该是把那道后世鼎鼎有名的江给忘记了，所以这次出兵根本提都没提，只是一个劲的强调卑沙城。
<br/>　　“鸭绿水？”李二皱眉想了想，却没想起来，不过马訾水他却是知道的，只是东征和马訾水有什么关系？
<br/>　　“父皇，我们完全可以选一支偏师，入鸭绿水，断了高句丽与辽东的联系，到时候给他们来个铁锁横江，辽东一带还不是任我们施为，搓扁捏圆想怎么搞他就怎么搞他。”李承乾一边说着，脑中想到的却是‘雄赳赳气昂昂，跨过鸭绿江……’。
<br/>　　“嘶……”老头子听完李承乾的解释，不由倒吸一口冷气。
<br/>　　这特么是绝户计啊，完全就是把高句丽一分为二，摁住一头往死里揍的节奏。
<br/>　　不过，似乎这个主意还真是不错，想到了马訾水的位置，李二突然觉得李承乾的这个想法的确有些可取之处。其中最大的优点在于如果今年拿不下辽东，完全可以休养一冬接着打，而不用在乎之前的战果前功尽弃。
<br/>　　“太子此计虽然不错，然我水师现在已经尽去辽东，何来偏师？”候君集到底还是兵部尚书，李承乾说的这些他也有权过问。
<br/>　　而且松州之战老候与李承乾有过很多次的接触，也知道自己能去松州是这小子举荐的，所以心里对这个刚刚成年的太子抵触的情绪多多少少的淡了些。
<br/>　　至于说当初为什么这小子处处针对自己，老候目前只能认为是自己长的太帅，把这小子给惊到了，所以才会那么做吧。
<br/>　　“偏师当然有，不过想要调回来却需要一些时间。”李承乾眉毛一挑说道：“难道大家都忘了去南海维和的那支舰队了么？独孤青云这几年在南海可是立下了赫赫威名，海战不知打过多少，把他调回来与高句丽水师一战不是正好？”
<br/>　　“青云么？”时隔这么多年，李二几乎快要把他当初的那个护卫给忘光了，现在被李承乾一提，立刻想了起来，眼中也露出了一丝怀念之色。
<br/>　　李承乾接过老头子的话头：“不错，现在独孤青云可是在南海闯下了‘海龙王’的绰号，想必一条小小的鸭绿水，应当难不倒他吧。”
