　　“楚前辈小心，这古权，一直以来都是我在古家之中的仇敌，处处跟我作对，想要至于我死地，我前去斩杀血剑魔尊，也是因为此人激起的。”
<br/>　　“这三位监察使，分别名为荒铁血，剑越，公孙傲，也都是七大宗门的弟子中的天才，实力极强，同境界无敌的人物，不好对付。”
<br/>　　古锦一边对着那古权开口，一边暗地里对着楚风眠传音道。
<br/>　　“现在情况不利，还请楚前辈先行离开避避风头，等这阵风头过去，我再来报答楚前辈的救命之恩。”
<br/>　　“避避风头？不需要，就这几个废物罢了，他们敢跟我作对，我就杀了他们。”
<br/>　　楚风眠轻描淡写的开口道。
<br/>　　其他宗门的弟子，招惹上，楚风眠也许还会给一次机会，不过这七大宗门的弟子，只要是敢招惹到楚风眠的头上，那楚风眠便是只有一个选择，就是杀。
<br/>　　听到楚风眠的话，古锦都是一愣，眼神中露出几分不可置信的眼光，看向楚风眠。
<br/>　　他如今提醒楚风眠，也是不想要让楚风眠跟这些人起冲突，毕竟是七大宗门的弟子，在这中域之中，几乎是没有任何人想要去招惹的。
<br/>　　就算是一些超越生死的老祖，也不会是轻易的去招惹七大宗门的弟子，可现在楚风眠话中的意思，居然是丝毫不将这三人，放在眼中？
<br/>　　“二小姐，我们也是奉命行事罢了，你就不要为难三位监察使了。”
<br/>　　古权嘴角露出几分冷笑，看向楚风眠的眼神中也露出了几分敌意，讥讽。
<br/>　　“三位监察使，请吧，先将这位阁下擒下，好好问出他的身份再说！”
<br/>　　“好！”
<br/>　　三位监察使同时长啸一声，猛然之间无数灵力便是落了下来，化为了一尊巨大无比的手掌，直接从天而降，向着楚风眠直接便是抓了过去。
<br/>　　这三人出手，便是没有任何的保留，爆发出了极为可怕力量，其中两人出手，压制住楚风眠的身形，让楚风眠动弹不得，就连逃跑都做不到。
<br/>　　而其中那荒铁血，便是全力出手，镇压楚风眠，要将楚风眠硬生生的给擒下。
<br/>　　“荒兄的大擒拿手，比起之前更要强大了，已经要大成了。”
<br/>　　古权在一旁笑着出声道，一双眼神中，都是露出了几分不屑的看向古锦，似乎是在嘲笑一般。
<br/>　　“一个不知名的小子，也敢跟我们七大宗门的弟子作对，很好，我会先擒下你，在好好折磨你，倒时候看看你要不要乖乖说出自己的身份来！”
<br/>　　荒铁血看着楚风眠，狰狞一笑，旋即大手一挥，这巨大无比的手掌便是冲着楚风眠直接抓了过去。
<br/>　　眨眼之间，这巨大无比的手掌，已经是落入到了楚风眠的身躯面前，要将楚风眠的身躯硬生生的给镇压。
<br/>　　另外两名监察使看到这一幕也是在冷笑，没想到到了这个时候，楚风眠居然你还出手，好像是被吓傻了一样。
<br/>　　本来他们觉得楚风眠前来，还是想要跟他们来争一争的，没想到居然这么快就被吓傻了。
<br/>　　这荒铁血的大擒拿手，就算是全力抵挡，同境界的武者之中能够抵挡下来的，也是寥寥无几，更何况到了现在，楚风眠都还没有出手，都像是被吓傻了一样。
<br/>　　这样的请客下，楚风眠被一掌镇压，几乎是板上钉钉的一件事。
<br/>　　“楚前辈！”
<br/>　　古锦在一旁都是担心的喊了一声。
<br/>　　要是楚风眠真的被镇压，落入到了这四人的手中，那楚风眠可就彻底完了，不死也要重伤，这些七大宗门的弟子出手，绝不会手下留情。
<br/>　　这灵力波动，引得周围不少武者，都纷纷过来，围观了起来。
<br/>　　当看到楚风眠交手的对象，居然是三位监察使的时候，不少武者都是摇了摇头。
<br/>　　三位监察使，在宁州古城接近是无敌的人物，除非是超越生死的老祖，不然都不是他们的对手。
<br/>　　在场武者都是只能摇了摇头，这就是有眼无珠，招惹到了不该招惹的人，如今楚风眠生死境巅峰的境界，今日被镇压，也算是板上钉钉的事情了。
<br/>　　几乎是没有任何人会觉得，在这样恐怖的镇压之下，楚风眠还可以脱身。
<br/>　　这巨大无比的手掌，在这个时候，已经是接近到了楚风眠的面前，距离楚风眠，不过是还有三尺之遥。
<br/>　　但是就是这三尺，却是仿佛如同天际一般的遥远，这巨大无比的手掌，在距离楚风眠身躯三尺的地方，仿佛是受到了一种无形的力量，突然给抵挡住了。
<br/>　　令这巨大无比的手掌，便是无法再进一步，再也无法靠近楚风眠。
<br/>　　“就你们这种实力，也敢在我的面前放肆？”
<br/>　　楚风眠缓缓的抬起头，眼神中尽是冷笑，只见他缓缓的抬起手掌，伸出一根手指，一指向着前方点了过去。
<br/>　　这手指一点，便是化为了一道金色光芒，直接冲了过去。
<br/>　　这巨大无比的手掌，在这一指点出的金色光芒面前，被一下击破，所有灵力尽数散去，不留任何一点痕迹。
<br/>　　而楚风眠，站在原地，却是毫发无伤，甚至是脚下就连动都没有动过，轻轻松松，便是轻而易举的化解了这一切。
<br/>　　“什么？”
<br/>　　“这怎么可能？毫发无伤？”
<br/>　　“这可是大擒拿手！荒古门的无上绝技！”
<br/>　　“这不可能啊，难道是我眼花了？”
<br/>　　一尊尊武者，都在揉着眼睛，似乎都无法相信这面前的一切，现场一片哗然。
<br/>　　“大擒拿手，就这样被挡下了？”
<br/>　　剑越，公孙傲都是目光呆滞，眼神中无比吃惊。
<br/>　　当然最为震惊的，还是那荒铁血，他的大擒拿手的威力，他自己清楚，同境界武者，能够抵挡住的，都可以说是寥寥无几。
<br/>　　而像是刚刚那样，楚风眠都没有率先出手，想要抵挡住这大擒拿手，更是几乎没有任何的可能，现在更是还有剑越，公孙傲他们二人的相助。
<br/>　　三人一起出手，镇压楚风眠，根本就是板上钉钉的事情，谁能想到，楚风眠居然是一指间，就足以打破他们的联手。
