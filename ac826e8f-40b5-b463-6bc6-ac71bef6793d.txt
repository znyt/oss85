        “喂，您好！”
<br/>
<br/>        虽然这大半夜的被人打扰到了睡眠，傅昼景仍旧保持着绅士风度，接听电话。
<br/>
<br/>        “您好，我们是滨江城南路派出所，请问您是季思妍小姐的男朋友吗？”对方是一口官方腔调。
<br/>
<br/>        傅昼景不假思索地回答道：“不是！你打错了！”
<br/>
<br/>        “对不起，打扰了！”对方刚把话说完，听筒那边，隐约间传来一个女人的无助的抽泣声。
<br/>
<br/>        “他真的是我的男朋友！阿景——你来接我回家好不好？呜呜——”
<br/>
<br/>        季思妍的呜咽声隐隐地从听筒里传来。
<br/>
<br/>        傅昼景瞬间睡意全无，却还是狠心将电话给挂断了。
<br/>
<br/>        既然她在派出所，就无性命之忧，更何况，她还有个哥哥。
<br/>
<br/>        她要是出了什么事情，她哥哥会去管她。
<br/>
<br/>        傅昼景心里这么想着，却翻来覆去怎么都无法再次入睡。
<br/>
<br/>        当他坐起身来，欲准备下沙发穿鞋的时候，无意间地抬眸，看到躺在床上的童筱颖，又犹豫了一下，最终还是将双脚重新放回了沙发上，躺了下来继续睡觉。
<br/>
<br/>        他现在已经有筱颖了，就不能再去管季思妍的任何事情了。
<br/>
<br/>        对，他已经有筱颖了……
<br/>
<br/>        傅昼景闭上眼睛，心里、脑海里，不停地给自己强调这句“我已经有筱颖了！”的话。
<br/>
<br/>        或许，是两个人之间住在一起久了，会在无形之中产生一种心灵感应。
<br/>
<br/>        童筱颖倏然睁开了双眼，双手半握成拳，轻轻地放在胸前，声音柔柔弱弱地唤了声：“昼景……”
<br/>
<br/>        “嗯？”傅昼景怔了一下，立即应了声，“你还没睡吗？”
<br/>
<br/>        “我……”童筱颖欲言又止，顿了顿后，才接着说道，“可不可以和你一起睡？”
<br/>
<br/>        她的声音里透着一丝娇羞。
<br/>
<br/>        傅昼景忍俊不禁地咧嘴道：“小傻瓜，当然可以！”
<br/>
<br/>        他说着，便又从沙发上坐了起来，然后穿上鞋，走到了床边。
<br/>
<br/>        童筱颖能感觉到身后的动静，立马翻了个身，面对着傅昼景，一对水灵灵的大眼睛，在昏暗的房间里却泛着微弱的光点。
<br/>
<br/>        她的眼睛里，透着对他的期待。
<br/>
<br/>        傅昼景上了床，轻轻地在童筱颖的身边躺了下来。
<br/>
<br/>        童筱颖下意识地往傅昼景的怀里钻，他还未抱住她，她便主动搂上了他的腰肢。
<br/>
<br/>        他身上有她喜欢的那种沐浴乳的味道，他用的沐浴乳，是她买的自己喜欢的那款香味。
<br/>
<br/>        不知道为什么，她有点儿贪恋他怀里的温暖了，想就这样一直躲在他怀里睡觉，永远也不要醒过来就好。
<br/>
<br/>        不知不觉中，童筱颖沉睡了过去。
<br/>
<br/>        傅昼景却仍旧睡意全无，这时候，他放在沙发上的手机又震动了起来。
<br/>
<br/>        这一回，他并未起身去接听，而是任由手机在那沙发上叫嚣。
<br/>
<br/>        手机振动了许久，才消停下来。
<br/>
<br/>        傅昼景心里也挣扎了许久，在手机的振动安静了后，将怀里的童筱颖抱紧了点，并低头吻了吻她的额头，缓缓合上了眼睛。
<br/>
<br/>        不知道，这世界上，是否也有男人像他今晚这种情况，在老婆和前任之间这样纠结过？
<br/>
<br/>        季思妍其实一直是他心头的一根刺，拔了会痛，不拔会难受。
<br/>
<br/>        一通又一通的电话无人接听后，季思妍坐在派出所里从失望到绝望。
<br/>
<br/>        她真的没想到，阿景会对她如此绝情。
<br/>
<br/>        “你们家还有谁会过来？”男警员皱着眉头问。
<br/>
<br/>        一旁，一个老大爷还在叫嚣：“反正就是她把我撞倒的！我骨折了，要去医院，她得对我负责！”
<br/>
<br/>        “喂，你这个老大爷活到这个岁数是真不打算要脸了是吗？我好心扶你，你到讹上我了！”季思妍顿时一肚子火气，站起身来，又跟这个老大爷吵了起来。
<br/>
<br/>        当时，她一个人因为心情不好，徒步回家，因为低着头疾步走路，自己又在想阿景的事情，岂料有人撞了一下她的手臂，紧接着，当她回神抬头的时候，就看到这位老大爷跌坐在她脚边喊着“疼”。
<br/>
<br/>        季思妍当时也没留心眼，见这老大爷含“疼”，连忙伸手去将这位老大爷从地上扶起来，结果，这老大爷就这样讹上她了，要她赔偿他五万块钱。
<br/>
<br/>        本来，她是打算送这老大爷先去医院的，但是这老大爷的态度十分恶劣，还跟她起了争执，把她气得连医院都不想送他去了，直接导致她报了警，让警察带着她和这位老大爷一起进了派出所。
<br/>
<br/>        “你家到底来不来人？”男警员不耐烦地又问了一声。
<br/>
<br/>        季思妍没好气地回了一句：“傅夜沉是我未婚夫，你们把他给我找来！”
<br/>
<br/>        “什么？”男警员听到“傅夜沉”这三个字后，下意识地和坐在旁边办公桌前的男同事面面相觑。
<br/>
<br/>        两人愣了一下，男警员才接着说道：“那麻烦你给你的未婚夫打个电话！”
<br/>
<br/>        “我跟我未婚夫吵架了，我不想打给他。你们自己想办法找他过来解决这件事情！”季思妍双手抱臂，翘着二郎腿，傲慢地坐在椅子上。
<br/>
<br/>        男警员和身旁的男同事交头接耳的商量着什么，片刻后，男警员的那个男同事便起身出去了。
<br/>
<br/>        半个小时后……
<br/>
<br/>        一个高大的身影，气喘吁吁地跑了进来。
<br/>
<br/>        “漫雪？”仓促的呼唤声中，磁性的嗓音，仍旧很动听。
<br/>
<br/>        傅夜沉一听是自己的未婚妻出了事，想都没想，就连问都没问清楚，就直接飙车赶到了这里。
<br/>
<br/>        季思妍看到傅夜沉出现在自己面前，还一脸焦急地呼唤自己“漫雪”的时候，简直是一脸懵然。
<br/>
<br/>        她完全没料到，傅夜沉居然会赶过来！
<br/>
<br/>        “怎么是你？”看到休息椅上坐着的人是季思妍，傅夜沉不悦地皱起了眉头。
<br/>
<br/>        季思妍见状，立即装出一脸无辜的样子，楚楚可怜地嘟着唇，委屈地说道：“夜沉哥，我、我这是好心没好报！我好心扶了那老大爷一把，他不感恩，反而讹上我了。”
<br/>
<br/>        “诶？你这女人怎么竟说反话！明明是你先撞倒我的！我的脚到现在还疼着了，你都不先送我去医院。”老大爷提高了音调，又跟季思妍吵了起来。
