
<br/>　　<br/>　　    王惊蛰之前在家里翻箱倒柜的时候，就已经发现家中有很多符纸，桃木，还有罗盘这些东西了，他虽然是失忆但不等于啥也不懂，他隐约觉得可能自己这个家庭挺不一般的。<br/>　　<br/>　　    失踪的老爹，神秘莫测的姐姐，这个家上空失踪都笼罩着神秘兮兮的气氛，王惊蛰不知道自己会何时能把这神秘的面纱给揭开。<br/>　　<br/>　　    齐泉开车拉着他们三个去了武侯祠附近的一个高档小区，停在一栋楼前说道：“这里有我的一栋房子，正好够你们三人住了，屋子里东西也比较齐全，你们先安顿在这里，休息一下，过个两三天我再给你们安排事做，本来今天要给你接风的，但我上面的林先生今天找我有事，所以接风就往后排一下吧”<br/>　　<br/>　　    丁武说道：“没事泉哥，以后机会有都是呢”<br/>　　<br/>　　    齐泉领着他们来到楼上，打开一间房门，房子装修和布置都不错，东西一应俱全，给他们送到屋里后，齐泉又把丁武拉到了一旁，从包里拿出两叠钱塞到了丁武的手里，说道：“你叫我泉哥，就不要拒绝，在我的眼里钱什么也不是，兄弟朋友最重要，这些你先花着吧，等过两天我忙完了，再去公司给你们安排一下，先安顿安顿吧”<br/>　　<br/>　　    丁武拿着钱，嗯了一声说道：“那我就不客气了，泉哥······”<br/>　　<br/>　　    丁武这人是最不善言辞的，一切行事准则都在行动上，他觉得动手和动脚远比动嘴皮子要实在得多，这也是齐泉比较欣赏他的地方。<br/>　　<br/>　　    齐泉走了以后，丁武回来了，王惊蛰看着他欲言又止。<br/>　　<br/>　　    “你和我这么生疏，我实在是真不习惯，要不你干脆跟我装的熟点行不行？”<br/>　　<br/>　　    王惊蛰抿了抿嘴，说道：“那个泉哥杀过人，还不是一个两个”<br/>　　<br/>　　    丁武笑了。<br/>　　<br/>　　    王惊蛰有点恼火的说道：“你笑什么，我说的是事实”<br/>　　<br/>　　    “多新鲜啊，我还杀过人呢”<br/>　　<br/>　　    “你那是误杀，是迫不得已的，他和你不一样”<br/>　　<br/>　　    丁武拍了拍王惊蛰的肩膀，说道：“首先你得明白，我俩本来就是在监狱里认识的，这一点就注定我们都有着不同寻常的经历，再一个就是，泉哥所做的就是见不得光的事，手上见血有人命也正常，而我以后跟了他能干的也是这种事，明白？”<br/>　　<br/>　　    王惊蛰皱眉说道：“你就没有别的选择了？”<br/>　　<br/>　　    丁武叹了口气，说道：“一个人有一个人的活法，乞丐的生活就是讨饭，老板的生活就是生意，而我和泉哥这种人生来就注定是要行走在刀刃上的，这就是我的活法，至于你问我有没有别的选择，那我只能说，也许会有，但绝对不会适合我，而我也不喜欢”<br/>　　<br/>　　    王惊蛰沉默了，他忽然想到也许自己在没有失忆之前，那会是什么活法呢？<br/>　　<br/>　　    休息了一会之后，晚间他们三人出来，在外面找了一间餐馆吃晚饭，这天晚上丁武要了不少的酒，他和王惊蛰喝的酩酊大醉，菜刀文则是滴酒不沾，王冬至交代的是，跟在王惊蛰的身边，平时他就相当于一个透明的人，没有什么存在感也可以，但当王惊蛰出现什么我危机的时候，他就得闪亮登场了。<br/>　　<br/>　　    一夜宿醉之后，清晨，王惊蛰起床后现在就习惯站在床边端详下这个对他来讲，不知道算是陌生还是熟悉的世界。<br/>　　<br/>　　    熟悉是因为，他已经活了二十几年了，陌生是因为他需要重新认知一下，所看到的所接触到的，都要从头来过。<br/>　　<br/>　　    “这种感觉真的是太讨厌了······”王惊蛰烦躁的晃了晃脑袋。<br/>　　<br/>　　    川中，是一座很有底蕴的城市，俗话说少不入川，老不出蜀，这个被称为天府之国的地方，有着太多让人流连忘返的东西了，这里出美女，这里的火锅最够味，这里的生活最悠闲。<br/>　　<br/>　　    这必然也是一座充满了故事的城市。<br/>　　<br/>　　    王惊蛰眺望远处的时候，客厅里的电视上正放着一首歌，是赵雷的成都。<br/>　　<br/>　　    “在那座阴雨的小城里，我从未忘记你，成都，带不走的只有你，和我在成都的街头走一走······直到多有的灯都熄灭了也不停留，你会挽着我的衣袖，我会把手揣进裤兜”<br/>　　<br/>　　    王惊蛰听着听着就跟上了节奏哼了起来，一首歌放完，最后一声曲调落下的时候，他恍然说道：“我好想听过这首歌，我也好想来过成都”<br/>　　<br/>　　    王惊蛰一脸惊愣，随即他穿上衣服随便洗了把脸然后就推开了房门，刚起床的菜刀文看他要出去，就问道：“你干什么去啊？”<br/>　　<br/>　　    “出去逛逛，顺便买点早餐，一会就回来了”王惊蛰头也不回的说道。<br/>　　<br/>　　    从家中出来，王惊蛰漫步在街上，走的很慢，然后左顾右盼的打量着四周的人群还有附近的建筑。<br/>　　<br/>　　    赵雷的一首成都忽然让王惊蛰想起，自己曾经听过这首歌，也曾经来过这里，熟悉的感觉跃然而上，他发现这也许是唤起他记忆的一种方式，既然来过这里，那就走走也许会看见熟悉的人或者物。<br/>　　<br/>　　    住的地方离武侯祠很近，没过多久王惊蛰就溜达到了武侯祠外，这个点还没有开门，他就站在门外仰头打量着这座建筑。<br/>　　<br/>　　    王惊蛰发现，武侯祠的上空萦绕着一股淡淡的气息，这是国内唯一的一座君臣合祀祠庙。<br/>　　<br/>　　    站了片刻之后，武侯祠外的游人开始多了起来，等着排队进去。<br/>　　<br/>　　    王惊蛰驻足半天，发现这个地方好像没有什么可以唤起他的，皱眉摇了摇头叹了口气后转身就走了，当他转身刚离去的时候，武侯祠的大门也开了，开门的是个五十来岁穿着工作服的中年，他正好看见了王惊蛰转身时的侧脸。<br/>　　<br/>　　    “这不是那个卖菜刀的小伙子么，怎么又来了······”
