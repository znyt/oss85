
向缺和陈冬到丽景湾酒店的时候，酒店的老板加两个董事会的人就已经拱手而立了，三人的嘴里喷着酒气，眼珠子通红的，他们三人分别是从三个不同的应酬中赶过来的，为啥这么急呢？因为，平时他们想给宝鑫系的掌舵人打电话的胆子都没有，但今个主动接到了陈冬的电话，天上就算是下刀子了，他们都得顶着头盔过来。<br/>　　<br/>　　宝鑫系在整个京津翼意味着的是什么概念呢，简单的说就是宝鑫系里出来一个总经理级别的人，到一个地级市说要投资，主管的一二把手全都得严阵以待了。<br/>　　<br/>　　迈巴赫停到酒店门口，陈冬冲着他们勾了勾手指，对方三人过来后，陈冬指着表说道：“查一下，二十多分钟以前，你们酒店的328房间里，出了什么事”<br/>　　<br/>　　“唉，唉，好叻”丽景湾老板点了两下脑袋，然后一路小跑回到了酒店里，把大堂和值班经理都给叫了过来。<br/>　　<br/>　　迈巴赫里，陈冬打听道：“姐夫，谁啊都把你劳驾的亲自出马了？”<br/>　　<br/>　　“朋友的孩子，也是同道中人，在家里住了几天昨天走的，不知道怎么回事今天说是被人给扣了，孩子打电话的时候那个委屈啊，听的我……”向缺砸吧了下嘴，说的挺煽情的，其实他自己对于让王惊蛰去趟雷这件事还是有点汗颜的。<br/>　　<br/>　　片刻后，丽景湾的老板回来了，低着脑袋说道：“陈总，酒店来扫黄的，328的客人被带走了”<br/>　　<br/>　　“呵呵，五星级的酒店，什么时候扫黄的都可以扫的这么随便了啊”陈冬鄙夷的嘲讽了一句：“行了，哪里抓的人？”<br/>　　<br/>　　“分局的，黄记言支队长”<br/>　　<br/>　　“唰”车窗升上，临近关闭之前，陈冬淡淡的扔下一句话：“自己整理下该整理的，明天我用的时候给你电话”<br/>　　<br/>　　迈巴赫走了，丽景湾的老板抹了下头上的冷汗，然后呲着牙笑道：“那个什么，让酒店的公关部门准备下，过两天我亲自去一趟宝鑫总部，拜会一下陈总……”<br/>　　<br/>　　十分钟后分局门口，停着一辆闪着警灯的警车，车里就开车的司机一个人，跟迈巴赫并排停着。<br/>　　<br/>　　警车里的人，一手拿着电话，扭头冲着陈冬点头示意，然后说道：“小黄啊，我是冷局，你在哪呢？”<br/>　　<br/>　　“冷局啊，没什么事，我在外面有点应酬”<br/>　　<br/>　　冷局直接笑呵呵的说道：“哪里应酬呢，我过去贴个桌吧”<br/>　　<br/>　　黄记言顿时一滞，憋了半天后就吐出一个字来：“啊？”<br/>　　<br/>　　“贴桌就不贴了，但我有点事找你，地址告诉我，我一会还有点别的事，你尽快吧”<br/>　　<br/>　　黄记言寻思了一会，说道：“我在宏鑫物流院内……”<br/>　　<br/>　　黄记言挂了电话，心里就有点敲鼓，他仔仔细细的寻思了半天也没想明白领导找他干啥，但他还必须得告诉人家自己在哪。<br/>　　<br/>　　“嘎吱”黄记言推开关着王惊蛰的房门，屋里面他仰着脑袋，身上看似一点伤没有，但顾飞的手里拿着一本厚厚的杂志正垫在他后背上。<br/>　　<br/>　　“我们局里领导找我不知道什么事”<br/>　　<br/>　　“爱谁谁吧……”<br/>　　<br/>　　“顾少爷，万一是冲着他来的呢”黄记言指了指椅子上的王惊蛰说道。<br/>　　<br/>　　“你可能是没想清楚，你们局里的领导跟我爸差了几道杠，他爱谁来就谁来呗，见我不照样得立正么？”顾飞斜了着眼睛说道。<br/>　　<br/>　　黄记言瞬间无言以对。<br/>　　<br/>　　片刻后，黄记言走出物流仓库，站在门口抽着烟，一根烟刚抽了几口就看见冷局的警车来了，但让他有点掉眼珠子的是，后面跟着一辆迈巴赫还有一辆奔驰。<br/>　　<br/>　　开始的时候，黄记言只是警察这两车，但等迈巴赫开的近了，他看清车牌号的时候腿肚子就有点转筋了，这个车的号码，在他们内部都是有备案的。<br/>　　<br/>　　“咣当”冷局推开车门，直接奔着黄记言就走了过来，手插在口袋里问道：“今晚局里没有扫黄的活动吧？”<br/>　　<br/>　　“没，没，没有……”黄记言说话的时候都有点磕巴了，可他眼睛一直都瞄着那辆迈巴赫，看见车里下来陈冬和向缺，他瞬间就感觉嗓子都干了。<br/>　　<br/>　　“没有活动，你从丽景湾抓了个人走了，谁给予你的权利啊？”冷局呵斥了一句后，瞪着眼睛吼道：“人呢？”<br/>　　<br/>　　“里，面呢……”黄记言的脑袋嗡嗡直响，心里把顾飞给骂了一万多遍，你他么不是说这人没背景么？<br/>　　<br/>　　我去你么的，这是一般的背景么，怎么把宝鑫系的当家老板都给捅咕出来了。<br/>　　<br/>　　“一边站着，明天去局里找我喊报告，怎么回事，自己主动好好交代”冷局手指戳着他胸口说完，就推开物流仓库的门进去了。<br/>　　<br/>　　陈冬和向缺迈步过去的时候，压根都没瞅黄记言一眼，但他已经贴着墙边堆坐在了地上。<br/>　　<br/>　　“姐夫，咱不用叫点人过来啊？”陈冬边走边问道：“我一老板，至于动手么？”<br/>　　<br/>　　“有我在，能用得着你们谁出手啊……”<br/>　　<br/>　　陈冬想了想，点头说道：“那倒也是，我这都多少年没见过你横刀立马了，今天开开眼界了呗？”<br/>　　<br/>　　“开啥啊，有你在的话，还用得着我么？”向缺斜了着眼睛问道。<br/>　　<br/>　　“这话确实挺有道理的”陈冬一本正经的点头说道，两人说的话相当矛盾了，很有点扯犊子的风采，总得来说就是他俩都没想到，今天会有什么流血事件发生。<br/>　　<br/>　　因为，人的名，树的影啊！<br/>　　<br/>　　两人边走边聊，就好像散步似的，走到仓库里面的一间房时，他俩才顿住脚，然后向缺抬腿一脚就踹开了房门。<br/>　　<br/>　　房间里面，王惊蛰被铐在了椅子上，顾飞正探着脑袋伸出一根手指抬着王惊蛰的下巴。<br/>　　<br/>　　“唰”顾飞扭头，转过脑袋看着门口的向缺，愕然问道：“你谁啊？”<br/>　　<br/>　　向缺瞅着椅子上的王惊蛰“噗嗤”一声就乐了：“大侄子，你这是虎落平阳被犬欺了么？”<br/>　　<br/>　　王惊蛰羞涩的说道：“向叔，啥也别说了，你们这地方也太不按套路出牌了，一个腾格尔愣是给我安了个**交易的名堂……”
