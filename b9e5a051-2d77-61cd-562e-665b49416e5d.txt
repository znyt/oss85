<br/>        以他傅夜沉的权利，他确实可以说到做到。

<br/>
<br/>        婚企中心的服务人员一直询问着李忆莎和落以柔的意见，但她们都一副无所谓的反应。
<br/>
<br/>        “请问是否有偏好的礼服款式？”
<br/>
<br/>        “随便。”李忆莎毫无准新娘的热情回答。
<br/>
<br/>        “特别订制的手工礼服呢？”
<br/>
<br/>        “随便。”
<br/>
<br/>        “理想的喜饼品牌及数量？”
<br/>
<br/>        李忆莎开始有些不耐烦地看向落以柔。
<br/>
<br/>        落以柔尽量以平静的表情说道：“李小姐，这方面可能要问过女方长辈了，但无论要多少喜饼礼盒，傅总都没意见。”
<br/>
<br/>        “还是随便吧！”李忆莎又给了个模棱两可的答案。
<br/>
<br/>        “又是随便吗？”落以柔柳眉微蹙，淡淡的扫了李忆莎一眼，不懂她为何用如此敷衍的态度来应付她。
<br/>
<br/>        “夜沉也是这样无所谓的吗？”李忆莎试探性地问，其实她已经心知肚明了。
<br/>
<br/>        落以柔一丝不苟地回答：“傅总的意思是由你决定。”
<br/>
<br/>        “不。”李忆莎可不是天真的大小姐，“他不在乎，所以他才派你来跟我决定这些琐碎烦人的事。他之所以觉得只要我说好就好，没有任何意见，其实是他不把结婚当自己的事。”
<br/>
<br/>        “李小姐，这是你和傅总的私事，没必要在这儿跟我抱怨什么。”即使是事实，落以柔还是很没好气地说。
<br/>
<br/>        “那么，是他要和我结婚，还是你要和我结婚？”李忆莎很是幽默的反问。
<br/>
<br/>        落以柔面无表情地笃定道：“当然是他要跟你结婚。”
<br/>
<br/>        她真不知道，这个市长家的千金大小姐，哪来这么多废话？
<br/>
<br/>        “好吧，暂且不讨论这个。他有说了要请几桌吗？”
<br/>
<br/>        “李小姐决定你自己这边的桌数就好，傅总那边的亲友桌数不会很多。”
<br/>
<br/>        “你可以确定他的亲友桌数多少？”
<br/>
<br/>        这个落以柔还真的可以确定，因为来着之前，她就已经询问过师父郑忠威和傅夜沉的那个死党医生连华生了。
<br/>
<br/>        “当然！所以，李小姐你只要提你的要求即可，无需过问傅总这边的事情。”落以柔回过神来，一本正经地回答。
<br/>
<br/>        虽然傅夜沉和李忆莎要结婚了，但是傅夜沉对李忆莎的态度，简直无法用语言去形容。
<br/>
<br/>        落以柔并不知道他们之间到底发生了什么事情，但是从傅夜沉对李忆莎的这种敷衍程度来说，她笃定，傅夜沉压根就不爱李忆莎。
<br/>
<br/>        至于他们为什么突然要结婚，其中的原因，落以柔就不得而知了，她也试探性地问过连华生，但是连华生也表示他不知内情。
