　　楚风眠现在的实力，已经是今非昔比，已经是踏入了生死境。
<br/>　　比起曾经楚风眠离开北邙学院的时候，不知道是要强大了多少倍，他现在的灵识一动，已经是足以将北邙圣域的大部分，都给尽收眼底。
<br/>　　可是楚风眠却是没有感觉到，玄北圣者的气息。
<br/>　　看起来玄北圣者，不知道是因为了什么理由，居然是离开了北邙圣域中。
<br/>　　玄北圣者的实力，现在楚风眠隐约之间已经是察觉到，玄北圣者只怕是已经是步入了生死台阶的第五重神物之境了，一旦是踏入到了神物之境的武者，他的实力便是会发生翻天覆地一般的变化。
<br/>　　就算是在七大宗门之中，神物之境的武者，也足以称得上是老古董一般的存在，平日里一直都是在圣地之中修炼，想要突破到更高的层次。
<br/>　　几乎是很少会有人出现在外的，甚至是离开。
<br/>　　尤其是玄北圣者的离开，更是完全没有留下任何的信息，显示着一切都并不正常。
<br/>　　“不管如何，现在夜家的想要对付我，也并不容易，以我现在的实力，就算是在碰到那夜牧，也丝毫不惧。”
<br/>　　楚风眠心中暗自想到。
<br/>　　他现在身上拥有这足足四百道无上剑气，境界也已经是达到了生死境八重的地步，在加上他的九域剑术大成，这一切的实力，就算是让他在碰到那夜家的夜牧。
<br/>　　楚风眠也有自信，可以将其斩杀。
<br/>　　楚风眠的实力已经是不亚于了北邙学院的一些老古董，北邙学院的人想要对付他，也要付出无数的代价才行。
<br/>　　更何况楚风眠还有着自己的退路，只要是有着那云羽天宫当做最后的退路。
<br/>　　就算是楚风眠不敌，他现在也足以可以全身而退，正因为这一点，楚风眠才可以肆无忌惮的回到北邙学院来。
<br/>　　想到这里，楚风眠的心中倒是没有在太多担心的事，便是坐在了地上，从他的空戒之中，那三棵淬金竹，都被楚风眠拿了出来。
<br/>　　这淬金竹，在九域难寻，一般来说只会在金石之气浓郁的地方才会生长。
<br/>　　这苏荷能够得到这淬金竹，便是曾经他们苏家的旧宅之中，藏有这不少的金石之气，这才可以将这淬金竹给生长起来。
<br/>　　这三棵淬金竹，都足足是有着十几个竹节，一般来说，也淬金竹淬炼灵器，仅仅其中一节竹节便是足够，这三棵淬金竹，足以是淬炼数十件灵器了。
<br/>　　只要是被淬金竹淬炼过的灵器，都足以脱胎换骨，威力强化一倍有余，在楚风眠所在的时代，都是无数人争抢之物。
<br/>　　当然这淬金竹的效果，并非是对所有的灵器都有用，像是楚风眠的人皇罐，这种极道仙兵，淬金竹的力量将是没有任何的意义。
<br/>　　同时楚风眠手中的战龙宝车，祖龙至尊剑，甚至是那云羽天宫，都无法利用淬金竹的力量淬炼，这三件灵器，都是天级灵器之中的佼佼者了，淬金竹的力量还无法影响到他们。
<br/>　　不过这倒是并不代表这三棵淬金竹对没用，在楚风眠身上所拥有的无数灵器之中，其中一件，却是最为适合淬炼，正是他亲手炼制的那黑玄剑阵。
<br/>　　黑玄剑阵是被楚风眠刚刚炼制出来，还不足一年时间的灵器，其中的力量还没有彻底定型，在这个时候，利用淬金竹去淬炼，便是最好的选择。
<br/>　　一旦是这黑玄剑阵被淬金竹再度淬炼，黑玄剑阵的力量都足以暴涨无数倍！
<br/>　　“黑玄剑阵！出！”
<br/>　　楚风眠轻喝一声，这黑玄剑阵便是全部从楚风眠的身体之中飞了出来，十万把灵剑盘旋在空中。
<br/>　　楚风眠手一挥，三棵淬金竹便是全部飞到了空中，在黑玄剑阵之中，全部破碎，化为了一颗颗金色的微粒，全部融入到了那十万把灵剑之中。
<br/>　　这三棵淬金竹，如果楚风眠故意挑选黑玄剑阵之中比较强大的那些灵剑淬炼的话，只怕是足以缔造出数十件天级灵器。
<br/>　　不过对着黑玄剑阵足足十万把灵剑来说，数十件天级灵器所产生的力量，也不过是杯水车薪。
<br/>　　黑玄剑阵最为强大的，便是这十万之数的灵剑，一旦是打了出去，层层叠叠，让无数武者都无法打破。
<br/>　　所以楚风眠才选择，这将三棵淬金竹，全部融入到黑玄剑阵之中去，虽然无法令其蜕变，但是足以让黑玄剑阵的整体实力，翻上无数倍去。
<br/>　　“这黑玄剑阵，是我现在最为依仗的底牌之一了，现在这黑玄剑阵的力量，足以是同时困住数名碎命之境的武者了。”
<br/>　　楚风眠感觉到黑玄剑阵的力量越来越强，他的心中也是极为欣喜。
<br/>　　楚风眠身上的奇遇众多，甚至可以说在年轻一代的弟子之中，绝对是没有任何人可以来跟楚风眠比奇遇的多少。
<br/>　　不过楚风眠的奇遇虽然众多，但是大部分的奇遇，却都是不敢轻易展现出来的。
<br/>　　像是祖龙至尊剑，战龙宝车这等，楚风眠还是可以动用的。
<br/>　　但是那无上剑气，跟人皇罐，甚至是楚风眠的造化之力，本命剑元，云羽天宫等，这一切都不能暴露的。
<br/>　　一旦是暴露，任何一件都会给楚风眠带来极大的危机，所以楚风眠无法轻易动用。
<br/>　　这黑玄剑阵，可以算得上是楚风眠足以光明正大动用的底牌之一了，所以楚风眠急需这黑玄剑阵的蜕变。
<br/>　　毕竟现在楚风眠所要面对的对手，可不是那些生死境的人物了，而是许多踏上生死台阶的老古董了。
<br/>　　黑玄剑阵现在威力越强，也越是楚风眠的一大助力。
<br/>　　淬金竹的力量，全部融入到了黑玄剑阵之中，逐渐的令黑玄剑阵之中的十万把灵剑，全部蜕变。
<br/>　　而就在这个时候，在北邙圣域之中，一处独特的空间之中，无数身影，都在其中穿行。
<br/>　　这里是每一尊身影，都是绝对的老古董，最弱之人，都是足足有着碎命之境的实力。
<br/>　　所有人，都是超越了生死，踏入了生死台阶的人物。
<br/>　　这里便是北邙学院的真正核心，北邙宫。
