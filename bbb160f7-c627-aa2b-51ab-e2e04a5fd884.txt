        随后的这些天里，童筱颖每天都收到表白卡片、玫瑰花、水果以及零食，而傅夜沉也有每天都给童筱颖打电话，但是童筱颖不接。
<br/>
<br/>        起初是不接，最后手机号码成了空号，他便知道是什么原因了。
<br/>
<br/>        童筱颖直接将傅夜沉的手机号码拉进了黑名单。
<br/>
<br/>        虽然，她不知道每天放在宿舍门口的表白卡片、玫瑰花、水果和零食是不是傅夜沉送的。但是，她和傅夜沉之间确实没有什么，她总觉得彼此频繁地单独见面不好。
<br/>
<br/>        大哥和弟妹若是每天单独见面，还一起吃饭，表面上好像没什么异常，但是这种感觉很奇怪。
<br/>
<br/>        所以，童筱颖只能再次冷落傅夜沉。
<br/>
<br/>        傅夜沉揣测着童筱颖有可能将自己的手机号码拉入黑名单后，不禁给自己的助理郑忠威打了通电话。
<br/>
<br/>        “临海大学的所有大一新生，军训时间由原来的半个月延长到一个月！然后，集体去军校和军校里的大一新生一起军训！”傅夜沉非常淡定地说。
<br/>
<br/>        这不是教育局管的事情吗？傅大少爷怎么管上这种事情来呢？
<br/>
<br/>        郑忠威心里虽然疑惑，但还是点头答应了。
<br/>
<br/>        以傅家在省城的地位，高官都要敬畏傅家三分，更何况是傅夜沉一句话就能办到的事情。
<br/>
<br/>        于是，当童筱颖和班上的同学一起为十五天军训即将结束而感到欢喜的时候，却被班主任告知，还必须去军校参加半个月的实训军训时，大家都纷纷抱怨起来。
<br/>
<br/>        中午下训后，童筱颖和严恩恩打了水回到宿舍时，就听到舍友蓝筱蝶和恭如兰在调侃。
<br/>
<br/>        “哎，我们又不是军校的学生，为什么要去军校体验什么军人生活啊！”恭如兰坐在书桌前，翘着二郎腿修脚趾甲。
<br/>
<br/>        蓝筱蝶则躺在床上看一边漫画，一边憧憬道：“去军校军训也不错，说不定还可以找个帅气的军人当男朋友！”
<br/>
<br/>        “筱蝶，你这花痴病又犯了吧！”童筱颖将手里的水壶放到了书桌上，打趣地说。
<br/>
<br/>        “筱颖，你将来想嫁一个什么样的男人？”蓝筱蝶合上手里的漫画，趴在上床边缘，好奇地问。
<br/>
<br/>        童筱颖会心一笑：“建筑设计师。”
<br/>
<br/>        “找同行的啊！”严恩恩插了一句。
<br/>
<br/>        “同行的不好吗？我觉得同行的挺好！”童筱颖满心欢喜地说，至少傅昼景就挺好的，只不过，他经常要加班，很忙。
<br/>
<br/>        “我听说上一届、上上一届，大一新生的时候，就没见要搞军训，怎么到了我们这一届，就这么倒霉！”恭如兰感慨道。
<br/>
<br/>        严恩恩非常认同地点了点头，接着说道：“咱们还是赶紧收拾一下行李吧！下午就得集合坐校车去军校了！”
<br/>
<br/>        童筱颖微微耸了耸肩，表示不知道。
<br/>
<br/>        殊不知，她才是这件事情的“罪魁祸首”。
<br/>
<br/>        临海大学所有大一新生，集体迁移去临海军校，与临海军校的大一新生合并军训的第一天开始，童筱颖和三个舍友在凌晨五点钟的时候，便被哨声叫醒，急匆匆地赶往操场集合。
<br/>
<br/>        大家哈欠连天地赶到操场，一个个无精打采，东边的太阳才冒出一丁点，天空灰蒙蒙的。
<br/>
<br/>        谁知道，赶到操场后，大家得集体站军姿，等着太阳全部从东边升起来。
<br/>
<br/>        听说，这次的教官是从特种兵部队里请过来的。
<br/>
<br/>        蓝筱蝶不禁跟童筱颖开玩笑地说：“跟教官谈恋爱也不错。”
<br/>
<br/>        童筱颖只笑不语。
<br/>
<br/>        恭如兰却白了蓝筱蝶一眼，碎碎念了一句：“小心你军嫂变寡嫂！”
<br/>
<br/>        站完军姿后，是分班集合跟教官正式见面。
<br/>
<br/>        童筱颖坐在草地上，不停地揉着小腿，肚子也开始咕咕叫了。
<br/>
<br/>        “哇哦——”
<br/>
<br/>        伴随着女生们的一声整齐的惊呼。
<br/>
<br/>        蓝筱蝶激动不已地一边扯了扯恭如兰的衣角，一边摇了摇童筱颖的胳膊，花痴道：“筱颖，你家大表哥原来是特种兵部队里的啊！”
<br/>
<br/>        大表哥？！
<br/>
<br/>        童筱颖一时间还未缓过神来，当她抬头朝前方看去时，顿时惊愕了。
<br/>
<br/>        大、大哥！
<br/>
<br/>        班主任领着傅夜沉走到了大家面前，并温和地介绍道：“同学们，这位是你们的冷教官，在接下来的这十五天时间里，由冷教官陪着大家度过在校的最后十五天。”
<br/>
<br/>        “好帅噢！”
<br/>
<br/>        有几个女生，纷纷发出感慨。
<br/>
<br/>        童筱颖顿时也看呆了眼。
<br/>
<br/>        这是她第二次见傅夜沉穿军装的样子，果然，只有肃穆的军装才适合他那健康系的小麦色肌肤。
<br/>
<br/>        傅夜沉给人一种玉树临风、英姿飒爽的感觉，倒是迷倒了一片女生。
