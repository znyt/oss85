　　此时，慕清霄已经来到冥河岸边，却不知自己与剧情主角们擦肩而过。
<br/>　　察觉到空间的波动，慕清霄侧眼望去，只见隐藏于空间中的天鬼皇已经显出身影，快速来到他的身前。
<br/>　　“阁下，东西拿到手了吗？”
<br/>　　慕清霄微微一笑，道：“已经拿到了，也是时候离开鬼界了。”
<br/>　　闻言，天鬼皇总算是松了口气，只要慕清霄在鬼界一天，他的心都提在嗓子眼上，现在终于能将这位祖宗给送走了。
<br/>　　最重要的是他也完成了对后者的承诺，若是拿不到火灵珠，他相信慕清霄会各种折腾。
<br/>　　毕竟，在这一路上，他完全能够看出，慕清霄似乎对火灵珠是志在必得，这倒是让他有些搞不懂，为何如此执着？
<br/>　　火灵珠虽是天地间的神物，但这种神物又岂是他们能够驾驭的，别说他们，哪怕鬼仙级别的修士恐怕都不行吧？
<br/>　　虽然不明白慕清霄要火灵珠干嘛，但不妨碍两人的交易。
<br/>　　多余的事情，天鬼皇也不问，他非常清楚，有的事情可以问，但有的事情就算是不明白，可就得憋在心里。
<br/>　　“恭喜，既然阁下已经拿到自己想要的东西，本座就送阁下离开吧。”
<br/>　　“走吧。”
<br/>　　说着两人就已经来到冥河岸边，不多时，冥河上就出现一个黑点，黑点越来越近，正是横行在冥河上的‘引路人’。
<br/>　　冥舟渐渐靠岸，黑袍身影依旧矗立在小甲板上，如同木偶一样。
<br/>　　经过先前的接触，慕清霄对此已经是见怪不怪，但引路人在他眼中，依旧是极其神秘的存在。
<br/>　　甚至，能让他联想到一些不得了的大人物。
<br/>　　走上冥舟，慕清霄真是觉得有些奇妙。
<br/>　　冥舟离岸，慕清霄并没有立刻打坐修炼，目光则盯着黑袍身影，他可以肯定，后者与先前的引路人绝对是同一个。
<br/>　　虽然天鬼皇曾说过，引路人有许多，但他就是如此确定。
<br/>　　而且，越是观察下去，慕清霄的心情就越发凝重。
<br/>　　他已经是大乘期四重的修为，在诸天万界中都算是一名颇有实力的修士，可却完全看不透眼前的黑袍人。
<br/>　　甚至，黑袍人给他的感觉，比古藤林中的地仙强者，古藤老人带给他的感觉更沉重，也更加的诡异。
<br/>　　难道后者比地仙更强？
<br/>　　想到此处，慕清霄有些无语，他只是来拿个火灵珠，被这样的人物盯上，也不知道是福是祸。
<br/>　　慕清霄左思右想，也想不出，自己身上究竟有什么东西吸引后者。
<br/>　　想了半天，他也没想出个所以然，于是就直接将烦恼抛之脑后，盘坐在椅子上，静静的修炼起来。
<br/>　　从极乐世界回阴寒域的路程很远，乘坐冥舟总共需要一周的时间，若非上面有禁制，慕清霄都想直接飞回去。
<br/>　　在冥舟上颇为悠闲，平时除了修炼，慕清霄就会煮煮茶，与天鬼皇之间也没有任何好聊的话题。
<br/>　　天鬼皇是鬼族，而他则是人族，就算是修炼上的事情，他们走的道路各不相同，也完全聊不到一起去。
<br/>　　缓行在冥河上，冥舟速度很慢，但对于修士来说，一周的时间实在太过短暂。
<br/>　　可能刚刚进入修炼状态，刚一闭眼，等睁开双眼的时候，一周已经过去。
<br/>　　来到初次登船的地方，黑袍人转身看向两人，做出一个请的姿势，天鬼皇率先下船，慕清霄紧随其后。
<br/>　　慕清霄刚转过身来，岸边哪里还有冥舟与引路人的身影，如此神鬼莫测的手段，令人心惊。
<br/>　　只是一个转身的时间，后者就已经消失不见，而且以慕清霄的实力，居然没有丝毫的察觉到，真是恐怖如斯。
<br/>　　走下冥舟后，天鬼皇早已将引路人忘得一干二净，他现在最想做的，便是将慕清霄送出鬼界。
<br/>　　只见天鬼皇拿出一块黑色玉石捏碎，约莫十多分钟后，阵阵沉闷的马蹄声便在远方响起……
<br/>　　随着马蹄声越来清晰，只见青冥骏已经拉着豪华的车厢，出现在两人的视线中。
<br/>　　……
<br/>　　……
<br/>　　与此同时，极乐世界。
<br/>　　红色的岩浆世界中，景天等人的身躯皆被灵气包裹，站在空间结界面前，露出一副焦急的模样。
<br/>　　望着周围一成不变的景象，景天有些不耐烦。
<br/>　　“白豆腐，怎么还没有打开，你不是吹牛说你们蜀山对破解结界有一手吗？都一个星期了，也不见结界削弱，你到底行不行啊？”
<br/>　　唐雪见被护在结界中，出奇的没有反驳景天的话语。
<br/>　　她实在是受不了了，整整一个星期的时间，居然连结界都没破开。
<br/>　　八十四个时辰，眼前的景色就没有变过，除了岩浆就是岩浆，如果不是龙葵陪她聊天，估计她都要被憋疯了。
<br/>　　此时，站在最前方，徐长卿手中握着仙剑，脸上挂满了冷汗。
<br/>　　“景兄弟，信息有误，火鬼王的实力，恐怕比预测的要高上许多，以我现在的实力，想要破开这结界，恐怕没有希望。”
<br/>　　话语刚落，唐雪见的俏脸上就露出一副生无可恋的模样。
<br/>　　景天脸色顿时变得认真起来：“白豆腐，你别开玩笑了，连你都打不开，那你先前还信誓旦旦的跟本大爷保证。”
<br/>　　深吸了口气，徐长卿将长剑收回剑鞘，微微摇头，道：“除非用掌门给我的符印，方能一击破开封印。”
<br/>　　唐雪见脸色有些难看，道：“那你怎么到现在都不用？”
<br/>　　“唐小姐有所不知，掌门给我的符印，蕴含仙人的绝强一击，若是贸然使用，整个极乐世界都会被轰碎，到时候更会引来鬼界的至强者。”
<br/>　　徐长卿也算是心思缜密，没有贸然动用符印，那可是地仙强者的最强一击。
<br/>　　“说了半天，还不是要本大爷出手，白豆腐，你给我让开，别等会误伤了你。”
<br/>　　不屑的撇了撇嘴，景天大马金刀的拔出背后黑色的魔剑，走到结界前，调动全身灵气，一剑向结界劈去。
<br/>　　在他全力一击下，空间结界轻微波动了一下。
<br/>　　见状，徐长卿脸色一喜，道：“有效果，结界的力量被削弱了，魔剑不愧是魔剑，景兄弟快继续。”
<br/>　　……
<br/>　　……
