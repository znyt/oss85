　　第234章
<br/>　　杨龙停选择拔刀，因为这一刻他已经无路可退。程大雷毕竟小看了这员沙场老将，三十年前，他便已靠着一柄刀扬名长安。
<br/>　　那一年，杨龙停在长安与崔大小姐成亲，崔大小姐的名字，程大雷自然不会听说过。可在当时，不知多少风流才子，王族少年愿意成为她裙下之臣。但崔大小姐最后却出人意料的嫁给一个出身并不好的杨龙停。
<br/>　　成亲那日，诸多弟子齐聚，为的就是给杨龙停点颜色看看，大概想表露的意思就是：这里是长安，这里是天子脚下，你这个外地人是龙给我盘着，是虎给我卧着。
<br/>　　最后肯定是打了起来，杨龙停靠一柄军中制式铁刀会战长安少年。他败一人，饮酒一碗，连战十八人，饮酒十八碗。
<br/>　　酩酊大醉三日，败尽长安少年。
<br/>　　通过那一战，长安接纳了这个外地人，大家以与他结交为荣，到后来杨龙停平步青云，直到成为幽州的实际统治者。
<br/>　　当然，如今不是三十年前，杨龙停已经老了。
<br/>　　可步步紧逼的程大雷，逼出了他求生的潜力，这柄刀并不弱于三十年前那柄。
<br/>　　锵，锵，锵！
<br/>　　连续清脆的兵器撞击声，程大雷刺出一剑，被杨龙停挡住，再刺出一剑，又被挡住，再刺出，再被挡住……
<br/>　　攻，攻得电光火石，守，守得风雨不透。
<br/>　　程大雷从未见过，有人可将一柄刀使得如此神乎其技，他连刺七剑，剑剑被杨龙停挡住。
<br/>　　杨龙停全身都罩着软胄，程大雷能攻击的不过是他的头部，如今杨龙停并没有露出任何破绽。
<br/>　　可他毕竟已经老了，曾经扬名长安的少年，也无可奈何在幽州那样的酷寒之地老去，年与时驰，意与日去，少年变成老贼。
<br/>　　他快要挡不住了。
<br/>　　可是，杨龙停并不用等太久，只要等到相府的人来解围，程大雷今日必死无疑，自己只要把时间拖下去就可以。
<br/>　　程大雷也明白这一点，所以他的攻击越发凌厉，他在争时间。
<br/>　　一个在争时间，一个在拖时间，二人都是争分夺秒。
<br/>　　“呔，贼人休狂！”
<br/>　　一柄枪，一条棍攻向程大雷，枪来自将军府一名布甲军人，棍来自于相府一位长袍书生。
<br/>　　二人一出手，程大雷便感觉到巨大的压力，二人任何一人的实力都不弱于程大雷。
<br/>　　“这下贼人绝对逃不掉。”有人欢呼道：“将军府和相府的两大高手同时出手了。”
<br/>　　“该不会是那将军府的铁面神时天寒和相府的哑巴书生周声声吧。”
<br/>　　“嘿，不是他们还能是谁，两大高手同时对敌，这可是从未有过的事情。”
<br/>　　铁面神时天寒，擅长铁枪，十万城防军教头。
<br/>　　哑巴书生周声声，他并不是一个哑巴，之所以被人称作哑巴书生，是因为此人惜字如金，每言必中。
<br/>　　而同时他也是一名了不得的用棍高手。
<br/>　　枪快，棍疾，程大雷必须退，杨龙停被隔开。
<br/>　　程大雷选择往后退，他的身法很快，禁卫军以他为中心调动。
<br/>　　“莫要放箭，生擒此人。”看台上的李乐天下令，明帝离开后，他已经成为发号施令的人。
<br/>　　程大雷面临的压力很大，只能一退再退，围上他的高手更多，可给程大雷带来压力的，还要数周声声与时天寒。
<br/>　　程大雷很危险。
<br/>　　从一群士子中闯出一人，他撞倒兵器架，拣了一柄铁枪冲过来，替程大雷接过时天寒。
<br/>　　是林冲，他是和林少羽一起来告御状，当林少羽被抓走时，他没有出手，因为二人今日都存了死志。
<br/>　　他一枪将时天寒的枪挑开，二人相对而立，互相都认识。
<br/>　　“林教头……”时天寒顿了顿：“十年前我败你一招，今日我们再分个高低。”
<br/>　　“今日你依旧败。”
<br/>　　林冲没有废话，提着枪冲了过去，时天寒手腕一抖，一条铁枪王若化作青蛇，向林冲叼去。
<br/>　　“嘿，还差得远呢。”
<br/>　　林冲身体一转，借着旋转的力量，狠狠抽向时天寒。
<br/>　　程大雷现在的情况依旧不太乐观，关键是这周声声也是个猛人，一条金箍囚龙棒如同附骨之蛆，死缠着程大雷不放。
<br/>　　而程大雷看到，杨龙停正在禁卫军的护送下逃离。
<br/>　　周声声一棍打在程大雷背心，程大雷身子一跌，就势在地上一滚。
<br/>　　周声声步步紧逼，手中的棍砸向程大雷的小腿，以他的力量，能够一棍将程大雷腿骨敲碎。
<br/>　　程大雷此刻已接近兵器架，兵器架刚才被林冲撞倒，此刻各种兵器散落一地。
<br/>　　程大雷抓起一柄宣花斧，往后一甩，将周声声逼退。借着这个机会，程大雷已经站起身，挥斧砸向周声声，另一边还剑入鞘，整个动作一气呵成。
<br/>　　程大雷本身就是用斧头的，至于剑，不过是附加技能。
<br/>　　周声声下意识用囚龙棍架程大雷的兵器，可他忘了程大雷此刻用的不是剑，而是斧头。
<br/>　　囚龙棍直接被程大雷砍成两段，周声声惊出一身冷汗，下意识往后退一步。
<br/>　　程大雷已经走过，或者说飞过，他纵身而起，脚尖点着禁卫军的脑袋向前，再往前行便是杨龙停。
<br/>　　此刻他在禁卫军的护送下，向演武场的小门逃去。
<br/>　　“劈脑袋啦。”
<br/>　　程大雷从天而降，杨龙停猛地回头，挥手砍出一刀。
<br/>　　好刀法！
<br/>　　依旧防得密不透风，滴水不漏，将全身要害罩在刀光之下，便是整个长安最好的剑客在此，也未必攻得破他的刀围，即便攻得破刀围，也未必能刺破他身上的软胄。
<br/>　　可程大雷没有用剑，他用的是斧头，宣花大斧。
<br/>　　大斧从天而降，破开杨龙停的刀围，破开软胄，破开肌肤，破开骨骼和腑脏……
<br/>　　一斧两段，杨龙停的身体像一堆烂肉般倒下，肚子里的血水淌了出来。
<br/>　　李乐天猛地站起，竟真的被此人杀死了杨龙停。
<br/>　　“抓住他，一定要抓住他。”李乐天大吼。
<br/>　　在皇城内，可以说当着整个帝国的面，帝国战神竟被人杀了。
<br/>　　帝国今天丢脸够大了。
<br/>　　程大雷长出一口气，扛在肩上整整一年的压力，在今日才完全卸去。
<br/>　　神清气爽。
