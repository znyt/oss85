　　九宫传人，一直都是苏莫斟酌选取，选择适合的人，还从来没有人自荐的情况。
<br/>　　随即，苏莫面露沉吟之色，他知道龙腾定然是为了传承，所以才会要求做九宫传人。
<br/>　　不过，这也没有什么，毕竟，强大的传承，任何人都会心动。
<br/>　　“龙兄，目前九大宫之中，还有雷霆宫、厚土宫、金煌宫没有传人，你的天赋只要合适，我可以考虑！“苏莫笑着说道。
<br/>　　他没有拒绝对方，他对于龙腾感觉还是不错的，对方天赋若是真的合适，倒也可以做一宫传人。
<br/>　　“我的天赋自然不用多说，只不过，你应该需要某种特殊的天赋吧？”龙腾沉声问道。
<br/>　　“的确，雷霆宫、厚土宫、金煌宫，分别需要传承者需要雷属性、土属性和金属性天赋！”苏莫点头道。
<br/>　　“雷属性、土属性和金属性？”
<br/>　　龙腾闻言面上露出了笑容，道：“苏兄，我拥有五爪金龙的血脉，虽然非常的淡薄，但体内的金属性天赋非常不错！”
<br/>　　“哦，金属性天赋？”苏莫闻言一笑，道：“若是你的金属性天赋，真的很高，那就让你做金煌宫的传人。”
<br/>　　“哈哈！你可以测试一番！”龙腾大笑道。
<br/>　　苏莫点了点头，随即意念延伸出苍穹神殿之外，不多时便找到了金一，让对方前来。
<br/>　　大殿中，东海龙王面带微笑，他自然也是希望龙腾能成为金煌宫的传人，如此的话必能得到惊天动地的大传承。
<br/>　　“见过宫主！”
<br/>　　不多时，金一来到了苍穹神殿，立刻向苏莫躬身一拜。
<br/>　　“金一前辈，你为龙兄测试一番金属性的天赋！”苏莫一指龙腾，对金一说道。
<br/>　　“是，宫主！”
<br/>　　金一点头，随即便开始为龙腾测试金属性的天赋。
<br/>　　金一取出了一块测试石，这测试石和上次水一为夕儿测试水属性天赋，所用的测试石相仿。
<br/>　　龙腾的手掌紧握测试石，霎时之间，测试石上金光耀眼，赫然爆发出了九道夺目的金色光环。
<br/>　　呃~
<br/>　　苏莫见此顿时瞪大了眼睛，心中有些无语，这龙腾的金属性天赋，居然如此之高。
<br/>　　上次夕儿测试水属性天赋，也才发出六道光环而已，而龙腾的金属性天赋，却是九道光环！
<br/>　　“禀宫主，龙腾小友的金属性天赋非常高，仅次于金属性灵体！”金一见此也是面带惊讶之色，立刻向苏莫禀报。
<br/>　　“不错！”
<br/>　　苏莫微笑点头，道：“龙兄，从今以后你就是金煌宫传人，将来的金煌宫宫主！”
<br/>　　“哈哈！”
<br/>　　龙腾闻言大喜，急忙向苏莫抱拳，道：“多谢苏兄……不……应该是多谢宫主！”
<br/>　　“龙兄不必客气！”
<br/>　　苏莫摆了摆手，笑道：“不过，龙兄可要明白，成为金煌宫的传人，就是苍穹神宫之人，必须要效忠苍穹神宫！”
<br/>　　苏莫言语随意，他只是要提醒对方一番，免得对方的了好处之后，不履行职责。
<br/>　　“哈哈！宫主放心，这点觉悟我龙腾还是知道的！”龙腾大笑道。
<br/>　　“那就好！”
<br/>　　苏莫微微颌首，随即对金一吩咐道：“金一前辈，你带龙兄去金煌宫吧！接受金煌大帝的传承！”
<br/>　　“是！宫主！”金一应声，随即便带着龙腾离开了苍穹神殿。
<br/>　　“苏莫小友，本王代小儿多谢了！”待龙腾离开之后，东海龙王向苏莫抱拳道谢。
<br/>　　成为金煌宫的传人，得到金煌大帝的传承，这对于龙腾来说，绝对是一生中最大的机遇，所以，东海龙王亦是为龙腾高兴。
<br/>　　“龙王前辈不必客气！”苏莫摆了摆手。
<br/>　　随即，苏莫又和几人闲聊了几句，便起身准备前去修炼。
<br/>　　“对了！”
<br/>　　蓦地，苏莫又想到了一些事情，对三人道：“三位，黄泉魔宗据点众多，三位若是有时间的话，可以先去将其全部拔除！”
<br/>　　“苏莫，这个你就无需操心了，一个月之内，保证全部灰飞烟灭。”东海龙王笑道。
<br/>　　“那就好！”
<br/>　　苏莫点头，随即又道：“不过，黄泉魔宗在东南西北四大洲都有一大魔殿，除了东洲的血罗殿已经被我灭了之外，其它三大洲应该还在。”
<br/>　　“西洲、北洲和南洲，分别有森罗殿、幽罗殿和煞罗殿，这三大魔殿实力很弱，殿主不过是低阶武皇而已，不足为惧！”东海龙王笑道。
<br/>　　“也罢，我去走一趟吧，将这三大魔殿全部诛灭！”帝释叹了口气道，帝玄宫被灭，他不大开杀戒一番，心中的怒气实在难消。
<br/>　　“嗯！那就交给龙王前辈和岳父了！”
<br/>　　苏莫点了点头，随即又商谈了几句之后，他便离开了苍穹神殿。
<br/>　　而后，帝释和东海龙王两人，则是离开了苍穹神宫。
<br/>　　至于海明江则是脱不开身了，他需要召集各地天涯海阁分阁之人，向苍穹神宫汇聚。
<br/>　　这一段时间以来，已经有数十处分阁之人，来到了苍穹神宫。
<br/>　　……
<br/>　　苏莫离开苍穹神殿之后，并没有去厚土塔，而是径直回到了自己居住的宫殿，回到了自己的新房。
<br/>　　他和夕儿刚刚成婚，就分开了近十日，早已想念不已，自然要来看看。
<br/>　　不过，房间中空空荡荡，夕儿并不在房间中。
<br/>　　而后，苏莫立刻又离开了宫殿，向水月宫而去。
<br/>　　不过，让苏莫郁闷的是，到达水月宫之后，水一告知夕儿正在闭关，不宜打扰，他只能无奈的离开了水月宫。
<br/>　　对此，苏莫也能理解，现在整个苍穹神宫都在拼命修炼，夕儿肯定也不会闲着。
<br/>　　而且，夕儿刚得到水月宫传承不久，更需要闭关消化所获的传承。
<br/>　　无奈的离开水月宫之后，苏莫来到了厚土塔，再次踏上了第九层。
<br/>　　“第十层吧！”
<br/>　　站在通向第十层的入口，苏莫深吸口气，决定踏上第十层。
<br/>　　如今第九层两万五千多倍的重力，对于他来说已经没有太大的压力了，淬炼玄力的效果自然也会差了一些，所以他要踏上第十层。
<br/>　　虽然第十层的重力将会达到五万倍出头，但以他目前的实力，应该勉强能撑的住了。
<br/>　　而且，为了能尽快突破修为，踏入武皇境四重境界，就算撑不住，苏莫也要踏入第十层。
<br/>　　咚！
<br/>　　苏莫刚刚踏上第十层，脚步乍一放下，便犹如一座大山般狠狠的砸下，发出沉闷的声响。
<br/>　　五万多倍的恐怖重力，简直如同十数座万丈大山一般，狠狠的镇压在了他的身上。
<br/>　　“恐怖！”苏莫心中暗暗震惊，以他的实力，都被压的全身骨骼咯吱作响，几乎直不起来脊梁。
<br/>　　而且，强大无比的重力压迫，让他全身的血液和玄力几乎停止了流动。
<br/>　　这股重力之强，若是能用来战斗，苏莫有信心轻易斩杀准帝强者。
<br/>　　可惜！这厚土塔只能修炼，不能用于交战。
