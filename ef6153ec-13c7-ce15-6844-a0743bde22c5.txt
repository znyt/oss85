        &lt;h3&gt;第三卷第三百九十七章  两地相隔千万里&lt;/h3&gt;
<br/>
<br/>        凌辰乐滋滋的在杭州配合搞活动，却接到了山西调查他公司的事情。
<br/>
<br/>        凌辰看着邮件，心里在想着谷正雄好本事啊，不，应该是宁少天好本事，竟然将手伸到他的老家了，这是怀疑他凌辰造假呢还是给他凌辰下马威？
<br/>
<br/>        不过身正不怕影子斜，凌辰是不怕查的，现在的这种情况闹得越大越好，而且有正规的部门出面比他们出来作这件事情还好。
<br/>
<br/>        “嗨，以后还得好好感谢感谢我的老丈人，要不是他，我这项工作开展起来怕是万分艰难啊！不行，我得给雅儿打个电话汇报一下行程。”
<br/>
<br/>        宁雅已经从陈子昂的公司离职了，全心全力投入到了自己的事业中，她在和胡婷一起看着装修得焕然一新的大楼，这里所有的设备都已经到位，人员也基本到位，韩国的那位专家再有两天就会上岗。
<br/>
<br/>        “啊，小雅，我们这是真的吗？”胡婷看着来来去去的工作人员，她有些不相信自己的眼睛。
<br/>
<br/>        “你说呢？”宁雅笑着说，她的心里还是很激动的，这一段时间太累了，她就没有好好的休息过一天。
<br/>
<br/>        “我感觉我在做梦，我们这个速度也太快了吧？”胡婷掐了一下宁雅的脸“小雅疼不疼？”
<br/>
<br/>        “我又没有做梦，你掐我干什么？”
<br/>
<br/>        “对，我应该掐狸猫，臭狸猫坏狸猫，他又闭关了！”
<br/>
<br/>        “得了，三句话不离你的狸猫，你赶紧滚回你狸猫的身边吧！你们不造人了？”
<br/>
<br/>        “他说他要放大招，爆更，没有时间造人。”
<br/>
<br/>        “你家狸猫还真敬业，连造人都放弃了。”
<br/>
<br/>        “他说他要努力一段时间，一鼓作气将这本书写完，不能太监或者乱尾，要不然对不起他的那些粉丝们，他只要想着造人他就没有心思写书了。”胡婷有些心塞，感觉狸猫的粉丝在狸猫的心中的地位比她都高。
<br/>
<br/>        “他闭关得多久啊。”
<br/>
<br/>        “谁知道啊，一两个月亦或者是半年？”胡婷心不在焉的东张西望。
<br/>
<br/>        “刚好，你还可以继续在这里干活了，你要是真造人呢，我就不敢用你了，还得把你当祖宗一样供着。”宁雅也捏了一下胡婷有点婴儿肥的脸。
<br/>
<br/>        “别捏，这肉肉本来就多，你再捏就更多了，我这肉都是你捏起来的。”胡婷拍掉了宁雅的手。
<br/>
<br/>        宁雅哈哈哈大笑：“你这浑身的肉肉都是我捏起来的可还行？”
<br/>
<br/>        “不是你捏的还能是谁捏的？动不动就捏我的脸，从上学到现在。”胡婷也笑着，她们是骨灰级别的人物，想怎么闹就怎么闹，不过说真的，她还是希望自己和宁雅一起做一番事情。
<br/>
<br/>        张黎有自己的事业，还有自己的爱好，反而是她一天在家里闲得没有事情可干，总是爱胡思乱想，只要张黎回来晚了她就心惊肉跳，东想西想，搞得她自己都有些神经了，好的是宁雅要做美容，现在他们的定位比之前不知道高了多少个档次，而她这一段时间忙忙碌碌的，找回了自信，心态发生了很多变化，回去见了张黎似乎俩个人之间有了一些话题。
<br/>
<br/>        “得了，上学的时候我们还在一起，你结婚之后和我在一起了吗？”
<br/>
<br/>        “怎么没有在一起，这一段时间不是经常和你在一起吗？你个小没良心的，小心以后嫁不出去。”
<br/>
<br/>        “嫁不出去就不嫁了呗，反正一个人也过得很好！”
<br/>
<br/>        “你这话要是被凌辰听见了不知道伤心到何种程度。”
<br/>
<br/>        “关他什么事情。”宁雅提起凌辰心里划过一丝暖流，这么多年俩个人马拉松式的恋爱终于被打破了，凌辰正式向她告白了。
<br/>
<br/>        “关他什么事？切!看你那满面春色遮不住的样子，还好意思说这样的话？”
<br/>
<br/>        “有吗？”宁雅伸手抚上了自己的脸。
<br/>
<br/>        “小雅，你们俩个到什么程度了？”胡婷好奇的盯着宁雅的脸，似乎想从宁雅的脸上盯出答案来。
<br/>
<br/>        “拉手的程度！”宁雅微微一笑。
<br/>
<br/>        “小雅！你真的是要气死我啊？”
<br/>
<br/>        “别了，不要讨论这件事情了，我们要讨论明天的工作该怎么进行，还有下一步的计划，胡婷，我告诉你，如果我们的效益好的话，我们就向外扩充，在别的省会城市开分院。”宁雅立马将胡婷的思路拉回到工作上。
<br/>
<br/>        “先把手头上的事情做好吧，我和一家卫校已经联系过了，到时候让他们定点给我们输送一些医护人员。”胡婷也毫不含糊。
<br/>
<br/>        “金教授再有半月就会到我们的机构来，另外韩国的文教授提前来了，后天就飞过来，我们一起去机场接他，大家先磨合一段时间，希望能留着文教授，让他一直在我们这里，最好是还能给我们再引进一些专业的人才。”
<br/>
<br/>        “放心吧，我们会越来越好的。”
<br/>
<br/>        俩个人讨论着工作，凌辰将电话打过来了。
<br/>
<br/>        “小雅，你还在忙吗？要注意身体，工作固然重要，身体更重要，没有工作你还有我。”凌辰心情很好，连环炮似的滴滴不停。
<br/>
<br/>        “我说凌辰，没有工作你还有我？就光说得好听，小雅在忙的时候你在哪里？怎么就不见你的人呢？”胡婷和宁雅坐在一起，凌辰说的话被她听到了，抢过宁雅的电话就噼里啪啦酱。
<br/>
<br/>        “胡婷，怎么是你？”凌辰一脸黑线。
<br/>
<br/>        “怎么就不会是我，这一段时间是我一直陪着小雅，你还好意思说有你？”
<br/>
<br/>        “你不陪着你的狸猫，你陪着小雅干什么？”
<br/>
<br/>        “我说你就是嘴上说得好听吧？小雅已经辞职了，我们现在在开展新的事情，这个你都不知道还好意思在这里哔哔哔？”
<br/>
<br/>        宁雅在旁边已经是捂着嘴笑个不停，这个胡婷真的是太厉害了。
<br/>
<br/>        “我当然知道小雅辞职了啊，我也知道小雅在开展新事业啊，你以为就你知道啊，我还是股东呢。”凌辰也毫不犹豫的怼回去了。
<br/>
<br/>        “你说什么？你还是股东？谁让你参加的？我怎么不知道？”
<br/>
<br/>        “什么都让你知道了我还是凌辰？赶紧的将电话给小雅，你有没有点眼色？我是和小雅谈情说爱又不是和你谈情说爱，你找你的狸猫去。”
<br/>
<br/>        “凌辰，算你狠！”
<br/>
<br/>        “小雅，我在杭州。”凌辰轻声对宁雅说：“原本要去北京看你的，但是杭州这边发生了一些事情，我得配合这边有些部门做一些工作，过几天我就去北京看你。”
<br/>
<br/>        “没事，你忙你的，你的款已经都收到了，陈子昂全部打到了我的工资卡里。”
<br/>
<br/>        “你就拿着好好的干你的新事业吧，这一段时间有胡婷陪着你我也就放心了很多。小雅，我们一起努力！”
<br/>
<br/>        “好，我们一起努力！”
<br/>
<br/>        相爱的人无需多说，一个字亦或者一个眼神都能知道彼此再想什么，凌辰没有说他在杭州干什么，宁雅联想到前几天的新闻也知道是怎么回事，他们需要更加努力！
<br/>
<br/>        两地相隔千万里，只言片语寄相思，待到春暖花开时，携手同游人间路！
