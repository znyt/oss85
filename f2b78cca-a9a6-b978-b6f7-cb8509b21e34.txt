　　第560章
<br/>　　华正则并非故意难为燕不归。当然，他也没有难为燕不归的资格。也并非嫌弃燕不归是个戎族，医者父母心，在他眼里，只要是条生命，能救还是要救的。
<br/>　　可黑贞现在的情况已没办法救了。趁着这个时间，你与她说些话不好么，又何必折腾一个必死之人。
<br/>　　黑贞的命对燕不归来说很是重要，但对华正则来说没那么重要。他见过的死人是多的，作为一个医者，华正则心极热，但心肠极硬。
<br/>　　也就是不愿燕不归苦苦哀求，他才肯动手一试。可试个什么劲儿么，明明知道她已要死了。
<br/>　　华正则深吸一口气，打开了医囊。一连排利刃金针，明闪闪，冷森森，有长有短，有粗有细。
<br/>　　燕不归发现，持刀在手的华正则气质截然一变，有了几分镇定自若，志在必得。
<br/>　　草原上有名的高手，二探万卷山都丝毫不惧的大汉，此刻心情竟紧张起来。忐忑不安的立在华正则身边，低声道：“大夫？”
<br/>　　“住口。”
<br/>　　华正则喝了一声，用烈酒消毒，开始给黑贞治病。他的思路很简单，先用利刃割开伤口，再用金针插入黑贞肺腑处，排出她体内淤血。当然，说说容易，想要做到极难。金针缓一缓，通不了患处，错半寸，却又可能伤到内脏。
<br/>　　这需要极端了解人体构造，同样用针的手也需要极稳。
<br/>　　不过，这两点对华正则来说并不算什么。能作为华氏一族的代表来到书院，也足见他的医术。
<br/>　　随着一根根金针的扎入，从黑贞身体内排出黑血。燕不归睁大眼睛，看到黑贞一张脸正慢慢由黑变红。
<br/>　　他不敢说话，呼吸都像是停住。程大雷窝在屋顶上，偷偷瞧着这一幕，心道：啊呀，这世上当真有活死人，肉白骨的医术。
<br/>　　突然，只听得燕不归发出一声大叫，口中不停唤着黑贞的名字。猛地抬起头，恶狠狠盯着华正则：“我让你救她，你为何杀了她。”
<br/>　　“这个……我刚才已经给你说了，她的命已经救不活，你让我试试，试试也是一样的么。”
<br/>　　黑贞已经气绝，歪躺在燕不归怀里。任燕不归千呼万唤，都没办法再给他任何回应。而华正则一张脸在此刻的燕不归看来，当然是分外可怖。
<br/>　　“庸医，你与我纳命来。”
<br/>　　华正则一激灵，就知道这恶人不会与自己善罢甘休。就算治好了黑贞，他也未必会放过自己。何况还治死了……呸，她本来就要死，怎么会是自己治死的。
<br/>　　但很显然，燕不归不会这么想。
<br/>　　华正则身体一跌，手中金针撒出，打的是燕不归身上三处大穴。
<br/>　　他是个医者，只救人不杀人。但不杀人，不代表不会杀人。一手落花金针，专打人周身大穴。
<br/>　　当然，他的手段也就这些了。凭几根金针，明显不是燕不归的对手。甩出金针拉开的距离，很快就被燕不归跨越，眼看大剑砸来，自己一条命就要交待在此地。
<br/>　　程大雷怎么能看他死，身子跃起又狠狠砸下，整个人从天而降，还伴随着一声大喝。
<br/>　　“呔，过分了啊。”
<br/>　　一剑从天而降，拦住了燕不归的重剑，生生隔在燕不归和华正则中间。
<br/>　　“高手兄……”华正则当然不知道这位从天而降的仁兄是谁，当然他也不关心这个，只是口中高呼：“救我。”
<br/>　　“我救你，你帮我救一个人。”
<br/>　　“好。”华正则没有半点拖泥带水。
<br/>　　燕不归死死盯着程大雷，他当然没见过程大雷，可能一剑将自己逼退，本事又怎会简单。
<br/>　　咚。
<br/>　　好巧不巧，程大雷坠下来时，挂下来一根横梁，不偏不倚正好砸在黑贞的尸体上。
<br/>　　呃……
<br/>　　程大雷满脑门子黑线，尴尬道：“那个……这房子当真该修了啊……哈”
<br/>　　燕不归双目沁血，恨不得要将程大雷千刀万剐，他拎着重剑就砍了过来。
<br/>　　“狗贼，去死。”
<br/>　　程大雷抖开匹夫剑，剑光如雨，在面前成一片风雨不透的光幕。燕不归一阵阵攻击，都被程大雷挡住了。
<br/>　　这令燕不归更加愤怒，坚定要杀程大雷的决心。但带给程大雷的却是胆战心惊。
<br/>　　所谓一人拼命，十人难敌。燕不归明显是已经走火入魔，黑贞死了，他感觉全天下人都该给黑贞陪葬。至于程大雷，他明显有该死的理由。
<br/>　　好端端的，自己和一个疯子拼命做什么。
<br/>　　程大雷后退一步，拎起华正则丢出屋子，自己也赶忙逃了出去，拎起他就往外逃……
<br/>　　燕不归像疯了一般跟在身后，挥舞着大剑要砍二人。最后，程大雷找了个桥底，甩掉了燕不归。
<br/>　　疯子力气往往是大的，只不过脑袋有些不好使。所以程大雷轻而易举甩掉了燕不归，他和华正则夺在桥下，呼哧呼哧喘着粗气。
<br/>　　好不容易平复下来呼吸，华正则双手一抱拳：“东海华正则，多谢阁下出手相救。”
<br/>　　“知道了，知道了。等这件事了结，随我去一救一个人。”
<br/>　　华正则一头雾水，看程大雷的模样像是很早就知道自己的名字。难道自己现在名气这么大么？
<br/>　　“那个，敢问阁下是？”
<br/>　　“欧巴。”
<br/>　　“欧公子……”华正则眼前一亮：“就是那一路闯山，连败书院七贤十二秀的欧公子。”
<br/>　　“呦，我的名气这么大。”程大雷摸了摸鼻子，觉得这华正则倒是个趣人：“你什么情况，怎么把人给治死了？”
<br/>　　“什么叫我把她治死了，明明是她已快死了。”
<br/>　　华正则大呼小叫，也就将一些程大雷没听到的话说了一遍。
<br/>　　程大雷摸了摸鼻子，除了华正则，他另外关心的一件事，便是山水经的下落。
<br/>　　“书不在燕不归手中么。”
<br/>　　他呐呐道了一声，如果是燕不归的手下将书盗走，此刻琅琊城封锁，他们应当还没有出城。可如此大一座城，想要找几个人，不亚于海底捞针。
<br/>　　“那个……天快亮了，欧公子随我一起回书院吧。”
<br/>　　“好。我们路上先吃些东西，吃完饭回去。”
<br/>(本章完)
