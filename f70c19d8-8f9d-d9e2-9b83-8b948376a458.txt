        “我怎么就过分了？实话实说，不对吗？”童筱颖反问。
<br/>
<br/>        她确实是来看裴金澈精神失常时，会是个什么样子。
<br/>
<br/>        因为，她很想知道，日后，她自己是不是也会变成他这个样子，然后六亲不认地去伤害自己身边的人？
<br/>
<br/>        夏玄灵正在气头上时，裴金澈拍了拍夏玄灵的肩头，示意她不要太在意。
<br/>
<br/>        “你一个人跑出来，阿沉知道吗？”裴金澈看着童筱颖，十分淡定地问。
<br/>
<br/>        童筱颖微微耸了耸肩，双手一摊，随即又嬉皮笑脸地说道：“他今天一天都不见人影，我也没主动跟他联系。不过，我去了哪里，见了什么人。他应该都知道。因为，那个小连医生在我身上安了纳米监听器。”
<br/>
<br/>        “看样子，阿沉对你，也不是百分之百的信任。”裴金澈突然冷嘲热讽道。
<br/>
<br/>        童筱颖倒是无所谓地又坐回沙发上，然后抬起头来，看了一眼壁钟上的时间，问道：“你一般几点钟开始精神失常啊？”
<br/>
<br/>        “童筱颖，你……”
<br/>
<br/>        “我每天晚上八点钟开始，会变成另一个连我自己都不知道自己在做什么的人。”裴金澈直接打断了夏玄灵的话，向童筱颖坦白。
<br/>
<br/>        夏玄灵不明白，裴金澈为什么要对童筱颖表现得如此有耐心。
<br/>
<br/>        明明这女人是来找茬的！
<br/>
<br/>        “那你一般什么时候又能恢复正常？”童筱颖不痛不痒地接着问道。
<br/>
<br/>        夏玄灵忍无可忍地怒吼：“童筱颖！你有完没完？”
<br/>
<br/>        “睡一觉醒来后。”裴金澈面对童筱颖的问题倒是回答得十分心平气和，顿了顿后，他才对夏玄灵说道，“阿灵，去做晚饭吧！算上筱颖的份。”
<br/>
<br/>        夏玄灵瞥了童筱颖一眼，闷哼了声，只身去了厨房。
<br/>
<br/>        “你和她结婚了？”童筱颖看着夏玄灵的背影，若有所思地问。
<br/>
<br/>        裴金澈随之在沙发上坐了下来，拿起茶几上的财经杂志，一边翻看，一边回答道：“没有。”
<br/>
<br/>        “像我们这种，在实验室里培养出来的人，是不是不能生育？”童筱颖接二连三地发问。
<br/>
<br/>        但被问到这个问题时，裴金澈却一脸诧异的看着童筱颖。
<br/>
<br/>        “傅家的长辈，还有小连医生，他们说，我不孕不育。”童筱颖微微垂下眼帘，抬起手来，抚住了自己的肚子，黯然神伤地叹气。
<br/>
<br/>        裴金澈不以为然地笑了笑：“你都没跟阿沉试过，怎么就知道不能怀？”
<br/>
<br/>        “你认为我可以怀上宝宝？”童筱颖顿时希冀地看向裴金澈。
<br/>
<br/>        裴金澈点了点头：“至少我那方面没问题，我想，你应该也没问题。”
<br/>
<br/>        “那你和夏玄灵打算什么时候要孩子？”童筱颖忍不住八卦了起来。
<br/>
<br/>        裴金澈下意识地看向厨房，剑眉微微蹙起：“至少得等我正常了才行。”
<br/>
<br/>        童筱颖顿时沉默了。
<br/>
<br/>        吃过晚饭后，裴金澈在书房里的一把特制的钢椅上坐下，夏玄灵则替他将手腕和脚腕锁上铁链。
<br/>
<br/>        童筱颖只能站在一旁看着，什么忙也帮不上。
<br/>
<br/>        待夏玄灵将裴金澈绑牢固后，便对童筱颖说道：“我们出去吧！不要在这里影响到他。”
<br/>
<br/>        “噢！”童筱颖疑惑不解地看了裴金澈一眼。
<br/>
<br/>        只见他十分淡定地坐在椅子上闭目养神。
<br/>
<br/>        童筱颖跟着夏玄灵离开了书房。
<br/>
<br/>        夏玄灵关上书房的房门时，还拿着钥匙将房门给反锁上。
<br/>
<br/>        就这样，童筱颖陪着夏玄灵坐在客厅的沙发上。
<br/>
<br/>        直到八点来临，书房里传来了裴金澈的哀嚎声。
<br/>
<br/>        夏玄灵听得揪心，整个身子蜷缩在沙发上，她一手抱着双膝，一手抵在嘴边，贝齿咬住右手的食指关节，紧蹙着眉头。
<br/>
<br/>        童筱颖按捺不住地从沙发上起身，拿起茶几上的那把钥匙直接去了书房。
<br/>
<br/>        “你想做什么？”夏玄灵吓得从沙发上跳下来，疾步追上童筱颖后，伸手拦在了她的面前。
<br/>
<br/>        “我得知道，他到底变成了什么样子？”童筱颖挪开夏玄灵的手，大步流星地走到书房门口，直接将房门给打开了。
<br/>
<br/>        此时，裴金澈像是发疯了般，用力拉扯着四肢上的铁链。
<br/>
<br/>        就在他抬头向房门口这边望来的时候，童筱颖清晰地看到，裴金澈的双眼的眼白里布满了血丝。
<br/>
<br/>        他张牙舞爪地向她们这个方向扑腾。
<br/>
<br/>        夏玄灵看到这一幕，鼻子一酸，连忙捂住嘴巴，忍不住抽泣了起来。
<br/>
<br/>        童筱颖看懵了后，回过神来时，双腿不听使唤地向裴金澈走去。
<br/>
<br/>        夏玄灵见状，连忙将童筱颖拉住：“不要靠近他！他会攻击你！”
<br/>
<br/>        “他这种状态，要持续多久？”童筱颖凝眉关问。
