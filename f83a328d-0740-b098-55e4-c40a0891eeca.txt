　　第30章
<br/>　　程大雷彻底崩溃，好吧，也许文科宅和理科宅是同一种生物，我们都不需要女人。
<br/>　　“大当家，不好了，小灵中毒了！”
<br/>　　正这个时候，徐神机急匆匆跑过来。程大雷也吓了一跳，道：“快带我去看看。”
<br/>　　一边走程大雷一边问：“她是怎么会事，怎么中毒了？”
<br/>　　“她在后山采蘑菇，看到地上长着红果子，就吃了一个，结果嘴里发烫，浑身冒汗……哎，这个傻闺女，就是嘴馋。”徐神机急的手足无措。
<br/>　　走到徐灵儿房间，黄三元正守着她，也十分着急。程大雷看到徐灵儿躺在床上，两腮通红，眼睛不停落泪。
<br/>　　“大当家，我……我怕是不行了……”
<br/>　　“灵儿，别怕，我一定想办法治好你。”
<br/>　　说是这么说，可程大雷对医术是一窍不通，整个山寨也没有任何人懂医术。现在连徐灵儿中的是什么毒都不知道。
<br/>　　“她吃了什么毒果？”
<br/>　　“就是这种，这个傻闺女，傻闺女……”徐神机捧着菜篮子：“山里的东西怎么能随便吃！”
<br/>　　程大雷接过菜篮，看到里面有一些野菜，蘑菇，还有……辣椒。
<br/>　　“她就是吃了这东西中毒的？”程大雷举着一枚辣椒。
<br/>　　“是啊，剧毒无比！”徐神机道。
<br/>　　程大雷哭笑不得，看看徐神机、黄三元，再看看炕上的徐灵儿。忽然心中一动，问：“你们是不是都不认识这是什么？”
<br/>　　三人同时摇摇头，徐神机道：“难道大当家认识这毒物？”
<br/>　　“这哪里是毒物，这叫辣椒，是一种蔬菜，也可以看做调味品。”
<br/>　　“可灵儿现在这个样子？”
<br/>　　“只是太辣，她又被吓着了，多喝点水就好了。”
<br/>　　程大雷无所谓的说着，忽然一顿，脑海中宛若有一道闪电劈过，劈得他四肢发颤。
<br/>　　徐灵儿，徐神机没见过辣椒，这一点程大雷不觉得奇怪，毕竟二人没什么见识。可黄三元曾经是韩虎踞的管家，这就有些奇怪了。那是不是就代表着……帝国根本没有辣椒。
<br/>　　历史上辣椒是什么时候传入国内的，明末，清代……程大雷抱着脑袋想，可他匮乏的历史知识无法告诉他确切时间，只知道是一个很晚很晚的时代。在那之前，华夏根本没有辣椒，人们用生姜，芥末，茱萸调味。
<br/>　　如今这个世界，历史有些混乱，程大雷不知道帝国可以和历史上哪个时间段对应。
<br/>　　系统奖励了自己一亩成熟辣椒，自己本来没放在心里，连找出来的兴趣都没有。可如果这个世界根本就没有辣椒，那这一亩辣椒所代表的意义将完全不同。
<br/>　　要知道，在程大雷前世，辣椒可是席卷全国，太多人无辣不欢。而在这个世界，只有自己掌握辣椒，有句话说得好……物依稀为贵呐。
<br/>　　程大雷心情激动起来，迅速拿着一枚辣椒跑向苏樱房间，他要求证这件事。
<br/>　　“你知道这是什么嘛？”
<br/>　　苏樱皱起眉头：“山果？”
<br/>　　程大雷一颗心砰砰跳动，就连苏樱都没见过辣椒，几乎已经可以确认这个世界尚无辣椒这种作物。记得历史上，辣椒是很晚才传入，却在极短时间席卷全国。而因为系统奖励，让自己提前上千年拥有这种作物，而且是独家。
<br/>　　独家就意味着垄断，垄断就意味着财富。
<br/>　　程大雷心中隐隐觉得，辣椒或许将成为自己挖掘第一桶金的资本。
<br/>　　“你觉得这个如果卖的话，能卖到一个好价钱吗？”程大雷问苏樱，想要参考对方的意见。
<br/>　　“什么味道，甜的？”
<br/>　　“不，是辣的。”
<br/>　　“我没有吃过，不知道。”
<br/>　　“那我可以留下给你尝尝，以后我们可以长期合作。”其实有些势力大的山寨，和城里的商行都有联系，程大雷觉得自己可以发展一下苏樱，她毕竟是四海商行的千金。
<br/>　　想到这里，程大雷忽然问：“对了，你已经被抢过来这么多天了，怎么没人来赎你，难道你没人要了？”
<br/>　　一直平静的苏樱，在听到这句话后脸色大变，双眸之中藏着怨气，也藏着无奈。
<br/>　　这个时候，程大雷满怀深情道：“放心，就算全世界背叛你，我也会和你一起，背叛全世界。”
<br/>　　还没等苏樱反应过来，程大雷已经飞速的离开屋子，他还要去问徐灵儿，辣椒是在哪里采到的。
<br/>　　徐灵儿虚惊一场，自己害的整个山寨担忧，再见到程大雷的时候有些窘迫。
<br/>　　她带着程大雷、徐神机，黄三元几人来到山寨外的一片山坡上。程大雷看到这里长着一颗颗成熟的辣椒，上面挂着红彤彤已经成熟的果子。
<br/>　　辣椒的亩产量要远远超过小麦，大豆，就是这一亩辣椒已经足有几千斤。按照物依稀为贵的道理，一斤辣椒就能换十几斤的粮食。
<br/>　　这要比山寨种植任何作物都来得实惠。
<br/>　　而且辣椒的成熟期较短，两三个月就可以成熟，山寨的十亩地都种上辣椒，冬天来临前就能有数万斤收获，换成粮食将会是一个天文数字。
<br/>　　程大雷激动起来，拍着徐神机肩膀道：“军师，把这些辣椒都摘掉，我们今年就种辣椒。”
<br/>　　“大当家，你没有开玩笑吧，我们进入要种毒，这到冬天我们吃什么？”
<br/>　　程大雷无语，和这种人真的无法沟通。他只有重重的一拍程大雷肩膀：“干活，这是命令。”
<br/>　　……
<br/>　　这个世界是很慢的，没有先进的生产力，一切都需要人工完成。而山寨满打满算，也就只有六个人。
<br/>　　由六人摘辣椒，再拉回山寨晒干，然后还要刨出里面的种子，留着种植。
<br/>　　时间一天天过去，山寨几个人都忙得热火朝天，但就算这样工作进度依旧很慢。
<br/>　　不过，就算再慢，程大雷也没有让苏樱加入，好不容易积攒一些好感度，他可不想被破坏掉。
<br/>　　“今天是少羽在山下值班嘛？”
<br/>　　“嗯呐，明天就该轮到我了。”秦蛮道。
<br/>　　辣椒要种，山寨的本职工作可是不能丢，所以程大雷安排自己、秦蛮，林少羽轮流在山下值班，他的任务还是要刷呀。
<br/>　　“大当家，明天也让我去打劫吧，辛苦活不能只让你们做。”
<br/>　　“住口！”程大雷呵斥他：“不要以为我看不出来你不想摘辣椒。”
<br/>　　就这个时候，林少羽缓缓从外面进来。程大雷忙问：“怎么样，今天有收获吗？”
<br/>　　连续好长时间，山下根本没人经过，程大雷的任务进度依旧是1|10 。
<br/>　　“算是有收获吧。”林少羽身后闪出个小女孩：“她是来找苏家小姐的。”
