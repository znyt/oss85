　　尹龙仙本以为楚风眠跟狂龙帝也仅仅只是交谈了一阵罢了。
<br/>　　可远远没想到，楚风眠可以治好狂龙帝的伤势。
<br/>　　哪怕是听到楚风眠的话，她还是有些感觉到不可置信。
<br/>　　“自然是真。”
<br/>　　楚风眠随意开口道，他也没有去解释。
<br/>　　等到狂龙帝度过天人大劫之后，一切都会尘埃落定。
<br/>　　楚风眠没有在继续开口，他的目光，看向了那宫殿之上的劫云之中。
<br/>　　天人大劫，楚风眠也只是在古籍的记载之中曾经听闻过，如今这还是楚风眠第一次亲眼所见。
<br/>　　他要好好观察这天人大劫之中的玄妙，来为未来楚风眠去度天人大劫来做好准备。
<br/>　　见到楚风眠不愿意去说，尹龙仙也没有在追问下去，她的目光紧张的盯着宫殿之中。
<br/>　　天人大劫，可是对寻常武者妖神来说最难以度过的一次劫难，也是最为重要的一次劫难。
<br/>　　一旦是度过天人大劫，便是羽化登仙，位列仙班，成就天人。
<br/>　　从此之后，蜕凡为仙。
<br/>　　度天人大劫，危险重重，稍有不慎，便是万劫不复。
<br/>　　这也是许多半步天人境的强者，不到万不得已的地步，都不敢去度天人大劫的原因，便是他们也不想要自己死在天人大劫之上。
<br/>　　海公主也在一旁仔细的看着宫殿之上开始凝聚的劫云。
<br/>　　对任何武者妖神来说，能够亲眼看到度天人大劫，都是一场机缘，海公主也在全神贯注这，观察着其中的玄妙。
<br/>　　劫云之中积蓄的力量，已经是越来越庞大，隐隐预约之间，楚风眠已经是听到了不少的雷声。
<br/>　　雷霆在这劫云之中，翻滚，咆哮着，恐怖的气息，周围都可以感觉的清清楚楚。
<br/>　　一旦是开始度天人大劫，只怕是整个苍龙星域的强者，都可以感觉得到。
<br/>　　这倒是一个麻烦事。
<br/>　　武者妖神渡劫结束的一刻，也是最为虚弱的一刻。
<br/>　　所以无数武者妖神，都会前去窥探，等到渡劫结束的一刻突然出手，抢夺果实。
<br/>　　整个苍龙星域，数千年间都没有出现过天人大劫了。
<br/>　　这一次狂龙帝渡劫，只怕是会吸引大量的武者妖神过来，这倒是一个麻烦。
<br/>　　正常情况之下，如果武者妖神要去渡劫，都会选择一处无比偏僻之地，这样就不会引起他人注意。
<br/>　　但是这一次，狂龙帝刚刚恢复了实力，天人大劫已经降至，已经是来不及寻找地方了，
<br/>　　“算了，帮人帮到底，就在帮一次吧。”
<br/>　　楚风眠自言自语一声，突然凝聚灵力在他的掌心之中，一掌打了过去。
<br/>　　这一股灵力之中，蕴含空间之道，笼罩着整个山峰之中，方圆千里的范围，都被楚风眠这一股灵力所笼罩。
<br/>　　楚风眠一出手，便是以空间之道的玄妙，将这方圆千里的范围，都给封锁了起来。
<br/>　　这天人大劫的气息，便是被封锁在了这空间之中，无法传递出去。
<br/>　　如此一来，狂龙帝渡劫，苍龙星域之中的强者也就无法察觉得到了。
<br/>　　“多谢楚公子。”
<br/>　　尹龙仙看到楚风眠的举动，感激的说道。
<br/>　　她也知道楚风眠此举，是在帮狂龙帝，免去有人窥探的麻烦。
<br/>　　“无妨，渡劫要开始了。”
<br/>　　楚风眠随意开口道，他的目光，一直是在注意着宫殿之上的劫云。
<br/>　　如今劫云之中积蓄的力量，已经是即将达到了极限，一层层的劫云，足有千百层之多。
<br/>　　从其中咆哮的雷霆，更是数不胜数，在这劫云之中，更是有着一道道由雷霆所化的雷龙，雷虎，种种以雷霆凝聚出的异兽，仿佛真正的生灵，在其中咆哮这。
<br/>　　这些雷霆异兽，每一尊都蕴含着纯粹的毁灭之力。
<br/>　　劫数，便是毁灭。
<br/>　　宫殿之中，狂龙帝一身实力，也已经是恢复到了巅峰，甚至说他现在的一身实力，已经是超越了曾经的巅峰，破而后立。
<br/>　　楚风眠看得出，这狂龙帝的一身实力，已经是跟真正的天人境无异了。
<br/>　　狂龙帝坐在宫殿之中，目光也看向头顶之上，凝聚的无数劫雷，他的嘴角，都是露出了无边的自信。
<br/>　　天地突然静止了下来，一切雷霆的咆哮之声，都全然消失了。
<br/>　　不过楚风眠知道，这只是天劫落下前的前奏。
<br/>　　在这短暂的静止结束之后，无边毁灭的气息，从这劫云之中爆发而出。
<br/>　　轰！
<br/>　　一尊尊雷龙，雷虎，雷豹，雷牛，无数雷霆异兽，都是从这劫云之中咆哮而下，他们的目标只有一人，便是狂龙帝，要将狂龙帝全然毁灭。
<br/>　　“龙逆天地！”
<br/>　　在那宫殿之中的狂龙帝，突然一步踏出了宫殿，他的身形，突然变化着，化为了一条巨大无比的远古战龙，足有千丈之长。
<br/>　　这远古战龙，就是狂龙帝的本体.
<br/>　　在面对这劫雷降临下来的一刻，这狂龙帝也爆发出了全部的力量，以本体来应对无数降临的劫雷。
<br/>　　顿时这一条远古战龙，直冲九霄之上，跟着一尊尊雷霆凝聚的异兽，厮杀起来。
<br/>　　一尊尊雷霆凝聚的异兽，都在这一条远古战龙的面前，被轰杀，被撕裂。
<br/>　　不过这些雷霆异兽，几乎是杀不尽，杀不完，每斩杀一尊，从这劫云之中，就会诞生出新的雷霆异兽出来。
<br/>　　一次次的厮杀，足足是一直过去了数个时辰的时间，那一座宫殿，都被其中的余波震的破碎不堪。
<br/>　　整座山峰，都已经是被彻底毁掉了，满布疮痍。
<br/>　　这些雷霆异兽，每一尊的实力，几乎都可以轰杀一位半步天人境的武者妖神。
<br/>　　可在狂龙帝的面前，几乎是砍瓜切菜一般，被全然毁灭。
<br/>　　“快渡劫完了。”
<br/>　　楚风眠缓缓的睁开眼睛，就在刚刚，他便是在参悟这天人大劫的玄妙。
<br/>　　不过这狂龙帝的天人大劫，却是跟古籍之中记载的，其他人的天人大劫，没有什么两样。
<br/>　　看来狂龙帝虽然曾经尝试过去度逆天大劫，可终究没有成功，他的天人大劫，跟寻常的天人大劫，没有什么区别。
<br/>　　楚风眠从其中探察出的奥秘，也不算深，毕竟楚风眠乃是天地异种，他的天人大劫，会跟这狂龙帝的逆天大劫，截然不同。
