　　杭山市儿童医院。
<br/>　　慕容飞手里抱着一个大纸箱跟在佟千雪身后，而佟千雪只是背着一个小挎包，不乏轻松的走在前面。
<br/>　　天空中虽然没有毒辣的太阳，可是在这闷热潮湿的雨季，即便是没有太阳，低气压也能将人身体中的水分和盐分，通过人的毛孔压榨出身体，这样的天气只要稍微一动就全身是汗，黏糊糊的非常不舒服。
<br/>　　按照慕容飞的说法，这样的天气就应该在开着空调的房间里，吹着空调，喝着冰镇饮料才是最佳选择，可是佟千雪根本没有给他这样享受的机会，直接把他从家里拉了出来，让他陪着自己到医院来，整理一些东西回家。
<br/>　　而最令慕容飞崩溃的是，佟千雪的汽车居然送去4S店保养去了，因为这段时间天气不好的缘故，所以车子还在4S店停着，慕容飞和佟千雪只能打车去儿童医院，整理完东西再打车回去。
<br/>　　慕容飞抱着一大箱子满头是汗的跟着佟千雪往医院大门外走去，慕容飞停下脚步，向上托了一下大纸箱子，对走在自己前面佟千雪说道：“我说佟医生，你就是回去养胎，过段时间就要回来上班，需要太这么多东西回去吗？”
<br/>　　“当然需要啊。”佟千雪停下脚步，转身看着跟在自己身后的慕容飞，说道：“有句话叫做活到老学到老，回去养胎的三个月时间不能白白浪费，这也算是胎教啊。”
<br/>　　“你这是打算从娃娃抓起，从小就打算把他（她）培养成跟你一样的医生啊。”慕容飞抬起膝盖，在纸箱子底部抬了一下，双手抱住纸箱子向上抬了抬，以更省力的方式抱着纸箱子，对佟千雪说道：“可是你带回去的东西也太多了，这么大一个箱子有一半是书，这么短时间，你来得及看完这么多书吗？”
<br/>　　“你这是在质疑作为学霸的我的学习能力吗？”佟千雪柳眉倒竖，美眸瞪着慕容飞，问道。
<br/>　　“没有，没有，没有。”
<br/>　　慕容飞连连摇头，这个时候他可不敢惹佟千雪生气，要不然自己老妈可是真的会劈了自己的。
<br/>　　“没有最好！”佟千雪转身朝着医院大门口走去，用命令的口吻对慕容飞说道：“回去做一碗面条给我吃，我饿了！”
<br/>　　“我这个做了一上午苦力的人都没说饿，你倒是先饿了。”慕容飞嘀咕着说道。
<br/>　　“废话，我现在一个人代表着两个人，你能比吗？”佟千雪瞪着慕容飞，说道：“你要是让我饿肚子，就是虐待我肚子里这个小的，你可别忘了，这可是你的孩子！”
<br/>　　这可是一招必杀技，瞬间就堵的慕容飞说不出话来。
<br/>　　回到佟千雪所住的小区，慕容飞替佟千雪把纸箱子放在客厅一个角落中，然后就去厨房忙活了；而佟千雪则是从纸箱子里拿了几本自己需要的书就去了书房。
<br/>　　佟千雪打开电脑，正准备工作的时候，电脑中突然弹出了一个窗口，上面用中文说明佟千雪电脑中的重要文件、文档、照片将被加密，如果想要解锁即将被加密的文件，要求佟千雪在四十八小时内以比特币的方式向一个账户支付1000比特币，而如果超时没有支付的话，支付价格会翻倍，并且被加密的文件有可能会遭到破坏，无法被恢复。
<br/>　　“慕容，我的电脑好像中毒了。”佟千雪对着厨房的方向，喊道：“你快过来给我看看到底是怎么回事。”
<br/>　　慕容飞在围裙上擦着手，从厨房里走出来，走进书房。龙隐部队对队员的要求比较高，每个人都必须熟练掌握电脑使用，不要求每个人都能达到子午那种传说级别的黑客水平，但至少学会几种破译密码的方法，这在特种作战中非常有用。
<br/>　　慕容飞查看了一下佟千雪的电脑不禁皱起了眉头，对佟千雪问道：“你电脑里有什么重要的东西吗？”
<br/>　　“也没有什么重要的东西，就是一些从国外医学网站上下载的国外医学论文。”佟千雪对慕容飞问道：“不能修复吗？”
<br/>　　慕容飞摇了摇头，对佟千雪说道：“我打电话请教一下高手。”
<br/>　　慕容飞拨通了云熙别墅里的电脑，电话没想几声就被接通了，不过接电话的不是云熙，而是云悠。
<br/>　　“喂，请问你找谁？”云悠脆生生的声音，从电话中传来。
<br/>　　“无常叔叔，你要找我爸爸吗？”云悠问道。
<br/>　　“不，我要找你妈妈，她在吗？”慕容飞说道。
<br/>　　“你稍等一下，妈妈刚才接了一个电话就上楼去了。”云悠放下电话正准备叫云熙听电话的时候，正好看到罗昊从楼梯上走下来，立刻把电话递给了罗昊，说道：“爸爸，是无常叔叔的电话，他要找妈妈。”
<br/>　　罗昊接过电话，“无常？”
<br/>　　“昊子，公主在不在？千雪电脑中了病毒，你也知道，在电脑上我就是个半吊子，想要请教一些公主要怎么处理。”慕容飞对罗昊说道。
<br/>　　“千雪电脑也中毒了？”罗昊皱眉问道。
<br/>　　“什么叫‘也中毒’了？”慕容飞立刻从罗昊的话里听出了一些不寻常，问道：“难道你们家里电脑也跟我中了一样的病毒？”
<br/>　　“这倒没有。”罗昊回答道：“你也知道我们家里的电脑都是小熙弄的，还是很安全的，但是这几天世界各地都爆发了一种蠕虫式勒索病毒，这种病毒通过系统漏洞自主进行传播，很多学校、医院、政府部门都遭到了攻击，小熙刚才就是接到了子午教官的电话，上楼跟子午教官探讨这个电脑病毒去了。”
<br/>　　“很棘手？”慕容飞问道。
<br/>　　“暂时还不清楚，但是刚才我问小熙，应该很棘手。”罗昊顿了顿，随后对慕容飞说道：“无常，根据子午教官的说法，这次电脑病毒的大规模爆发似乎并不简单。”
<br/>　　“跟他们有关系？”慕容飞神情立刻变得严肃起来，问道。
<br/>　　“目前还不确定，但是不排除这种可能性，但是做好随时归队的准备。”
